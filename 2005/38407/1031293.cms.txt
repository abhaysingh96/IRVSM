लेख: जाता जाता
मालिकेतील मराठी
[ Thursday, February 24, 2005  07:56:54 pm]


चकोर रात्री ऑफिसमधून परतला. चकोरी टीव्हीसमोर होती. पण आवाज नव्हता.

' जाहिराती वाटत नाहीत या! ' चकोरने पडद्यावरची दृश्यं बघून विचारलं.

' मराठी मालिका. ' चकोरीने त्याच्याकडे न बघता उत्तर दिलं. दरम्यान ज्यु. चकोरीने आणून दिलेलं पाणी पिणाऱ्या चकोरला तिच्या उत्तराने ठसका लागला.

चकोरची अवस्था काय झाली आहे हे चकोरीच्या लक्षात आलं.

' मुद्दामच आवाज म्युट ठेवला आहे. ' तिने खुलासा केला.

' का बरं ?' चकोरने आवाजात सहजपणा ठेवला, पण आश्चर्याचे भाव त्याला लपवता आले नाहीत.

' मराठीवरील अत्याचार ऐकवत नाहीत. ' चकोरीने शांतपणे सांगितलं. ' म्हणजे काय करतात ?' बूट काढणाऱ्या चकोरचं कुतूहल वाढलं.

' हिंदी मराठी बोलतात. ' चकोरीच्या बोलण्यात किंचित संताप होता.

बॅगेतून डबा काढणाऱ्या चकोरने चेहरा असा केला की त्याला आपलं म्हणणं कळलेलं नाही हे चकोरीच्या लक्षात आलं.

' अहो, हे डायलॉग रायटर्स आधी हिंदीत संवाद मनातल्या मनात तयार करतात आणि त्याचं मराठी भाषांतर कागदावर उतरवतात. ' तिने त्याला समजावून सांगण्याचा प्रयत्न केला.

पण चकोरच्या डोक्यात फारसा प्रकाश पडलेला नाही हे तिला जाणवलं.

' म्हणजे बघा! ' तुमचं घर कमी आहे का माझ्या घरावर नजर ठेवण्यापेक्षा ' हे कशाचं भाषांतर आहे असं वाटतं तुम्हाला ? किंवा ' आजकाल माझ्याबद्दल काय होतं आहे ' किंवा ' माझी मदत करू नका '. चकोरीने क्षणार्धात ही वाक्यं त्याला ऐकवली.

चकोरने मनातल्या मनात त्यांचं मूळ हिंदी शोधलं.

' पटतं म्हणणं तुझं. ' त्याने बॅग जागेवर ठेवत तिला दाद दिली.

' अहो, असे अनेक अत्याचार चालू आहेत. सौरभ हा शब्ददेखील हिंदी सिरीयलमध्ये उच्चारतात तसा इथे चार वर्षाचं पात्र उच्चारतं. बोला आता! ' चकोरी रागाने नुसती लाल झाली होती.

' बघता कशाला असल्या मालिका ?' चकोरने शांतपणे विचारलं.

यावर चकोरी गांगरली.

' नेहमी नाही बघत. आज तुम्हाला उशीर झाला म्हणून लावला टीव्ही. ' तिने खुलासा केला.

' अहो भाषा भ्रष्ट होते ही एक बाजू झाली. पण संस्कृतीही बिघडते. ' छटपुजा ' आणि ' करवाँ चौथ ' सारखे प्रकार मग दिसतात मालिकेत! आपल्या मराठी माणसाची काही वेगळी श्रद्धास्थानं असतात. ती विसरली जातात. ' चकोरी तळमळीने बोलत होती.

' मला वाटतं, पत्र लिहून तू भावना कळव या मंडळींना. ' चकोरने पडद्याकडे बोट दाखवत सुचवलं.

चकोरीने रिमोटने टीव्ही ऑफ केला.

चकोर
