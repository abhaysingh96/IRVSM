लेख: जाता जाता
शिवाजीमहाराज की जय!
[ Tuesday, January 18, 2005  07:45:49 pm]


' शिवाजीराजांच्या पुतळ्यांना सध्या चांगले दिवस आलेले दिसताहेत. ' चकोरीने म. टा. वाचता वाचता कॉमेंट केली.

' काय झालं! शिवजयंतीला तर वेळ आहे. ' चकोरने कुतुहलाने विचारलं आणि चकोरीला माहितीही दिली.

' महापुरुषांच्या नामस्मरणासाठी जयंतीपर्यंत थांबण्याची गरज नसते. त्यामुळे मुंबईच्या उपमहापौरांनी, छत्रपती शिवाजी टमिर्नसच्या इमारतीवर शिवरायांचा पुतळा उभारण्याची सूचना केली आहे. ' चकोरीने पेपर त्याच्यापुढे सरकवत सांगितलं.

चकोरने ' जाता जाता ' हे पटेल सेनेचे की भाजपाचे ?' असा प्रश्ान् विचारला.

' बहुधा शिवसेनेचे असतील. कारण सेनेला पटेल वगैरे आडनावाच्या नेत्यांचं प्रेम जास्त असतं. ' चकोरची मल्लीनाथी.

' सेनेत पटेल वगैरेंना इतक्या छोट्या पदावर ठेवत नाहीत. थेट आमदार खासदार करतात. ' चकोरने सेनेचा इतिहास आठवत तिच्या ज्ञानात भर घातली.

' बाबा, काय बोअर करता तुम्ही! कोण पटेल आणि कोण सेना! ' ज्यु. चकोरीने एकदम डायलॉग टाकला.

' अगं, आम्ही महाराजांबद्दल बोलत होतो. त्यामुळे सेनेचा उल्लेख येणारच ना! ' चकोरीने तिला विचारलं.

' महाराजांच्या सेनेबद्दल तुम्ही बोलत नाहीत. भारतीय सेनेबद्दलही नाही. भलत्याच सेनेबद्दल तुमची चर्चा चालू आहे. ' ज्यु. चकोरने त्यांची गाडी रुळावर आणली.

' अरे आम्ही शिवबाबद्दल बोलतो आहोत. ' चकोरीने पुन्हा त्याला आठवण करून दिली.

' मग त्याच्यात पटेल कशाला ? आम्हाला पटेल असं काही तरी बोला ना! ' मुलाने तेवढ्यात ' पीजे ' मारला, पण त्याच्या मुद्यात तथ्य आहे अशा अर्थाने त्याला बहिणीने मान हलवत साथ दिली.

' आमचा फोकस राजेच आहे. पटेल हा दुय्यम विषय आहे. ' चकोरने आता त्याची समजूत काढली.

' तसं असेल तर त्यात पुतळ्याला चांगले दिवस, सीएसटी स्टेशन वगैरे कशाला! ' मुलाने पुन्हा बिनतोड सवाल टाकला.

चकोर दांपत्याच्या मुखावर, ही दोघं आपली गंमत करताहेत की गंभीरपणे बोलत आहेत हे कळत नाही असे भाव पसरले.

' तुम्ही असं काय करता ? कोणीतरी पटेल नावाचे उपमहापौर राजांचा पुतळा स्टेशनच्या इमारतीच्या वर उभारा असं सांगत आहेत. त्यातून हा विषय निघाला. ' चकोरीने त्याला नीट समजावून दिलं.

' तेच सांगतो आहोत आम्ही. पुतळा, पटेल, स्टेशन, सेना हे सगळं फिजूल आहे. महाराजांचं चरित्र नीट वाचावं आणि त्यांनी घालून दिलेले धडे गिरवावेत आणि पक्के करावेत. मॅटर एन्ड्स! ' मुलाने विषय संपवला.

चकोर
