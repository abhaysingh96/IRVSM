महामुंबई
'युवा', 'वीर जारा'ला फिल्मफेअर
[ Sunday, February 27, 2005  12:05:50 am]


म. टा. प्रतिनिधी

मुंबई : सवोर्त्कृष्ट चित्रपटाच्या पुरस्कारासह ' वीर जारा ' साठी चार पुरस्कार आणि सवोर्त्कृष्ट दिग्दर्शकाच्या (कुणाल कोहली) पुरस्कारासह ' हम तुम ' साठी पाच पुरस्कार तसेच ' धूम ' साठी दोन पुरस्कार असे तब्बल 11 पुरस्कार पटकावून यश चोप्रा यांच्या ' यशराज फिल्म्स ' ने फिल्मफेअरच्या सुवर्णमहोत्सवी सोहळ्यात बाजी मारली. मात्र, मणिरत्नमचा, बॉक्स ऑफिसवर फारशी जादू दाखवू न शकलेला ' युवा ' सर्वाधिक, सहा फिल्मफेअर मिळवून गेला. शाहरुख खानला ' स्वदेस ' साठी तर राणी मुखजीर्ला ' हम तुम ' साठी सवोर्त्कृष्ट अभिनयाची पारितोषिके देण्यात आली. याबरोबरच ' युवा ' साठीही सवोर्त्कृष्ट सहाय्यक अभिनेत्रीचा पुरस्कार पटकावून राणी हीच यंदाची राणी ठरली. आणि बादशहा शाहरुखने ' फिल्मफेअर पॉवर ' पुरस्कारही पटकावला.

हिंदीचे पहिले सुपरस्टार दिलीपकुमार, बुजुर्ग संगीतकार नौशाद आणि गानसम्राज्ञी लता मंगेशकर यांचा सुपरस्टार अमिताभ बच्चन आणि मुख्यमंत्री विलासराव देशमुख यांच्या हस्ते विशेष गौरव करण्यात आला. रमेश सिप्पी दिग्दशिर्त शोले ' हा ' फिल्मफेअर ' च्या 50 वर्षांतील सवोर्त्कृष्ट चित्रपट ठरला. एकेकाळी देशभरातील तरुणींच्या हृदयावर राज्य करणारे सुपरस्टार राजेश खन्ना यांना कारकीर्द गौरव पुरस्काराने गौरविण्यात आले. वांदे-कुर्ला संकुलातील एमएम- आरडीएच्या विस्तीर्ण मैदानावर फिल्मफेअर सन्मानसंध्या तब्बल चार तास रंगली.

' युवा ' मध्ये ' कभी नीम नीम, कभी शहद शहद ' असे जोडपे साकारणाऱ्या अभिषेक बच्चन आणि राणी मुखजीर् यांनी सहाय्यक अभिनेता आणि अभिनेत्रीचा पुरस्कार पटकावला. ' ऐतराज ' मधील प्रियांका चोप्राने साकारलेली मादक आणि खतरनाक अँटिहिरोइन तिला खलभूमिकेचा पुरस्कार देऊन गेली. ' हमतुम ' साठी सैफ अली खान विनोदी भूमिकेतील सवोर्त्कृष्ट अभिनेता ठरला.

पदार्पणातील सवोर्त्कृष्ट कामगिरीचा पुरस्कार ' टारझन द वंडर कार ' साठी आयेशा टाकिया हिने पटकावला. संगीतक्षेत्रातील दमदार पदार्पणाचा आर. डी. बर्मन स्मृती पुरस्कार ' मर्डर ' मधील ' भीगे होंठ तेरे ' गाणाऱ्या कुणाल गांजावालाने पटकावला. याच गाण्यासाठी त्याने सवोर्त्कृष्ट गायकाचा पुरस्कारही पटकावला. ' हमतुम ' मधील ' साँसों को साँसों मे ' या मखमली गाण्यासाठी अलका याज्ञिक सवोर्त्कृष्ट गायिका ठरली. ' मै हूँ ना ' ला अनु मलिकने दिलेले संगीत सवोर्त्कृष्ट ठरले. सवोर्त्कृष्ट गीतकार जावेद अख्तर हेच असतील, हे स्पष्ट होते. ' वीर जारा ' तील ' तेरे लिए हम है जिए ' साठी त्यांनी तो पटकावला.
