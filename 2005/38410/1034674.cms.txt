अर्थव्यवहार
'क्वॉलिस'ची जागा घेणारी 'इनोव्हा' दाखल
[ Sunday, February 27, 2005  08:30:30 pm]


वृत्तसंस्था

नवी दिल्ली : जगातील तिसऱ्या क्रमांकाची सर्वात मोठी मोटार उत्पादक कंपनी म्हणून ओळखल्या जाणाऱ्या जपानी टोयोटा मोटर कॉपोर्रेशनने आपल्या ' क्वॉलिस ' या प्रमुख कारची जागा घेणारी नवी ' मल्टि-युटिलिटी ' मोटार ' इनोव्हा ' भारतीय बाजारपेठेत गेल्या गुरूवारी दाखल केली. पेट्रोल व डिझेल या दोन्ही इंधनांचा वापर असलेल्या ' इनोव्हा ' मोटारींच्या मॉडेल्सच्या किंमती अनुक्रमे 6.82 लाख व 7.42 रु. असतील.

देशभरात या कॅलेंडर वर्षामध्ये 45,000 ' इनोव्हा ' गाड्या विकण्याचे लक्ष्य कंपनीने ठरविले आहे. टोयोटा मोटरची भारतीय उपकंपनी असलेली ' टोयोटा किलोर्स्कर मोटर ' या नव्या मोटारींची देशांतर्गत बाजारपेठेत विक्री करणार असून, पहिल्याच महिन्यात 4,500 ' इनोव्हा ' मोटारी विकण्याचा प्रयत्न असेल, अशी माहिती कंपनीकडून देण्यात आली.

' इनोव्हा ' देशातील पाच शहरांत एकाच दिवशी विक्रीसाठी दाखल करण्यात आली असून, दुसऱ्या वर्षात 60 हजार मोटारी विकण्याची कंपनीची योजना आहे. टोयोटा मोटर कॉपोर्रेशनचे प्रेसिडेंट फुजिओ चो यांनी पत्रकारांना सांगितले की, बंगलोर येथील आपल्या ' ट्रान्समिशन प्लॅण्ट ' मध्ये कंपनीने याआधीच दोन हजार कोटी रुपयांची गुंतवणूक केली आहे. ' इनोव्हा ' मोटारींच्या उत्पादन प्रकल्पातील गुंतवणूक सहाशे कोटी रु. आहे.
