लेख: जाता जाता
बजेटचा मूड
[ Sunday, February 27, 2005  09:11:23 pm]


' काय हो, आज दांडी वाटतं! ' चकोरीने चकोरचा मूड बघून विचारलं.

' कशाबद्दल ?' चकोरने आश्चर्यानं विचारलंं.

' घरी बसून बजेट बघायचं निमित्त! ' इतकं बोलून चकोरी हसली. ' त्यापेक्षा ऑफिसमध्ये धमाल असते. बजेटच्या नावाखाली अनेकजण नुसतेच टाईमपास करत असतात. अत्यंत गंभीर चेहऱ्याने चर्चा करायची आणि देशाचं कसं होणार असं म्हणून सुस्कारे टाकायचे. ' चकोरने आपण घरी बसणार नाही, असं सूचित केलं.

चकोरीने सुस्कारा सोडला.

' तुला का इतका आनंद झाला मी ऑफिसला जाणार म्हणून ?' चकोरने विचारणा केली.

' तुम्ही घरी बजेट बघणार म्हणजे दर थोड्या वेळाने तुम्हाला चहा लागणार, खायला काय, असा प्रश्ान् विचारणार. त्यातून सुटले मी! '

' कोणाला तरी वाटेल मी बकासुरासारखा खात असतो. ' चकोरने तक्रारीचा सूर लावला.

' मी असं म्हटलं का! पण तुम्हाला नवनव्या कल्पना सुचतात अशा वेळी. नेहमीचं जेवण नको असतं तुम्हाला. ' चकोरीने स्पष्टपणे सांगून टाकलं.

' डॅड, तुम्हाला बजेटमध्ये इतका का इंटरेस्ट असतो ?' मुलाने गंभीरपणे विचारलं.

' माणूस आशेवर जगतो म्हणून! ' चकोरच्या उत्तराने तिघेही चक्रावले.

' तुम्हाला बजेटच्या दिवशी एवढी कसली आशा असते ?' मुलीने प्रश्ान् टाकला.

चकोरने तिच्याकडे बघितलं.

' तुला नाही कळायचं ते! ' त्याने एका वाक्यात तिला उडवलं.

' सांगा तर खरं! नाही समजलं तर मी तुम्ही मला एक्स्प्लेन करा. ' मुलीने त्याला गळ घातली.

' अगं, आम्हा पगारदारांना दरवषीर् आशा असते की फायनान्स मिनिस्टरांना आमची दया येईल. थोडा इन्कमटॅक्स कमी करतील. मग आमच्या हातात थोडे जास्त पैसे येतील. ' चकोरने समस्त पगारदारांचं दु:ख बोलून दाखवलं.

' मग का नाही कमी करत ?' मुलाने वडिलांची भूमिका समजली अशा अर्थाने विचारलं.

' हात पाय न हलवता सरकारी तिजोरीत करोडो रुपये पडत असतील तर कशाला आमची फिकीर करतील ?' चकोरने त्याला समजावलं.

' म्हणजे! ' मुलीच्या विचारण्यात आश्चर्य, कुतूहल असे अनेक भाव होते.

' पगारातून कंपन्या परस्पर टॅक्स कापतात. इन्कमटॅक्स डिपार्टमेंटला कष्ट काहीच नाहीत. ' चकोरने सांगितलं.

' तुम्ही प्रोटेस्ट का नाही करत ?' मुलाच्या विचारण्यात सहानुभूती होती.

' म्हणजे नक्कीे काय करायचं ? आम्ही हेल्पलेस आहोत. ' चकोरचा स्वर कातर झाला.

चकोर
