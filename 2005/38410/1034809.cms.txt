महाराष्ट्र
'स्व'देसच्या शोधाची सांजसफर!
[ Sunday, February 27, 2005  11:25:17 pm]


म. टा. प्रतिनिधी

पुणे : वेगाने बदलत चाललेली सामाजिक मूल्ये, परस्परांमधील नाती याचा संदर्भ शोधत गावाकडून शहराकडे अन् शहराकडून परदेशात होत जाणारे स्थलांतर..., शाहरूख खानला त्याच्या ' इमेज ' मधून बाहेर काढण्यासाठीचे प्रयत्न..., चित्रपटात गुंफलेली स्वावलंबनाची गुंफण, अशा अनेक विषयांना स्पर्श करत रविवारी सायंकाळी ' स्वदेस ' च्या निमिर्तीमागची कथा उलगडली.

निमित्त होते ' केसरी ' तफेर् ' स्वदेस ' चित्रपटाचे दिग्दर्शक आशुतोष गोवारीकर यांच्याशी आयोजित करण्यात आलेल्या चर्चासत्राचे. समर नखाते, श्रीरंग गोडबोले, डॉ. केशव साठ्ये, सुहास घटवई आणि संजय दिनकर यांनी ' दृक-श्राव्य ' माध्यमातून त्यांच्याशी संवाद साधला.

' आपल्या देशाचा ' प्रॉब्लेम ' च आहे ', ' काय होणार पुढे ', असं सारखं म्हणण्यापेक्षा प्रत्येकानी स्वत:चं काम वेळेत केलं, तर ' क्रांती ' होऊ शकते, हा आशय चित्रपटातून प्रेक्षकांपर्यंत पोहोचवण्याचा मानस होता, असं गोवारीकर यांनी स्पष्ट केलं. गावातून शहरात आणि शहरातून परदेशात जाण्याची ओढ प्रत्येकालाच असते ; पण शहरांच्या विकासाबरोबर खेड्यांच्या मूलभूत विकासाकडेही लक्ष दिलं गेलं पाहिजे. त्यामुळे समाजजीवनात अंतर्बाह्य बदल होऊ शकतील, अशी अपेक्षा त्यांनी व्यक्त केली.

' सिनेमॅटिक ' चित्रपटांऐवजी हिंदी चित्रपटसृष्टीमध्ये सामाजिक समस्यांवर प्रकाश टाकणाऱ्या चित्रपटांची सातत्याने निमिर्ती झाली, तर प्रेक्षकांच्या दृष्टिकोनातही फरक पडेल, असं त्यांनी सांगितलं. स्वातंत्र्यानंतर गेल्या 50 वर्षांच्या प्रवासात, साध्य न झालेल्या गोष्टी प्राप्त करण्यासाठी आणि देशाच्या सर्वांगीण उन्नतीसाठी आत्तापासून प्रयत्न झाले, तर पुढील 50 वर्षांनंतर देशाचं समृद्ध चित्र प्रत्यक्षात येऊ शकेल, असं मतही त्यांनी मांडलं. समाजातील प्रत्येकाच्या मनात कुठे ना कुठे दडलेल्या मोहन भार्गवला दृश्य रूपात प्रकट करू शकलो, असं सांगताना ' लगान ' सारखं व्यावसायिक यश या चित्रपटाला मिळणार नाही, हे गृहित धरलं असल्यानं ' बॉक्स ऑफिस ' वरील उत्पन्नाचा फारसा विचार केला नाही, असंही त्यांनी नमूद केलं.

आमिरचं पारडं जड ...

' लगान ' मध्ये आमिरबरोबर काम केलं असलं, तरीही चित्रपटातील ' मोहन भार्गव ' ची व्यक्तिरेखा त्याला साजेशी नव्हती. विविध चित्रपटांतून न्यूयॉर्क, लंडनमध्ये वावरणारा ' शाहरूख ' चं त्या भूमिकेसाठी समर्पक होता, असं आशुतोष गोवारीकरांनी स्पष्ट केलं. मात्र, दोघांच्या तुलनेत गुणांमध्ये आमिर सरस ठरतो, असं मतही त्यांनी व्यक्त केलं.
