लेख: जाता जाता
कोकणी क्रूझ!
[ Thursday, March 10, 2005  09:08:46 pm]


वॉव, डॅड, स्टार क्रूझ! ' ज्यु. चकोरी चित्कारली.

चकोरने बातमी वाचून शांतपणे पेपर चकोरीकडे सोपवला.

' तरंगता महाल भारताच्या किनाऱ्यावर फिरणार. चैन करणाऱ्यांना आणखी एक निमित्त. ' चकोरीने भाष्य केलं.

' बाबा, किती चार्ज करतील ?' मुलीने कुतूहलाने विचारलं.

' गेलाबाजार दोन-चार लाख. अशा क्रूझवर लोअर डेक, अप्पर डेक अशी सोय असते की नाही देव जाणे! ' चकोर बालपणात रमला.

' तुमची कोकणात जाणारी बोट वाटली काय! लोअर डेक कसलं आलं! ' चकोरीने त्याची चेष्टा केली.

' क्रूझपेक्षा आमची जनताबोट केव्हाही सरस. अख्खी चाळ जमली असं वाटायचं. प्रत्येकाच्या टिफिनमध्ये वेगवेगळे पदार्थ. सगळ्याचा मिळून असा मस्त स्वाद खाऱ्या वाऱ्याबरोबर दरवळायचा. ' चकोर जरा जास्तच हळवा बनला.

' हो आणि बोट लागलेल्यांचा उपदव व्हायचा तो! ' चकोरीने त्याला पुन्हा स्वप्नातून जमिनीवर आणलं.

' तो एक त्रासदायक मामला असायचा हे खरं! पण तोही टिपीकल चाळ कल्चरप्रमाणे सहन केला जायचा. ' चकोरने माहिती दिली.

' पण बाबा किती बोअर जनीर् असणार! चौवीस तास बोटीत बसून काढायचे. सगळीकडे फक्त समुद. ' मुलीने नाकं मुरडली.

' माझ्या तोंडून तू हे वर्णन लहानपणी ऐकलं आहेस. पण तुला पुन्हा ऐकायची हौस आहे असं दिसतं. अगं, बोटीत कोठेही फिरायला मिळायचं. इंजिन रुमपासून कँटिनपर्यंत सर्वत्र संचार असायचा आमचा. शिवाय समुदपक्ष्यांना काही खायला फेकलं की ते ज्या वेगाने टिपतात, सनसेट आणि सनराईझ हा मेमोरेबल एक्स्पीरिअन्स असतो. ' चकोरने तपशिलात वर्णन केलं.

' तुम्ही सनराइझ बघायला तेव्हा उठायचात वाटतं! ' चकोे विचारलं.

' सूयोर्दय बघायला मिळणार की नाही यावर माझा प्रभातउदय अवलंबून असतो. ' चकोरने मुंबईत सहजतेने बघायला न मिळणाऱ्या सूयोर्दयावर भाष्य केलं.

' बाबा, तुम्हाला बोटीचा प्रवास इतका आवडतो तर आपण अशाच ठिकाणी फिरायला जाऊया या समरमध्ये. ' मुलीने मध्येच गुगली टाकला.

हे ऐकून चकोरीचे डोळे लकाकले.

' चलो अंदमान! ' मुलाने अचानक आदेश दिला.

चचेर्ला लागलेलं वळण बघून चकोर चक्रावला. तिकिटं, बुकिंग, बॉसला मस्का मारून सुट्टी मिळवणं अशी मालिका त्याला दिसू लागली.

क्षणभर विचार करून त्याने मान डोलावली. ज्यु. कंपनीने ' हुई ' अशी आरोळी ठोकली.

' ट्रॅव्हल प्लॅन तुम्ही आखायचा. मी फक्त पैसे देईन. ' त्याने संधीचा फायदा घेत मुलांच्या गळ्यात काम अडकवलं.

चकोर
