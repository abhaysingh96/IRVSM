धावते जग
धोक्याची घंटा
[ Thursday, March 10, 2005  09:05:44 pm]


युनायटेड लिबरेशन फ्रंट ऑफ आसाम ' उर्फ उल्फा या अतिरेकी संघटनेने गेल्या काही दिवसांत पुन्हा एकदा हिंसाचाराचा मार्ग अवलंबल्याने काही काळ शांत असलेला आसाम पुन्हा पेटणार, अशी लक्षणे दिसू लागली आहेत.

बुधवारच्या एका दिवसातच ' उल्फा ' ने आठ ठिकाणी बॉम्बस्फोट घडवून आणले. सुदैवाने यात मोठी प्राणहानी झाली नसली, तरी या स्फोटांनी आसामी सत्ताधाऱ्यांमध्ये अस्वस्थता निर्माण करण्याचे काम मात्र केले आहे. हे बॉम्ब मुळातच कमी क्षमतेचे होते.

ते मोठ्या प्राणहानीसाठी नव्हे तर केवळ घबराट पसरविण्यासाठी घडवून आणले असावेत, असे यावरून स्पष्ट होते. गेल्या वषीर् 15 ऑगस्टला आणि नंतर यंदा 26 जानेवारीला ' परेड ग्राऊंड ' वर दोन स्फोट घडवून आणण्यात आले होते. त्याहीआधी 25 व 26 ऑगस्टला लहान मुले व अन्य निष्पाप नागरिक यांना ' उल्फा ' ने लक्ष्य केले तेव्हा त्यांच्याविरुद्ध जाहीर नाराजीचा सूर घुमला होता.

हजारो लोक ' उल्फा ' विरुद्ध रस्त्यावर आल्याचे दृष्य दिसले होते. 1997 साली समाजसेवक संजय घोष यांच्या हत्येनंतर जसे लोक उतरले होते, तसेच यावेळी उतरले. ' उल्फा ' च्या या हिंसाचारावरून भविष्यात काय घडणार आहे, त्याची चाहूल लागली होती. बुधवारी रात्री एक ऑईल इंडिया लि.ची पाइपलाईन उडवून देण्याचा प्रकार घडला. तो अधिक गंभीर म्हटला पाहिजे.

आसामात गेल्या 25 वर्षांपासून अशांतता आहे. आधी अखिल आसाम विद्यार्थी संघटना (आसू) आणि नंतर उल्फा यांनी वेगवेगळ्या कारणांसाठी व वेगवेगळ्या प्रमाणात हिंसेचा वापर केला.

' उल्फा ' ची स्वतंत्र आसामची मागणी आहे. कालांतराने विद्यार्थी संघटनेचे आंदोलन शमले असले तरी जवळपास एक तप ' उल्फा ' ने आसाम हादरवून सोडत आहे. त्यांच्याशी लोकशाही मार्गाने वागून चचेर्द्वारे तोडगा काढण्याचा मार्गही याआधी अवलंबण्यात आला. परंतु काही काळाने पुन्हा गाडा पूर्वपदावर येतो. आता ' उल्फा ' च्या हिंसाचाराबद्दल ' आसू ' बंद पुकारते आणि तो यशस्वी होतो, असे चित्र दिसते. यात ' आसू ' ची लोकप्रियता दिसत नसून लोकांना आता हिंसाचार नकोय, असा त्याचा अर्थ लावावा लागेल.

' आसू ' च्या भूमिकेला ' उल्फा ' वेगळ्या शब्दांत आव्हान देते. आसाम आंदोलनात ' आसू ' कार्यरत असताना नेल्ली आणि गोहपूर येथे जी हत्याकांडे झाली त्याचे काय, असा सवाल ' उल्फा ' कडून केला जातो. परंतु, या म्हणण्याला काही अर्थ नाही.

' उल्फा ' चे अनेक लोक मारले जात असतात, तरी त्यांना सदस्य मिळविण्यात कधी अडचणी येत नाहीत. याचे कारण आसामातील गरिबी. नोकऱ्या कमी मिळत असल्याने आथिर्कदृष्ट्या खालच्या स्तरावरील लोक काही उत्पन्न मिळावे म्हणून मुलांना ' उल्फा ' त जाण्यासाठी प्रवृत्त करतात, असे सांगितले जाते. म्हणजेच हा वेगळा सामाजिक प्रश्न्ही आहे. त्याकडे केवळ राजकीय दृष्टिकोनातून पाहणे चुकीचे ठरेल.
