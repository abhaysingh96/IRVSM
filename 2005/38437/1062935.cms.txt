खास बात
अधाशासारखे वाचन
[ Saturday, March 26, 2005  08:26:54 pm]


... असाच खेळता खेळता मी इंग्रजी तिसरीच्या परीक्षेस बसलो. परीक्षा संपली. उन्हाळ्याच्या सुट्या सुरू झाल्या. अंगणातल्या पिंपळाची पाने गळली व त्याला नवे कोंब आले. आठवडे-हाटात खरबूज-टरबुजांचा गंध दरवळू लागला. विहिरी आटल्या. धूळ वाढली. ' बावाजी ' च्या बर्फाच्या गाडीभोवती गदीर् गोळा होऊ लागली. माझे विटीदांडूचे व अन्य खेळ जोमात आले...

पांढुर्ण्याहून राजा तावरे हा चपराशी मला घेण्यासाठी आला होता. उन्हाळ्याच्या सुटीसाठी मला हे वडिलांचे बोलावणे होते... इथे, मलकापुरास मी पुन्हा येईन अशी माझी जवळ जवळ निश्चिती होती. पण ती व्यर्थ होती. माझे मलकापूरचे अन्नोदक संपलेले होते... माझे पांढुर्ण्याचे आयुष्य सुरू झाले. इथेही बंगल्याच्या डाव्या अंगाला आवाराबाहेर एक पिंपळ होताच. मात्र तो पिंपळ प्रचंड होता. रात्रभर त्याच्या हात पसरल्यासारख्या फांद्यांची सळसळ चाले.आम्ही उन्हाळ्याचे मागच्या अंगणात पलंग टाकून झोपत असू. पांढुर्ण्याला रात्रीचा वारा फार असे. तो उष्णता टिकूच देत नसे. आमच्या मागच्या अंगणात एक मोगऱ्याचे खूप मोठे झाड होते. ते माझ्या पलंगाशेजारीच होते. त्या मोगऱ्याला दळदार फुले शेकड्यांनी लागत. त्याचा दरवळ माझ्या अंथरुणाभोवती फिरे. जवळच एक निळसर हिरव्या पानांचे छोटे सतापाचे झाड होते. त्या झाडाकडे पहात मी झोपी जाई.

पहाटे माझे वडील त्यांच्या ' प्रिमॅफोन ' वर गाणी लावत. गंधर्वांचे ' पांडुनृपति ', ' पुष्पराग ', ' मम आत्मा ' व ' सकुलतारकसुता '; जानकीबाईंचे ' मुन्सफी दुनिया ' ' दिल एकही से लगा '; मास्टर कृष्णांचे ' लपविला लाल ' व ' ललना मना '; ह्यांविना प्यारासाहेब, महमद हुसेन, मास्टर लब्बू वगैरे... पहाटे गाणी लावून ती ऐकत बसण्याचा हा प्रकार मला अगदी नवीन होता...

पांढुर्णा हे गाव जुन्या मध्य प्रदेशातल्या छिंदवाडा हिंदी प्रभागात होते. तिथे विठोबा पाटलांनी जांब नदीच्या काठावर एक सुबक मंदिर बांधले होते. त्या मंदिरात एक वाचनालय होते. पुस्तकांनी गच्च भरलेली चारदोन कपाटे होती. पुस्तके म्हणजे बहुतेक सगळ्या कादंबऱ्या व नाटके! ' सुरस वाङ्मय ', ' मनोरंजक ग्रंथप्रसारक ', ' भारत-गौरव ' ह्यांनी प्रकाशित केलेली मुख्यत: ती पुस्तके होती... ह्या ग्रंथमालांची अनेक पुस्तके भाषांतरित असत. अलेक्झांडर ड्यूमा, व्हिक्टर ह्यूगो, मारी कॉरेली, मिसेस हेन्री वुडपासून थेट रेनॉल्ड्स् व चार्ल्स गाव्हिर्सपर्यंत अनेक परदेशी लेखकांचा परामर्श ह्या ग्रंथमाला घेत असत. पैकी ' मर्दानी सौंदर्य ', ' त्रिवेणीऊर्फ तिरंगी सामना ', ' तीन शिलेदार ', ' दु:खाअंती सुख ', ' माणिकबाग ' ह्यांसारख्या भाषांतरित कादंबऱ्या मी मलकापुरातच वाचल्या होत्या. आता येथे पुस्तकांचे नवे घबाड मला सापडले... डिकन्स, थॅकरे, ब्रॉन्टे भगिनी, जॉर्ज इलियट, थॉमस हाडीर्, व्हिक्टर ह्यूगो, अलेक्झांडर ड्यूमा... तेव्हाचे हे बहुतेक कीतिर्मंत मराठीत अवतरले होते व मला वाचनालयांच्या कपाटांतून उपलब्ध होते.

मंदिरातील कपाटांच्या जोडीला श्रीमंत बाबाराव धर्माधिकारी ह्यांची कपाटे, गावातल्या छोट्याशा वाचनालयाची कपाटे व शाळेतील पुस्तकांची कपाटेही मला मोकळी झाली. एवीतेवी मला उन्हाळ्याची सुटीच चालू होती... तेव्हा मला तूर्त वाचत बसायला काहीच हरकत नव्हती... आजवर हातात सापडेल ते पुस्तक आधाश्याप्रमाणे वाचून टाकायची मला सवय झालेली होती. नाटक, कादंबरी, चरित्र, प्रवासवर्णन काहीही मला चालत होेते. जेवण्यापेक्षा वाचण्याला माझ्या आयुष्यात जास्त महत्त्वाचे स्थान होते. कसलेतरी पुस्तक वाचल्याविना मला झोपसुद्धा येत नसे.

पु. भा. भावे
