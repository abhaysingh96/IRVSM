महामुंबई: मुंबई टाइम्स
दिलसे नही, ब्रेनसे काम लो...
[ Wednesday, March 02, 2005  04:33:05 pm]


रेश्मा मुरकर

' तो ' आणि ' ती ' इंटरनेट चॅटिंगवरून संवाद साधतात... त्यांच्या ' चॅटींग ' चं रूपांतर हळूहळू प्रेमात होतं... मग सुरू होते प्यारभऱ्या ' एसएमएस ' ची देवाणघेवाण...दोघं बागेत भेटतात... पिकनिकला जातात... कधीतरी ' दांडिया ' च्या रात्री ' तो ' तिला जवळ ओढतो...वगैरे वगैरे... अगदी हिंदी सिनेमाला शोभेल, असा हा ' सीन ' आजच्या युवा पिढीच्या ' ये दिल माँगे मोअर ' म्हणत बेफाम जगण्याला साजेसा! पण, या जगण्याला ' एड्स ' च्या जीवघेण्या वास्तव्याशी सामना करावा लागतो तेव्हा... ? तेव्हा ' दिल ' पेक्षा ' ब्रेन ' माँगे मोअर ' चा, म्हणजेच ' डोकं जाग्यावर ठेवा ' चा वस्तुपाठ घालून द्यावा लागतो.

तरुणांना त्यांच्याच भाषेत वास्तवाची जाणीव करून देणारे हे बोल एका नाटकातून आपल्यासमोर येतात आणि तेही पपेट््स (बोलक्या बाहुल्यां) च्या तोंडातून. त्यात कुठेही ' ब्रेनवॉश ' करण्याचा आव नसतो. ' कूल ड्यूड ' म्हणत साध्या-सोप्या भाषेत तरुणांशी संवाद साधण्याचा प्रयत्न असतो. मीना नाईक यांनी ' ये दिल माँगे मोअर ' हे नाटक रंगमंचावर आणलंय खास तरुणांच्याच भाषेत!

' वाटेवरती काचा गं... ', ' ये बेनकाब चेहरे ' सारख्या नाटकांतून मीना नाईक यांनी बालकांच्या लैंगिक शोषणावर प्रकाश टाकला होता. आता त्यांनी ' एड्स ' कडे लक्ष वळवलंय. ' एड्स ' नियंत्रणाचं काम हे सेक्स वर्कर्स, ट्रक ड्रायव्हर, पोलिस यांच्यापर्यंतच मर्यादित आहे. ' टिनेन्जर्स ' यापासून काहीसे दुर्लक्षितच आहेत. मुंबईत साधारणपणे 33 टक्के तरुण लग्नाआधी ' सेक्स ' अनुभवत असल्याचं मुंबई जिल्हा एड्स कण्ट्रोल सोसायटीच्या निरीक्षणात दिसतं. यावरून या प्रश्नाचं गांभीर्य लक्षात यावं. म्हणूनच तरुणांसाठी तरुणांच्या भाषेत बोलणारं नाटक लिहिण्याचा निर्णय घेतला, असं नाईक यांनी सांगितलं.

कॉलेजांबरोबरच सामाजिक संस्थांमध्ये या नाटकाचे 24 प्रयोग झालेत. तीन मार्चला युनिव्हसिर्टीच्या कलिना कॅम्पसमध्ये नाटकाचा 25 वा प्रयोग होत आहे. ' एड्स कसा होतो ' हे सांगण्याऐवजी तरुणांमध्ये ' अवेअरनेस ' निर्माण करण्यावर नाटकाचा भर आहे. पिढ्या-पिढ्यांमधील हे अंतर मिटवण्यासाठी पालक-विद्यार्थ्यांदरम्यान संवाद हवा. त्यासाठी पालकांनाही ' माइण्डसेट ' कसा बदलायचा, याची जाणीव नाटकात करून देण्यात आली आहे.

समाजात ' टॅबू ' असलेल्या विषयावर काम करताना मिळालेला ' रिस्पॉन्स ' चांगला होता, असं या नाटकात काम करणारी ऋचा पाटकर सांगते. तिच्या मते हे नाटक मुलांबरोबरच पालकांनीही पाहायला हवं. नाटकादरम्यान या विषयावर तरुण मोकळी चर्चा करतात. नाटक तरुणांना ' अपील ' होत असल्याचं लक्षात येतं, तेव्हा कष्ट सार्थकी लागतात, असं मीना नाईक यांनी सांगितलं.
