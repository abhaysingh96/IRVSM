महामुंबई: मुंबई टाइम्स
नाटकांचा 'रिमिक्स' अवतार!
[ Monday, February 21, 2005  05:42:11 pm]


अमित भंडारी

' ओल्ड वाइन इन न्यू बॉटल ' नेहमीच बाजारात खपते आणि तिची मागणीही वाढते. नव्या नाटकांच्या बुकिंगला उतरती कळा लागत असल्यामुळे निर्माते आणि दिग्दर्शकांनी जुन्याच नाटकांना नव्या स्वरुपात सादर करायचं ठरवलंय. नव्या नाटकांनी बुकिंग खिडकीवर प्राण सोडण्याऐवजी जुन्या नाटकांची लोकप्रियता एन्कॅश करण्याची क्लृप्ती नाटकवाले अवलंबताना दिसत आहेत.

पुनरुज्जीवित नाटकांची संख्या झपाट्यानं वाढतेय. बुकिंगचा यशस्वी फॉर्म्युला असल्यामुळे जुन्या नाटकांना पसंतीही मिळतेय. त्यामुळे प्रेक्षक नाटकाकडे पाठ फिरवत नाहीत, असं बुकिंगचा आकडा सांगतो. नव्या नाटकांचं बुकिंग आठवड्याभरात साधारण पाच ते दहा हजारांच्या घरात जातं आणि वीकएण्डला त्यात सुमारे पाच हजारांची भर पडते. पण नव्याने रंगभूमीवर आलेल्या जुन्या नाटकांचं बुकिंग नऊ ते 22 हजार रुपयांनी सुरू होतंय आणि वीकएण्डला 20-22 हजार रुपयांचा आकडा ओलांडतंय. विनोदी नाटकांना तर एखादा शनिवार वा रविवारी ' हाऊसफुल्ल ' चा बोर्ड पाहायचं भाग्यही मिळतंय. या जुन्या नाटकांनी नेपथ्य, प्रकाश, संगीत आणि निमिर्तीवर जास्त खर्च करण्याचं टाळल्याचं प्रकर्षानं जाणवतंय. त्यामुळे प्रेक्षकांच्या शाबासकीसोबत आथिर्क नफ्याचं समीकरणही उमगलं आहे.

' आपलं बुवा असं आहे ' या नाटकावर आधारित ' हम तो तेरे आशिक है ' या नाटकाच्या सादरीकरण आणि पुनर्लेखनामुळे त्याचं रूपच बदललं आहे. ' श्रीमंत दामोदरपंत ' हे नाटक ' बापाचा बाप ' या नावाने हिट जातंय. विनय आपटेंच्या ' तुमचा मुलगा करतो काय ' ने बऱ्यापैकी गल्ला जमवलाय. ' सुयोग ' ने ' असा मी असामी ' पुन्हा रंगमंचावर आणलंय. तसंच ' तू फक्त हो म्हण ' पुन्हा गदीर् खेचतंय. '... अहो, प्रेमात सगळंच माफ ' हे त्या वेळी आलेल्या ' लैला ओ लैला ' या नाटकापेक्षा जास्त गदीर् खेचतंय. ' अधांतर ' जुन्या संचातल्याच कलाकारांसोबत अवतरल्यामुळे त्यालाही अपेक्षित प्रतिसाद मिळाला. गिरणी कामगारांसाठी झटणाऱ्या संघटनेच्या मदतीसाठी ते नव्याने रंगभूमीवर आलं. तसंच ' तन- मन ' हे एड्सवरील रत्नाकर मतकरींचं नाटकही पुन्हा नव्याने आलं ते एड्स जिल्हा नियंत्रण कक्षाच्या मदतीने!

एकांकिकांवर आधारित नाटकांनाही चांगला प्रतिसाद मिळतोय, हे ' यदाकदाचित ' हे नाटक सिद्ध करतंच आहे. तसंच गोपाळा रे गोपाळा, लाली- लीला, आयला हो रे ही नाटकंही बुकिंगवर यशस्वी लढाई लढत आहेत. मराठी प्रेक्षक खऱ्या अर्थाने रंगभूमीकडे वळतोय, हे महत्त्वाचं!
