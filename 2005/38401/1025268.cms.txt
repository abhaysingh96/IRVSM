लेख: नोंद
मृणालिनी गडकरी
[ Friday, February 18, 2005  08:51:43 pm]


अनुवादासाठी असणारा साहित्य अकादमीचा पुरस्कार मृणालिनी गडकरी यांना जाहीर झाला आहे. हा पुरस्कार त्यांना शरच्चंद चटर्जींच्या गाजलेल्या ' देवदास ' च्या मराठी अनुवादासाठी मिळाला आहे.

' देवदास ' शरदबाबूंनी 1894 मध्ये वयाच्या अवघ्या 17-18 व्या वषीर् लिहिली, पण ती प्रसिद्ध झाली ती 1917 च्या सुमारास! मामा वरेरकर आणि शरदिनी देशमुख यांच्यानंतर श्रीमती गडकरी यांनी ' मेहता पब्लिशिंग हाऊस ' साठी ' देवदास ' 2002 मध्ये पुन्हा अनुवादित केला.

जयवंत देशमुख यांच्याकडे बंगाली शिकलेल्या श्रीमती गडकरी यांनी ' रवींदनाथ टागोर आणि मराठी समकालीन भाषेचा तौलनिक अभ्यास ' यावर पुणे विद्यापीठाची पी. एचडी. मिळवली. साहित्यिक सुभाष भेंडे यांनी ठेवलेला जी. ए. कुलकणीर् पुरस्कारही त्यांना तीन वर्षांपूर्वी मिळाला होता.

गडकरी यांनी यापूर्वी शरदबाबूंच्या तसेच रवींदनाथ टागोरांच्या कलाकृतींचेही अनुवाद मराठीत आणले आहेत. शरदबाबूंचे ' अंधारे आलो ', ' स्वामी ', ' निष्कृती ' या कलाकृती त्यांना भावल्यामुळे त्यांचेही अनुवाद करणे हा एक आनंद देणारा अनुभव होता.

तस्लिमा नसरिन यांच्या ' आभार मे मेला ' ( माझं कुँवरपण) या पुस्तकाच्या अनुवादाबद्दल श्रीमती गडकरी यांना तीन वर्षांपूवीर् जी. ए. कुलकणीर् पुरस्कार मिळाला होता. सध्या त्या बंगाली लेखक सुनील गंगोपाध्याय यांच्या ' शेई समय ' ( तो काळ) या कादंबरीचा अनुवाद करत आहेत.

मुंबईत जन्म झालेल्या गडकरी यांचे शिक्षण मात्र पुण्यात झाले. बी.ए. होताना त्या जर्मन भाषाही शिकल्या. पुढे पुणे विद्यापीठात मराठीतून एम. ए. करताना त्यांनी ' टागोर आणि तत्कालीन मराठीतील निसर्गकाव्य ' हा एम. फिल.साठी त्यांचा विषय होता. शरदबाबूंना काहीसे जवळचे आपल्याकडे राम गणेश गडकरींसारखे लेखक होऊन गेले,

शरदबाबूंच्या प्रत्येक लिखाणामागे माणूस हाच केंदस्थानी असून त्याच्या मनाच्या तळाचा शोध घेतल्यामुळेच या पुस्तकाची दखल साहित्य अकादमीला घ्यावी लागली, असे त्या आवर्जून सांगतात.
