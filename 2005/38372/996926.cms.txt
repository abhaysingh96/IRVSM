लेख: जाता जाता
उत्तरेचा तडाखा
[ Thursday, January 20, 2005  10:48:18 pm]


' बापरे, काय ही थंडी! काही सुचत नाही काय करावं! ' स्वेटर घातलेल्या चकोरीने वर शाल गुंडाळत, पाय खुचीर्वर घेत, स्वगत म्हटलं.

' याला उपाय आहे. ' ज्यु. चकोरने सुचवलं.

' नको! ' चकोरीने मान हलवत आणि तोंडानेही त्याची कल्पना काय असणार हे ओळखून त्याचा मुद्दा मोडीत काढला.

' पण मी काय सांगतो ते ऐकून न घेताच तू नकाराधिकार कसा काय वापरलास ?' मुलाने वकिली थाटात विचारलं.

' तू सांगणार की उकळता चहा आणि गरमागरम पोहे किंवा साबुदाण्याची खिचडी यांचा आस्वाद घ्यावा. पण मला या थंडीत इतके कष्ट सकाळी सकाळी घेण्याची अजिबात इच्छा नाही. ' चकोरीने हसत हसत स्पष्ट केलं.

' खरं सांगू का आई, गोधडीत गुडुप झोपावं अशी माझी कल्पना आहे. आमची अजिबात इच्छा नाही की तुला या वातावरणात त्रास द्यावा. पण आता तू विषय काढून आमची जीभ चाळवली आहेस. ' ज्यु. चकोरीने अत्यंत गंभीर स्वरात सांगितलं.

' तुम्हाला मध्यरात्रीही, स्वप्नातही असे पदार्थ दिसतात. पण मला इतकं त्राण नाही आत्ता. ' चकोरीने ठामपणे सांगितलं.

' काय हो तुम्ही काही बोलत नाही! ' चकोरीने म. टा. वाचणाऱ्या चकोरला विचारून विषय बदलण्याचा प्रयत्न केला.

' अशी गार चर्चा करून तोंडाची वाफ दवडणं योग्य नाही. ' चकोरने खुलासा केला.

' ही थंडीची लाट उत्तरेकडून का येते समजत नाही! ' चकोरीने पुन्हा विषयाला फाटा फोडला.

' हिमालय उत्तरेकडे आहे म्हणून! ' मुलाने उत्तर दिलं.

' मुंबईत गेल्या काही वर्षार्ंत आलेल्या सगळ्या लाटा उत्तरेकडूनच आल्या आहेत. ' चकोरच्या या विधानावर तिघांनी, एकमेकाकडे बघून ' याचा अर्थ काय ' अशी विचारणा केली.

' मुंबईत येणारे लोंढे उत्तरेकडूनच येत आहेत ना! ' चकोरने आपला मुद्दा स्पष्ट केला.

तिघांनी चकोरला दाद दिली.

' हो, पण ही लाट फार काळ टिकत नाही. लगेच ओसरते. ' चकोरीने शाल आणखी घट्ट गुंडाळत भाष्य केलं.

' उत्तरेच्या थंड लाटांना तोंड देण्यासाठी ओट्याकडून गरम गरम लाटा कधी यायला सुरुवात होणार आहे! ' चकोरने जांभई देत विचारलं.

ज्यु. कंपनीच्या आशा पल्लवीत झाल्या.

पण या प्रश्ानवर चकोरी आणखी गारठली.

काही क्षणांच्या शांततेनंतर चकोरी उठली.

' काम करणं हाच थंडीला तोंड देण्याचा उत्तम मार्ग असतो. ' तिने मुलांकडे बघत सांगितलं आणि गॅस पेटवला.

चकोर
