लेख: जाता जाता
लेखणी मोडा आणि...
[ Thursday, January 27, 2005  11:38:22 pm]


' कोण मेश्राम का ? मी खरे बोलतोय. शुभेच्छा देण्यासाठी फोन केलाय. '

' सुरुवातीला वाटलं त्यापेक्षा हा मामला फारच गुंतागुंतीचा दिसतोय. बोकड कापून झालाय. आता कोणाला कापतात बघायचं. तुमची काय स्थिती आहे ? अजून हवा तापली नाही तुमच्या गावात! '

' काही जणांनी शं.नां.च्या आडून बाण मारण्याचा प्रयत्न केला, पण सुदैवाने श.ना. चतुर निघाले. त्यांनी कोणत्याही वादात पडायचं नाही असं ठरवलेलं दिसतं आहे. सुटलो त्यामुळे आम्ही. '

' आमच्याकडे राष्ट्रवादी आणि बिगरराष्ट्रवादी असे गट पडले आहेत. पण राष्ट्रवादीवाले पॉवरफुल आहेत. मॉबची काळजी करू नका, दणक्यात भाषण करा असं सगळेजण सांगून जाताहेत मला. '

' सरप्रायझिंगली आमच्याकडे अजून पॉलिटिक्स आलं नाही. मच्छिंदपण गप्प आहे. त्याच्यातला कोकणी बाणा जागा झालेला नाही हे नशीब. '

' मला पूवीर् वाटायचं की मुंबई, पुण्याचे लोक जास्त चतुर असतात. पण इथे, गोदावरीकिनारी बघतो तर एकाहून एक बेरके लोक आहेत. सतत, साहेबांना विसरू नका, त्यांचं नाव मेन्शन करा असं सांगून जातात. '

' कोण साहेब ?'

' इथे साहेबांचं स्नेहसंमेलन भरलं असावं इतके साहेब आहेत. देव जाणे कोण साहेब यांना अभिप्रेत आहेत. '

' मग काय करणार तुम्ही ?'

' साहेबांचं साहित्यप्रेम आणि संमेलनप्रियता याबद्दल बोलणार नाव न घेता. सगळ्या साहेबांचे चेले खुश. '

' गेल्या वषीर् साहेबांनी एकाच दगडात, भांडारकरवर दगड मारणारे आणि विद्वान यांना सुखावणारं भाषण केलं होतं. यंदा काय करतील असं वाटतं ?'

' यंदा कार्यर्कत्यांचा मेळावा आहे असं त्यांना वाटलं नाही म्हणजे नशीब. नाहीतर साखर, जागतिक व्यापार अशा विषयावर बोलतील. साहित्यवादींपेक्षा राष्ट्रवादींची गदीर् जास्त होणार असं दिसतं आहे. '

' पण तसं झालं तर साहेबांच्या भाषणानंतर सगळे निघून जातील. तुमचं भाषण ऐकणार कोण नंतर! '

' खरे, तुमची स्थिती पण माझ्यापेक्षा वेगळी असेल असं मला वाटत नाही. '

फोनवर मोठा सुस्कारा सोडल्याचा आवाज.

' खरं आहे तुमचं म्हणणं. सगळ्यांचे डोळे बिग बी आणि मिथुनकडे लागले आहेत.

' मग तुम्ही काय सांगणार तुमच्या भाषणात ?'

' हिंदी नाट्य आणि चित्रसृष्टी किती थोर आहे हे सांगावं असा विचार आहे. '

' पण मेश्राम तुमचं काय! '

' साहित्यिकांनी लेखण्या मोडून ट्रॅक्टर हाती घ्यावा व साहेबांचे हात बळकट करावेत असं मी आवाहन करणार आहे. '

फोनवर दोन्ही बाजूंनी केविलवाणं हसण्याचा आवाज.
