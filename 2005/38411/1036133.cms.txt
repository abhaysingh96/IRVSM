लेख: जाता जाता
...मोडणार नाही
[ Monday, February 28, 2005  08:45:53 pm]


' अरे राजसाहेब यायचा टाईम झाला. कॅसेट लाव पटकन! अरे ' मराठी पाऊल पडते पुढे... '

नको , ' चढती जवाँनी... ' पाहिजे यार. तुला बोललो ना आता नवीन रिमिक्सचा स्टॉक ठेव. '

' पण गेली चाळीस वर्षं साहेब हीच कॅसेट मी लावतो आहे. शिवजयंती असो, नाहीतर शाखेचा वर्धापनदिन. मोठ्या आणि धाकट्या साहेबांच्या वाढदिवसाला पण हीच गाणी वाजवतो साहेब आपण. '

' जादा शाणपत्ती करू नको. आज कोणते साहेब येणार आहेत, हे तुला बोललो होतो ना! '

' पण हे साहेब येण्याचा आणि रिमिक्सची कॅसेट लावण्याचा काय संबंध आहे ? शाखा स्थापन केली तेव्हापासून आपल्याला आदेेश आहे ही कॅसेट लावायचा. आपल्या विजयादशमी मेळाव्यात साहेब आपण हीच सप्लाय करतो ना! '

' अरे किती बोलतो रे तू! आता आपला ' राजा ' वेगळा आहे हे ध्यानात घे. त्याला वेगळी कॅसेट लागते. त्याच्या वयाला शोभेल अशी. '

' फार काय ट्यूब पेटली नाही आपली, पण आपले बांद्याचे साहेब पण आपल्या साहेबांच्या एजचेच आहेत ना! मग त्यांना कोणती कॅसेट लावायची ?'

' अजून त्यांनी भाषणात काय आपल्याला आदेश दिलेले नाही. '

' हल्ली छोटेसाहेब आपल्याला भाषणातून आदेश द्यायला लागले काय! कोणत्या पेपरातून ? आपल्या पेपरात पहिल्या पानावर काय आदेश दिसला नाय. पूवीर् शाखेवर बोर्ड रंगवायचो आपण आदेशांचे. याद आहे ना तुम्हाला! आपल्याला रातच्या टायमाला वडापाव तुम्हीच खिलवायचा आणि नंतर कटींग चाय. इलेक्शनच्या टायमाला बिर्याणी. '

' अरे बाबा, तो टाईम अलग होता. साहेबांशी डायरेक्ट आपला काँन्टॅक्ट असायचा. केव्हाभी जा, साहेब प्रेमाने बोलायचे, घरच्यांची चौकशी करायचे. आपल्या बापाला नावाने ओळखायचे. बोलायचे ढाण्या वाघ होता तुझा बाप! '

' पण लास्ट टायमाला कॉपोर्रेशनच्या इलेक्शनला वाघाच्या बछड्याला तिकीट नाय भेटलं. बंगल्याबाहेर किती तास आपण उभे होतो याद हाय ना! '

सुस्कारा सोडून , ' छोड यार! आपलं बॅडलकच खराब! त्या पक्याने सॉलीड फिल्डींग लावली. आपल्या एमएलएने पण दगा दिला. '

अचानक मोटारींचा आवाज, पोलिसांच्या शिट्यांचा आवाज. यामध्ये ' आवाज कोणाचा! ' ही घोषणा विरून जाते.

' ए, चल काँटा लगा...लाव. '

माईकवरून घोषणा होते. ' कोणीही ताठ बसायचं नाही. सगळ्यांनी वाकून बसायचं. '

हे ऐकून छोटे साहेब कपाळावर हात मारतात.

चकोर
