लेख: जाता जाता
सरकार काय करतं!
[ Thursday, February 17, 2005  01:03:51 am]


' काय अवस्था आली आहे आपल्यावर! ' चकोरीने म. टा.वर ओझरती नजर टाकली आणि कॉमेंट केली.

' बरं चाललं आहे की सगळं! लालू, विलासराव यांच्या भवितव्याचा आणि आपल्या अवस्थेचा काही संबंध नाही. तुझी लेकरं वकील, डॉक्टर नसल्यामुळे त्यांना सेवाकर भरावा लागणार नाही आणि महाराष्ट्रात आणखी आय.आय.एम. आली किंवा न आली तरीही तुला काही घेणंदेणं नाही. मग कशाला उसासे टाकतेस ?' चकोरने ऐतिहासिक नाटकातला डायलॉग मारत पेपरच्या संपूर्ण पहिल्या पानाचं अॅनॅलिसीस केलं.

' बरोब्बर विमानतळाची बातमी तुमच्या नजरेतून सुटली म्हणा किंवा तुम्ही टाळलीत. ' ' ती बातमी पण वाचली आहे मी. पण आपण कशाला जातो आहे एअरपोर्टच्या वाटेला! काही म्हणा, शरद पवारांनी लक्ष घातलं नाही तर प्रश्ान् काही सुटत नाही. ' चकोरने मत मांडलं.

' लाडक्या शरदरावांचं नाव घेतल्याशिवाय तुम्हाला चेैन पडत नाही काय ?' चकोरीने पुन्हा चकोरला रोखलं.

' काय बोअरिंग डिस्कशन करता सकाळी सकाळी! ' ज्यु. चकोरीने ट्रॅक बदलण्याची तयारी केली.

' मग काय कराव्वे बरे! ' चकोरने कधीकाळी पाहिलेल्या प्रायोगिक नाटकातल्या नटाची नक्कल केली.

' मॉम खायला काय आहे! ' तिने लाडिकपणे विचारलं.

' दुसरं काही सुचतच नाही यांना! ' चकोरीने वैतागून म्हटलं.

' सही! ' इतका वेळ शांतपणे चर्चा ऐकणाऱ्या ज्यु. चकोरने आईला दुजोरा दिला.

' काय रे तुला काय झालं इतकं खुष व्हायला ?' चकोरने आश्चर्याने विचारलं.

' आईला काहीतरी इंपॉर्टंट पॉईंट मांडायचा आहे, पण तो तुम्ही खाणं आणि शरदराव असल्या टॉपिकवर वेस्ट करताहात. आईने प्रोटेस्ट लॉज केल्याबद्दल तिचं अभिनंदन करतो आहे. ' त्याने आपली भूमिका मांडली.

चकोर गडबडला. आपण चकोरीकडे दुर्लक्ष केलं हे त्याला जाणवलं.

' बोला बाईसाहेब! ' कन्येनं नाटकी अदबीने तिला विनंती केली.

चकोरी काहीक्षण थांबली. वाजपेयी, दिलीपकुमार यांच्यासारखा मोठ्ठा पॉझ तिने घेतला.

' विमानतळ विमानांसाठी खुला, रेल्वे रुळांच्या बाजूची अनऑथॉराइज्ड बांधकामं पाडल्यामुळे गाड्या वेळेवर धावतात, पुलावरचे हॉकर्स हलवल्यामुळे प्रवासी आनंदात अशा बातम्या आपण वाचतो. ' इतकं बोलून ती थांबली.

' मग! ' चकोरने उत्सुकतेने विचारलं.

' मला कळत नाही की विमानतळ, रस्ते, रेल्वे यांच्यावर अशा बातम्या वाचायची वेळ येईपर्यंत आपलं सरकार काय करत असतं! ' तिच्या सवालावर सगळे विचारात पडले.

चकोर
