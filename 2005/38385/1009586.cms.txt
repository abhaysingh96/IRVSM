अर्थव्यवहार
खाजगी मोबाइल कंपन्या, 'बीएसएनएल' यांच्यात 'रोमिंग रेव्हेन्यू'वरून तंटा
[ Wednesday, February 02, 2005  11:17:24 pm]


वृत्तसंस्था

नवी दिल्ली : मोबाइल फोनधारकांकडून देशांतर्गत व देशाबाहेरून ' रोमिंग ' सेवेसाठी वसूल केल्या जाणाऱ्या उत्पन्नाच्या वाटपावरून सरकारी भारत संचार निगम लि. (' बीएसएनएल ') व खाजगी मोबाइल फोनसेवा कंपन्या यांच्यात तंटा उभा राहिला आहे.

खाजगी मोबाइल कंपन्यांची संघटना ' सेल्युलर ऑपरेटर्स असोसिएशन ऑफ इंडिया ' ने (' सीओएआय ') इंटरनॅशनल रोमर्सकडून वसूल केल्या जाणाऱ्या ' रोमिंग रेव्हेन्यु ' तील 50 टक्के हिश्श््याची मागणी ' बीएसएनएल ' कडे केली असून , ' पब्लिक सेक्टर ' ची ( ' बीएसएनएल ' ची) याबाबतची भूमिका अरेरावीची असल्याचा आरोप केला आहे.

याउलट , ' बीएसएनएल ' च्या अधिकाऱ्यांचे म्हणणे असे आहे की , ' बीएसएनएल ' व इतर ( खाजगी) सव्हिर्स प्रोव्हायडर्स यांच्यातील नेहमीच्या ' इंटरकनेक्शन अॅग्रीमेंट ' आणि मोबाइल फोन रोमर्सची ' ट्रफिक ' ( देशांतर्गत व देशाबाहेरील रोमिंग) या दोन स्वतंत्र गोष्टी आहेत.

' बीएसएनएल ' ने 1 फेब्रुवारीपासून लागू झालेल्या सुधारित चाजेर्ससह नवे ' इंटरकनेक्शन ' दर लागू करण्याबाबत गेल्याच आठवड्यात एक र्सक्युलर काढले आहे. मात्र खाजगी मोबाइल कंपन्यांशी नव्या ' इंटरकनेक्शन ' करारावर स्वाक्षऱ्या करणे ' बीएसएनएल ' ने एक आठवडा पुढे ढकलले आहे.

' ट्राय ' कडे साकडे

' सीओएआय ' ने टेलिकॉम उद्योगाची नियंत्रक संंस्था असलेल्या ' टेलिकॉम रेग्युलेटरी अर्थारिटी ऑफ इंडिया ' कडे (' ट्राय ') दाद मागितली असून, या प्रकरणी हस्तक्षेप करण्याची मागणी केली आहे. ' ट्राय ' चे सदस्य डी. पी. एस. सेठ यांनी प्रसारमाध्यमांशी बोलताना सांगितले की , ' सीओएआय ' ने मांडलेल्या गाऱ्हाण्यांवर ' ट्राय ' विचार करीत असून, त्याबाबत लवकरच निर्णय घेतला जाईल. या प्रकरणी ' बीएसएनएल ' ची बाजू समर्थनीय आहे का, असा प्रश्ान् विचारण्यात आला असता काहीही बोलण्यास त्यांनी नकार दिला. पण हा काही मोठा प्रश्ान् नाही, असे ते म्हणाले.

ग्राहकांच्या हिताला बाधा

' बीएसएनएल ' खाजगी सेल्युलर ऑपरेटर्सकडून अतिरिक्त ' कॅरेज चाजेर्स ' वसूल करण्याचा प्रयत्न करीत असून, हे ग्राहकांच्या हिताचे नाही, असे ' सीओएआय ' चे महासंचालक टी. व्ही. रामचंदन यांनी ' ट्राय ' ला सादर केलेल्या निवेदनात म्हटले आहे. ' ट्राय ' ने 6 जानेवारी रोजी लागू केलेल्या अधिसूचनेव्यतिरिक्त ' बीएसएनएल ' एकतफीर् वेगळे दर वसूल करू पाहत आहे. सर्कलअंतर्गत व वेगवेगळ्या सर्कल्सदरम्यान प्रति मिनिट 30 पैसे असा सार्वत्रिक आकार लागू असताना ' रोमिंग सबस्क्रायबर्स ' साठी स्वतंत्र ' इंटरकनेक्शन अॅग्रीमेंट ' ची आवश्यकता नाही, असे ' सीओएआय ' चे म्हणणे आहे.
