लेख: जाता जाता
खोटी लोकल
[ Wednesday, February 02, 2005  10:44:30 pm]


' काय चेष्टा करतात आमची! ' ज्यु. चकोरीने पित्यापुढे म. टा. भिरकावत तक्रार केली.

चकोरने पेपर चाळला. त्याला आपल्या कन्येची, तिच्या पिढीची चेष्टा केल्याचं काही दिसलं नाही.

' काय गं काय झालं तुला संतापायला! ' त्याने आश्चर्याने विचारलं.

' हा फोटो बघा ना! ' तिने खुलासा केला.

कोणत्या फोटोमुळे ती रागावली हे बघायला चकोरीने पेपरात डोकं खुपसलं. पण तिला काही बोध झाला नाही.

' एक्झाम जवळ आल्यामुळे ही असंच काहीतरी बोलत असते. ' ज्यु. चकोरने तिला चिडवलं.

' नको रे तिला त्रास देऊ! ' आईने मुलीची कड घेतली.

' मुली, काय झालं सांगशील का ?' चकोरने पित्याच्या मायेने विचारलं.

' हा फोटो बघा. आपल्या लोकलच्या लेडिजमध्ये असा सीन असतो का! ' तिचा गुस्सा कायम होता.

तिघांनी आता फोटो नीट बघण्यास सुरुवात केली.

' या मुली सतत काहीतरी खरेदी करत असतात. अगदी रियॅलॅस्टिक दाखवलं आहे. ' ज्यु. चकोरने संधी साधली.

' लेडिज कंपार्टमेंटमध्ये जेन्ट्स इथेपण आहेत. ' चकोरने त्याला पुस्ती जोडली.

' गप्प बसा जरा! ही काय म्हणते ते ऐका! ' चकोरीने दोघांना दटावलं.

' सांग बघू तुझी ऑब्जेक्शन्स काय आहेत. ' आईने मुलीला समजावलं.

' आपण लालूंना सांगू ती! ' ज्यु. चकोरने आपला बाणा सोडला नाही.

चकोरीने त्याच्याकडे फक्त बघितलं.

मग मुलीकडे. पण एका क्षणात तिच्या नजरेतले भाव बदललेले होते.

भावाचा पडका चेहरा बघून ज्यु. चकोरीचा चेहरा उजळला.

' हे बघ मॉम, लोकलच्या र्फस्टला पण पीक अवर्समध्ये किती रश असतो हे तुला माहीत आहे, इथे तर गुडी गुडी दाखवलं आहे. ' तिच्या ऑब्झवेर्शनवर चकोरीने मान हलवून सहमती दाखवली.

अॅक्च्युली, त्यांनी सेकंड क्लास कंपार्टमेंट घ्यायला हवा होता. आम्ही कशी मारामारी करतो आत जाण्यासाठी हे मस्त दाखवता आलं असतं सगळ्या व्हीव्हीआयपीना. प्रेसिडंेटना पण कळली असती आमची हालत. ' मुलीने समस्त महिला प्रवाशांची बाजू मांडली. आता चकोरही तिचं म्हणणं लक्षपूर्वक ऐकू लागला. ज्याला बोचतं त्यालाच वेदना कळतात हे त्याला जाणवलं.

' यावर उपाय काय! ' चकोरीने अत्यंत गांभिर्याने विचारलं.

' सेकंड लेडिजमध्ये काय अवस्था असते याचं फिल्मिंग करून सगळ्यांना पाठवा. ' ज्यु. चकोरने तितक्याच गंभीरपणे सुचवलं.

ज्यु. चकोरीने मान डोलावली.

चकोर
