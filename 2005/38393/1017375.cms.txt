लेख: जाता जाता
यंग लूक
[ Thursday, February 10, 2005  08:05:41 pm]


' आता ही जोडी महापालिकेत एकत्र लढणार म्हणजे! ' चकोरीने म. टा. पूर्ण चाळायच्या आतच कॉमेंट नोंदवली.

काहीतरी वेगळा मामला दिसतो आहे हे लक्षात येऊन बाकीच्या तिघांनी पेपर ताब्यात घेण्यासाठी प्रयत्न केले.

पण ज्यु. चकोर अधिक चपळ निघाला. त्याने चटकन नजर टाकून, बातमीचं सार्वजनिक वाचन केलं.

' लग्नाला सॉलिड गदीर् होणार यांच्या! ' चकोरने मत नोंदवलं.

' अॅक्च्युली तिथे कोणते कलर्स डॉमिनन्ट असणार हे बघणं इंटरेस्टिंग आहे. ' मुलीने वयानुरूप डायलॉग टाकला.

' म्हणजे! ' आईने विचारलं.

' अगं, शिवसेना म्हणजे सॅफरन असायलाच पाहिजे. शिवाय ' व्हॅलेन्टाईन डे ' म्हणजे गुलाबी रंगाची उधळण. '' मुलीने खुलासा केला.

' दोन्ही रंग असतील तिकडे! ' चकोरीने, सहजतेने सांगून टाकलं.

' यही तो गडबडका मामला है! ' तिने हिंदीचा आश्रय घेऊन परिस्थितीचं गांभीर्य मांडलं.

' गडबड कायका! सबको गडबड अभी मालूम पडा है! ' चकोरने हसत मल्लीनाथी केली.

' बाबा, गडबड अशी की व्हॅलेन्टाईन डेला या दोघांच्या पाटीर्ने अपोझ केलं आहे. मग गुलाबी रंग कसा चालेल पाटीर् लिडर्सना! ' मुलीने शंका काढली.

' गुलाबी रंग, प्रेमाचं सिंबॉल आहे. आता त्याच दिवशी त्यांनी मुहूर्त शोधला हा सोयीचा भाग झाला. ' चकोरने सांगितलं.

' तिच्या व्होटर्सचं काय! ' चकोरीने चर्चा एकदम प्रॅक्टिकल लेव्हलला नेली.

' हा बोअर विषय आहे. आपण तो नंतर डिस्कशनला घेऊ. ' चकोरने जाहीर केलं.

' त्या दिवशी मेन्यूमध्ये पण या रंगांना प्रेफरन्स असेल. जिलबी, केशरी भात, गाजर हलवा... ' चकोर आवडत्या विषयावर आला.

' कॅटररची टेस्टच आहे असा मेनू निवडायचा म्हणजे! ' ज्यु. आता रिंगणात आला.

' नीलेश लिमये, संजीव कपूर वगैरे सुपरशेफना बोलवायला हवं हे ठरवायला. ' चकोरीने सुचना केली.

' तू काय कमी आहेस का! अनेक पदार्थ तुला सांगता येतील. ' चकोरने तिचं कौतुक केलं.

' काही नाही तर या दोन रंगांचे पापड आणि कुरड्या नक्की. ' मुलीने आईची चेष्टा केली.

' स्ट्रॉबेरी, मँगो आईस्क्रीम, शेक यांचा कलरपण मॅच होईल या ऑकेजनला. ' मुलाला चटकन आठवलं ते त्याने सांगितलं.

' तुम्हाला खाण्यापिण्याशिवाय काही सुचत नाही काय! ' चकोरने गंभीरपणाचा आव आणला.

यावर तिघेही खदाखदा हसले.

मुलीने ' बाबा तुम्ही सांगता हे! ' अशा नजरेने पाहिलं.

' माझ्या दृष्टीने महत्त्वाचं आहे कॉपोर्रेशन आणि पॉलिटिक्स यंग होत चाललं आहे. गुड फॉर कंट्री. ' चकोरीने निष्कर्ष काढला.

चकोर
