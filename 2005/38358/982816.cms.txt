अर्थव्यवहार
नवीन वर्षातला कार धमाका
[ Thursday, January 06, 2005  09:05:15 pm]


म. टा. वाहन प्रतिनिधी

मुंबई : कार उत्पादक आणि ग्राहक अशा दोघांसाठीही 2005 हे वर्ष पर्वणीचे ठरणार आहे. कारण मारुती उद्योगपासूून पार जनरल मोटर्सपर्यंत सगळ्याच कंपन्या विविध मॉडेल्स या वषीर् बाजारात आणत आहेत. भारतात एकाच वषीर् इतक्या मोठ्या संख्येने कार मॉडेल्स बाजारात येण्याची ही पहिलीच वेळ आहे. विशेष म्हणजे जवळपास प्रत्येक सेग्मेंटमध्ये ही मॉडेल्स येत आहेत. बड्या परदेशी कंपन्यांना भारतातली बाजारपेठ किती महत्त्वाची वाटते हेच यातून दिसून येते.

मारुती उद्योग लिमिटेड ' स्विफ्ट ' हे मॉडेल आणत आहे. ' आल्टो ' या मिनी-बी सेग्मेंटमधील कारचेच हे भावंड असले, तरी ' आल्टो ' पेक्षा अधिक स्टाइल आणि वैशिष्ट्य ' स्विफ्ट ' मध्ये असल्याचे बोलले जाते. प्रिमियम हॅचबॅक (डीक्कीशिवाय!) सेग्मेंटमध्येही गदीर् होण्याची चिन्हे आहेत. ' गेट्झ ', ' पालियो ' अशी या सेग्मेंटमधली मॉडेल्स लोकप्रिय आहेत. किफायतशीर, आरामशीर आणि तरीही लक्झरी अशा सर्व मागण्या पुरवणाऱ्या सेग्मेंटकडे अनेक कंपन्यांचे लक्ष गेले आहे. विशेष म्हणजे हे सेग्मेंट लवकरच मिनी-बीलाही मागे टाकेल, अशी चिन्हे आहेत.

स्पोर्ट््स युटिलिटी सेग्मेंटमध्ये ( एसयूव्ही) सुझुकी सुधारित ' ग्रँड व्हिटारा ' बाजारात आणत आहे. होंडाच्या ' सीआर-व्ही ' मॉडेलशी टक्कर घ्यायची त्यांची योजना आहे. याच स्पधेर्त ह्युंदै ही कंपनीही उतरली आहे. ते ' टुस्कन ' हे मॉडेल आहेत. जनरल मोटर्स ' डेवू माटिझ ' पुन्हा आणत आहेत. ' शेवरोले स्पार्क ' या नावाने ती ओळखली जायची. ' कॅलोस ' ही त्यांची आणखी एक कार येत असून , ' इंडिका ' शी तिची थेट स्पर्धा असेल, असे बोलले जाते.

प्रिमियम सेग्मेंटमध्ये स्कॉडा कंपनी ' फेबिया ' आणि नवी ' ऑक्टाव्हिया ' मॉडेल्स आणत आहे. विशेष म्हणजे सध्याची ' ऑक्टाव्हिया ' ते कायम ठेवतील. टोयोटाच्या बलाढ्य ' कोरोला ' ब्रँडशी टक्कर घेण्यासाठी होंडाने ' सिव्हिक ' वर लक्ष केंदित केलं आहे. युुुटिलिटी व्हेइकल सेग्मेंटमध्ये टोयोटा ' इनोव्हा ' लॉँच करत आहेत. तर फियाट ' पांडा ' आणि ' मल्टिप्ला ' ही दोन मॉडेल्स आणत आहे.
