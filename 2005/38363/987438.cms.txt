अग्रलेख
लाल सलाम!
[ Tuesday, January 11, 2005  07:29:23 pm]


केवळ मराठीच नव्हे, तर भारतीय चित्रपटसृष्टीत चैतन्य आणणारा '' श्वास ' आज ' ऑस्कर ' च्या उंबरठ्यावर उभा असताना सेल्युलॉइडच्या 35 मि. मी.च्या पट्टीवर ' समांतर जग ' चित्रित करणारे ख्यातकीर्त दिग्दर्शक मृणाल सेन यांना दादासाहेब फाळके पुरस्काराने सन्मानित करण्यात येत आहे. हा योगायोग खचितच नाही.

भारतात गेल्या शतकात फाळके यांच्यापासून व्ही. शांताराम, पी. सी. बारुआ, मेहबूब खान यांच्यापर्यंत आणि शाम बेनेगल, मणी कौल यांच्यापासून एम. एस. सत्थू, गोविंद निहलानी, सईद मिर्झा यांच्यापर्यंत अनेक बुजुर्ग दिग्दर्शक होऊन गेले.

राज कपूरपासून बी. आर. चोप्रांपर्यंत अनेकांनी बॉलीवूडचे नाव रोशन केले. पण मृणाल सेन यांचे नाव, स्वातंत्र्यानंतर भारतात नवचित्रपटांची मुहूर्तमेढ रोवणारे सत्यजित राय, ऋत्विक घटक अशा प्रतिभाशाली दिग्दर्शकांबरोबरीने घ्यावे लागते.

मुळात भारताला समांतर वा नवसिनेमाची ओळख करून देण्याचे काम 1952 मध्ये झालेल्या पहिल्या आंतरराष्ट्रीय चित्रपट महोत्सवाने केले. या महोत्सवात दाखविल्या गेलेल्या ' बायसिकल थिफ ' या विट्टोरिओ डी ' सिकाच्या चित्रपटाने अनेकांचे डोळे उघडले.

खरे तर त्यापूवीर्च आलेल्या राज कपूरच्या ' आवारा ' या चित्रपटातून सामाजिक वास्तवाचे भान भारतीय दिग्दर्शकांना आल्याचे दिसू लागले होते. पण या महोत्सवानंतर तीन वर्षांनी सत्यजित राय यांचा ' पाथेर पांचाली ' आला आणि भारताचे नाव आंतरराष्ट्रीय स्तरावर गेले.

त्या आसपासच ऋत्विक घटक आणि मृणाल सेन हे चित्रपट निमिर्तीच्या तयारीत होते. घटक यांचा ' नागरिक ' हा ' पथेर पांचाली ' च्या आधी आला होता आणि मृणालदांचा ' रातभोर ' हा पहिला चित्रपट त्यानंतर आला. पण त्यांचे नाव खऱ्या अर्थाने रोशन झाले, ते 1969 मध्ये आलेल्या ' भुवनशोम ' चित्रपटामुळे.

1970 च्या दशकात भारतात ' न्यू वेव्ह ' चित्रपटांची जी लाट उसळली, तिचा हा सिनेमा पायाचा दगड होता. अर्थात गुरूदत्तच्या ' कागज के फूल ' सारख्या चित्रपटांनी भारतीय दिग्दर्शकांच्या प्रतिभेवर शिक्कामोर्तब केलेच होते. 1969 मध्येच आलेला मणी कौल यांचा ' उसकी रोटी ' आणि पुढे दोन वर्षांनी आलेला ' आषाढ का एक दिन ' तसेच 1973 चा शाम बेनेगल यांचा ' अंकुर ' अशा चित्रपटांनी 1970 चे ते ' अस्वस्थ दशक ' भारावून गेले होते. पण मुळात मृणालदांचा सिनेसृष्टीशी काही संबंध येण्याचे कारणच नव्हते.

1923 मध्ये मृणाल सेन यांचा जन्म आजच्या बांगलादेशामध्ये असलेल्या फरीदपूर गावात झाला आणि पुढे महाविद्यालयीन शिक्षणासाठी ते कोलकात्याला गेले. तेथून पुढे त्यांच्या आयुष्याने जी काही वळणे घेतली, त्यावरच खरे तर एखादा वास्तववादी चित्रपट निघायला हरकत नसावी! पदार्थविज्ञानाचा अभ्यास करताना त्यांचा संबंध कोलकात्यातील कम्युनिस्ट पक्षाच्या ' कल्चरल विंग ' शी आला आणि त्यांच्या कलाविषयक जाणिवा तर प्रगल्भ झाल्याच ; शिवाय डाव्या विचारांचे संस्कारही त्यांच्यावर होत गेले.

' इंडियन पीपल्स थिएटर ' शीही ( इप्टा) त्यांची नाळ याच सुमारास जोडली गेली. पण तेव्हाही आपण आयुष्यात कधी सिनेमा काढू, असे त्यांना वाटले नव्हते. शिक्षण पूर्ण झाल्यावर त्यांनी मेडिकल रिप्रेझेंटेटिव्ह म्हणून नोकरी पत्करली. पण त्यात त्यांचे समाधान होणे शक्यच नव्हते. अखेरीस त्यांनी फिरतीची वणवण सोडून पुन्हा कोलकात्यात बस्तान बसविण्यासाठी एका फिल्म स्टुडिओत ' ऑडिओ टेक्निशियन ' म्हणून काम सुरू केले.

त्याच सुमारास ' रातभोर ' ची बीजे त्यांच्या मनात अंकुरत असावीत. पुढे ' नील अक्षर निचे ' या चित्रपटाने त्यांचे नाव सिनेसृष्टीत चचिर्ले जाऊ लागले आणि नंतरच्या ' बाइशे श्रावण ' या चित्रपटाने त्यांचे नाव एकदम आंतरराष्ट्रीय स्तरावर गेले. त्यानंतर वर्षा दोन वर्षाच्या अंतराने त्यांनी ' पुनश्च ', ' प्रतिनिधी ', ' आकाशकुसुमे ' असे काही सिनेमा काढले आणि 1969 मध्ये आला ' भुवन शोम '. भुवन शोम हा चित्रपटाचा नायक. एकाकी विधुर. आपल्या नेहमीच्या चाकोरीबद्ध जीवनातून एक दिवस ' ऑफ ' घेऊन तो खेड्यात जातो.

बैलगाडी चालवतो. गाई- म्हशींबरोबर वेळ काढतो. तिथे त्याला गौरी नावाची एक ' गाँव की गोरी ' भेटते... पण अखेरीस आपल्या चाकोरीबद्ध जीवनातून आपण बाहेर पडू शकत नाही, हा साक्षात्कार त्याला होतो. एवढीच या चित्रपटाची कथा. पण उत्पल दत्त आणि सुहासिनी मुळ्ये यांना घेऊन, मृणालदांनी त्यातून मानवी मनोविकारांचे थक्क करून दाखविणारे दर्शन घडविले आहे.

' एक अधुरी कहानी ', ' मृगया ' आणि ' खंडहर ' हे त्यांचे हिंदीतील आणखी काही चित्रपट. या सर्वातून मृणालदांची चित्रपट माध्यमावरची पकड जशी दिसून येत त्याचबरोबर सामाजिक जाणिवांचे त्यांना असलेले भानही दृग्गोचर होत जाते. त्याला कारण अर्थातच त्यांच्यावर तरुणपणी झालेले साम्यवादाचे संस्कार यात शंका नाही. फाळके पुरस्काराच्या निमित्ताने मृणालदांना लाल सलाम!
