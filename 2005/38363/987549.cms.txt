अर्थव्यवहार
तुमच्याकडे 'पॅन' कार्ड आहे काय?
[ Tuesday, January 11, 2005  08:53:47 pm]


नोकरदार, व्यावसायिक किंवा आथिर्क उलाढाल करणाऱ्या व्यक्ती, संस्था तसेच कंपन्यांना ' पॅन ' कार्डचे महत्व ठाऊक आहे. ' पॅन ' शिवाय कुठलाही मोठा व महत्वाचा आथिर्क व्यवहार होत नाही. त्यामुळे नवीन नोकरी मिळालेले किंवा व्यवसाय सुरू करणाऱ्यांना लगेचच ' पॅन ' मिळवणे गरजेचे आहे. आता सरकारनेदेखील ' पॅन ' सक्तीचे केले आहे.

आथिर्क व्यवहारांत पारदर्शकता येण्यासाठी ' पॅन ' चे महत्व वाढत असले तरीही अनेकांना ' पॅन ' हा काय प्रकार आहे, असा प्रश्ान् पडतो. ' पॅन ' म्हणजे ' परमनंट अकाउंट नंबर '. सरकारने, विशेषत: प्राप्तीकर विभागाने, कर भरणाऱ्यांसाठी सुरू केलेली ही सुरळीत पद्धत. यात प्राप्तीकर विभागाकडे आपले निवास, जन्मतारीख आदींच्या पुराव्यासह अर्ज करून ' पॅन ' कार्ड घ्यायचे.

घर किंवा इतर कोणतीही स्थावर मालमत्ता घेताना आणि विकताना, कर्जासाठी अर्ज करताना, प्राप्तीकर भरताना एवढेच नव्हे तर बँकेत मोठ्या रकमेचा डिमांड ड्राफ्ट काढायचा असला तरी ' पॅन ' लागतो. परदेशी जाताना, बँकेत खाते उघडताना अशा अनेक बाबींसाठी सरकारने ' पॅन ' सक्तीचे केले आहे. म्हणूनच प्रत्येक कमावत्या व्यक्तीकडे ' पॅन ' कार्ड असणे आवश्यक आहे. आथिर्क बाबींव्यतिरिक्त ' पॅन ' कार्ड हा शासकीय व प्रशासकीय कामकाजासाठी महत्वाचा पुरावादेखील मानला जातो.

' पॅन ' मिळवण्यासाठी पूवीर् आयकर कार्यालयात उपलब्ध असणारा ' फॉर्म 49 ए ' भरून द्यावा लागत असे. मात्र आता ' पॅन ' देण्याची जबाबदारी आयकर विभागाने ' युटीआय ' ( युनिट ट्रस्ट ऑफ इंडिया) आणि ' एनएसडीएल ' ( नॅशनल सिक्युरिटी डिपॉझिटरी लिमिटेड) या संस्थांकडे सोपवले आहे. यासाठी 65 रुपये खर्च येतो. अर्ज भरून सोबत रंगीत फोटो आणि ओळख पटवणारे दाखले दिल्यानंतर अंदाजे महिनाभरात ' पॅन ' कार्ड घरपोच मिळावे, अशी अपेक्षा आहे. या शिवाय इंटरनेटद्वारे ' पॅन ' मिळवण्याची सुविधा उपलब्ध झाली आहे. 222. द्बठ्ठष्श्ाद्वद्गह्लड्ड 3 द्बठ्ठस्त्रद्बड्ड.द्दश्ा 1 . द्बठ्ठ या साइटवर गेल्यास डाव्या बाजूला न्श्चश्चद्य 4 क्कड्डठ्ठ असा आयकॉन दिसतो. त्यावर क्लिक केल्यास ' तात्काळ पॅन ' चा फॉर्म 49 ए अर्ज समोर येतो. ऑनलाईन अर्ज भरताना सोबत द्यायची 66 रुपयांची फी चेक, डिमांड ड्राफ्ट किंवा क्रेडिट कार्डने देणार असा प्रश्ान् असतो. क्रेडिट कार्डचा पर्याय निवडल्यास अर्ज भरल्यानंतर लगेचच ' पोच पावती ' मिळते.पोचपावतीचा प्रिंट घेऊन त्यावर रंगीत फोटो लावायचा आणि ओळख दाखवणारे पुरावे सोबत जोडून नमूद केलेल्या पत्त्यावर 15 दिवसांत पाठवावा. अर्ज मिळाल्यानंतर आणि त्यानंतर कागदपत्रे पोस्टाने मिळाल्यानंतर पुढील पाच दिवसांतच ' पॅन ' मिळण्याची हमी आयकर खाते देते. पैसे देण्याचा पर्याय चेक किंवा डिमांड ड्राफ्ट निवडल्यास ऑनलाईन अर्जावर चेक क्रमांकांचा उल्लेख करावा.
