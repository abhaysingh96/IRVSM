लेख: जाता जाता
राणी 'डे'!
[ Tuesday, January 11, 2005  07:24:10 pm]


' बघा, बाबा, तुम्ही आम्हाला नेहमी बोलत असता ना, पण राणी पण ' डे ' ऑबर्झव्ह करते. ' ज्यु. चकोरीने मुंबई टाइम्स चकोरपुढे सरकवत काहीशा तक्रारीच्या आणि काहीशा जिंकल्याच्या आनंदात सांगितलं. चकोरने पटकन मजकुरावरून नजर टाकली.

' अच्छा! सध्या ही कोण तुझी राणी आहे तिचा डेली ' ब्लॅक डे ' चालू आहे वाटतं! ' चकोरने विचारलं.

' आमचा एक दिवस ' डे ' असतो तर तुम्ही किती चेष्टा करता आमची. ' तिने आपलं म्हणणं सोडलं नव्हतं. ' क्युट दिसते ना! ' तिने विचारलं.

आता चकोरीने राणीप्रकरण वाचून काढलं.

' ह्यॅ! ' तिने पेपर भिरकावला.

' काय झालं आई! ' मुलीने आश्चर्याने विचारलं.

' ब्लॅक ' सिनेमा चालावा म्हणून केलेली आयडिया आहे. तसं पेपरात म्हटलं आहे. ' तिने खुलासा केला.

ज्यु. चकोरीने नकाराथीर् मान हलवली.

' तसं असेल तर रंगीला सिनेमाच्या वेळेला ऊमिर्ला आणि अमीरने फक्त रंगीबेरंगी कपड्यात फिरायला हवं होतं. ' तिने मुद्दा मांडला.

इतका वेळ ज्यु. चकोर हे शांतपणे ऐकत होता. ' या न्यायाने वॉर मुव्हीजच्या सगळ्यांना सैनिकांचा युनिफॉर्म चढवून, रस्त्यात परेड करावी लागेल ', त्याने चर्चा वेगळ्या मोडमध्ये नेली.

' मुगले आझम ' लागला तेव्हा दिलीप, मधुबाला असे फिरत होते काय ?' चकोरने विचारलं. चकोरीने फक्त नकाराथीर् मान हलवली.

' राणीचे कपडे कोणी स्पॉन्सर केले आहेत का ?' चकोरीने मुलीला विचारून विषय बदलला.

' ब्लॅक सिनेमाच्या प्रोड्युसरने! ' ज्यु. चकोरने मत मांडलं.

' जस्सी कशी सतत सीरियलमधले कॉस्च्युम घालून वावरत असते. तशी राणी काळे कपडे घालून फिरते. ' मुलाने माहिती दिली.

' अशाने सिनेमा चालतो काय रे! ' चकोरीने कुतुहलाने ज्युनियर कंपनीला विचारलं.

' सिनेमा कसा आहे यावर तो धावतो, चालतो, नाहीतर पडतो. आणि याच क्रमाने प्रोड्युसरची अवस्था होते. ' मुलीने एका वाक्यात सार मांडलं.

' मग कशाला असली नाटकं करतात ?' आईचा प्रश्ान्.

' ग्लॅमर र्वल्डमध्ये टिकायचं असलं की हे अपरिहार्य असतं. ' चकोरने माहिती दिली.

' म्हणून तर सिनेमा रिलिजची वेळ आली की त्यातील हीरो, हीरॉईन मुलाखती देत फिरतात. आपला रोल कसा ग्रेट आहे असं पुन्हापुन्हा सांगतात. ' मुलीने आईच्या ज्ञानात भर घातली.

' तुम्ही काही म्हणा, मला वाटतं त्सुनामीमुळे झालेलं दु:ख व्यक्त करण्यासाठी राणी अशी फिरत असावी. ' चकोरीच्या या कॉमेंटवर सगळे गप्पगार झाले.

चकोर
