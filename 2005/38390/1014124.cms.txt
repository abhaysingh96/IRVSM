लेख: जाता जाता
अमराठी नाटक!
[ Monday, February 07, 2005  08:36:57 pm]


' अय्या, आता मराठीतही हिंदी नाटकं यायला लागली बघा! ' ज्यु. चकोरीने म. टा. जेमतेम चाळला आणि कॉमेंट केली.

चकोर दांपत्याला तिच्या बोलण्याचा संदर्भ कळला असल्यामुळे दोघेही शांत होते.

' ते मराठी नाटक आहे. नीट वाच. ' आईने मुलीला उपदेश केला.

' पण हे लोक नाटक सिनेमाला हिंदी, इंग्लिश नावं का ठेवतात! ' ज्यु. चकोरी आपली बाजू सोडायला तयार नव्हती.

' तुझ्यासारख्यांना नावं ठेवायला संधी मिळावी म्हणून! ' चकोरने शाब्दिक खेळ केला.

' अॅक्चुअली, जे प्रेक्षकांनी म्हणायचं ते नाटकाला नाव दिलं आहे. ' दरम्यान ज्यु. चकोरने पेपर वाचत कॉमेंट केली.

' माझ्या प्रश्ानचं उत्तर कोणी देत नाही. ' मुलीने तक्रार केली.

' आमच्या वेळी नाटक, सिनेमाची नावं कशी असायची! प्रपंच, मानिनी, कुंकू, शेजारी, सौभद, जय जय गौरीशंकर. ' चकोरी एकदम भूतकाळात शिरली.

' थँक्यू, मिस्टर ग्लॅड! ' चकोरने तिला चिडवलं.

' तसं क्वचित असायचं. ' तिने बाजू मांडली.

' आता बघावं तर ' ऑल द बेस्ट ' काय, ते पुरेसं नाही म्हणून ' ऑल द बेस्ट अल्टिमेट ,' ' डॅडी, आय लव्ह यू ', ' नॉट ओन्ली मिसेस राऊत ' काय, चकोरीने यादी पाठ असल्यासारखी घडाघड म्हणून दाखवली.

' माझ्या प्रश्ानचं उत्तर कोणी देईल काय! ' मुलीने पुन्हा रेकॉर्ड लावली.

' याच्यापुढे भट, वाघ यांनी हिंदी नाटकं काढली तरी आश्चर्य वाटायला नको. अमिताभला संमेलनाच्या उदघाटनाला बोलावलं. आता त्याला कॅरेक्टर रोल देऊन नाटक. शिंदे, गोडबोले हे नव्या दमाचे लेखक नाटक लगेच पाडतील. ' चकोरी ' आधुनिक मराठी नाट्यसृष्टी ' या विषयावर पीएच. डी. करते आहे की काय असा चकोरला संशय आला.

' पण भट, वाघ या नाटकाचे प्रयोग कोठे करतील ?' चकोरने शंका काढली.

' का! मंुबईत हिंदी नाटकं चालणार नाहीत काय! उलट यापुढे हिंदीच नाटकं चालतील. छठपूजेची गदीर् दरवषीर् कशी वाढते आहे हे पाहिलं नाहीत वाटतं! ' चकोरीने मुद्दा मांडला.

' म्हणजे आता मराठी व्यावसायिक नाटकं ' छबिलदास ' मध्ये करायची वेळ येणार असं दिसतं! ' चकोरने शंका काढली.

' प्रेक्षकांची संख्या लक्षात घेता, छबिलदासही मोठं होईल. ' चकोरी.

' माझ्या प्रश्ानचं उत्तर ?' मुलीने हट्ट सोडला नाही.

' अगं, ग्लोबलायझेशन म्हणतात ते हेच. निदान नावात तरी आंतरराष्ट्रीय फ्लेवर. ' चकोरीने तिची समजूत काढली.

तिनं मान डोलावली.

चकोर
