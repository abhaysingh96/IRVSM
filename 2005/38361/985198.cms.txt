महाराष्ट्र
मराठी चित्रपटांच्या स्मृतींचा सप्तरंगी आलेख!
[ Sunday, January 09, 2005  08:14:33 pm]


म. टा. प्रतिनिधी

पुणे : कृष्णधवल चित्रपटांपासून ते ऑस्करच्या वारीपर्यंतचा मराठी चित्रपटांचा आलेख दृक-श्ााव्य माध्यमातून, चित्रपटाशी संबंधित कलाकार, दिग्दर्शक यांच्या आठवणींद्वारे उलगडला आणि ' श्यामची आई ते श्वास ' या कार्यक्रमाने आशय सांस्कृतिक आयोजित पुलोत्सवाची शनिवारी सांगता झाली.

मराठी चित्रपटांच्या ' प्रभात ' युगातील ' घन:श्याम सुंदरा ' ने सुरू झालेल्या या कार्यक्रमात मराठी चित्रपटांच्या वाटचालीतले अनेक टप्पे मांडण्यात आले. त्याचबरोबर त्या काळातील चित्रपटांशी संबधित कलाकार, निर्माते, दिग्दर्शक यांनी त्यांच्या स्मृतीतील क्षणांना निवेदक राहुल सोलापूरकर आणि आदिती देशपांडे यांच्याशी बोलताना पुनश्च एकवार उजाळा दिला.

' माणूस ' मधील ' कशाला उद्याची बात.... ' आणि ' श्यामची आई ' मधील ' छडी लागे छमछम... ' सारख्या गीतांनी कोथरूडच्या यशवंतराव चव्हाण नाट्यगृहात उपस्थित असलेल्या रसिकांना ' कृष्णधवल ' काळात नेले, तर ' श्याम ' माधव वझे, ज्येष्ठ रंगकमीर् चित्तरंजन कोल्टकर आणि ज्येष्ठ चित्रपट दिग्दर्शक राम गबाले यांनी मराठी चित्रपटांच्या सुवर्णकाळातील काही आठवणी रसिकांसमोर कथन केल्या.

गदिमा, राजा परांजपे आणि सुधीर फडके या त्रिकुटाच्या कारकिदीर्मुळे मंतरलेल्या चित्रपटांमधील अजरामर गाण्यांच्या पुनर्पत्ययाचा आनंद रसिकांना लुटता आला. ज्येष्ठ कवी सुधीर मोघे यांनीही काही प्रेक्षकांपर्यंत न पोहोचलेले पडद्यामागचे क्षण मांडले. त्याचबरोबर तमाशापटातील फक्कडबाज लावण्या रंगमंचावर सादर झाल्याने टाळ्यांच्या कडकडाटात प्रेक्षकांना त्याला योग्य दादही देता आली.

दादा कोंडके यांच्या ' रौप्यमहोत्सवी ' चित्रपटातील काही प्रसंग आणि अशोक सराफ- लक्ष्मीकांत बेडेर् या विनोदवीरांच्या करामतींनी प्रेक्षागृहात हास्याची लकेर निर्माण केली. जयराम कुलकणीर्ंनीही प्रेक्षकांच्या लाडक्या ' लक्ष्या ' च्या आठवणी सांगितल्या, तर जब्बार पटेल, अमोल पालेकर यांच्यामुळे मराठी चित्रपटांची बदललेली वाट चोखाळण्याचा विश्वास सुमित्रा भावे आणि सुनील सुकथनकर यांनी व्यक्त केला.

गजेंद अहिरे, प्रमोद प्रभुलकर या नव्या दमाच्या दिग्दर्शकांनी मराठीतील बंद पडलेली ' प्रयोगशाळा ' चालू ठेवायची आहे, असे सांगितले. त्यानंतर ' श्वास ' मधील दृश्याने कार्यक्रमाची सांगता झाली ; पण कौटुंबिक, सामाजिक, विनोदी, ऐतिहासिक, पौराणिक, संतपट आणि प्रायोगिक अशा सप्तरंगातील मराठी चित्रपटांचा हा मागोवा रसिकांना पुलोत्सवाची सांगता भेट देऊन गेला.
