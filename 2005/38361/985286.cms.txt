महामुंबई
'जातीय दंगे शिक्षेविना सुटतात हीच खंत'
[ Sunday, January 09, 2005  09:41:49 pm]


म. टा. विशेष प्रतिनिधी

मुंबई : ' एखाद्याने चोरी केली तरी त्याला सजा होऊ शकते ; पण राजकीय स्वार्थासाठी सत्ताधाऱ्यांनी घडवून आणलेल्या हजारो निरपराध्यांचे शिरकाण घडवणाऱ्या जातीय दंग्यातील आरोपी मात्र बिनबोभाट सुटतात, ही आमची मुख्य खंत आहे. तीच दाखवण्यासाठी आम्ही ' अमु ' काढला आहे, असे ' अमु ' च्या दिग्दशिर्का शोनाली बोस हिने आज वार्तालापात सांगितले.

' मामि ' चे कार्यवाह किरण शांताराम यांनी शोनालीशी खास गप्पांचा कार्यक्रम महोत्सवात आयोजिला होता. ' अमु ' हा माझा पहिलाच चित्रपट! तो लवकरात लवकर भारतात प्रदशिर्त व्हावा, हीच इच्छा असल्यामुळे सेन्सॉर बोर्डाने या चित्रपटाला दिलेले ' ए ' सटिर्फिकेट व आठ ' कट्स ' मान्य नसूनही मी स्वीकारले. ' मंत्र्यांचाच 1984 च्या शिखांच्या शिरकाणात हात होता ', असा आरोप करणारे शेजारणींचे उद्गार सेन्सॉरने मला ' मूक ' करायला लावले, ते मी मान्य केले. कारण ते मौन मला अधिक बोलके वाटते. मात्र, हिंसाचार, सेक्स असे काहीही घटक नसलेल्या या चित्रपटाला मिळालेले ' ए ' सटिर्फिकेट बदलून ' यू ' मिळावे, यासाठी कोर्टात जाण्याचा आपला विचार आहे, असेही शोनालीने सांगितले.

कोंकणाची गुणवत्ता ' मिस्टर अँड मिसेस अय्यर ' मध्येच मला जाणवली होती, म्हणून तिला तीन महिने अमेरिकेत नेऊन तिचे इंग्रजी उच्चारण मी बसवून घेतले. माझ्या अपेक्षेप्रमाणेच तिने अप्रतिम काम केले आहे, असे समाधानही शोनालीने व्यक्त केले. कुठल्याही विशिष्ट धर्मांतील तणाव मला ' अमु ' त दाखवायचा नाही. कारण जातीय तणाव प्रत्येकालाच आयुष्यातून उठवतात, हेच मला इथे दाखवायचे आहे, असेही दिग्दशिर्केने सांंगितले.
