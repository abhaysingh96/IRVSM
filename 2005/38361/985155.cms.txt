लेख: जाता जाता
मराठीच मराठी
[ Sunday, January 09, 2005  06:46:00 pm]


आज आपल्या बातम्यांमध्ये खास पाहुणे आहेत चकोर आणि ते आपल्याशी संवाद साधणार आहेत ' मराठी लँग्वेजचे फ्युचर ' या विषयी.

' चकोरसाहेब आपणास लाँग टर्ममध्ये मराठी भाषा टिकेल असं वाटतं का ?'

' एक हजार वर्षं ही तुमच्या भाषेत लाँग टर्म किंवा शॉर्ट टर्म असली तर आणखी हजार वर्षं तरी या भाषेला चिंता नाही.

' साहेब आमच्या दर्शकांना सांगा की एक हजार वर्षं म्हणजे किती बिलियन ?'

' तुमच्या प्रेक्षकांना सांगा की शंभरला दहाने गुणलं की एक हजार होतात. '

' वॉव, किती मोठी हिस्टरी आहे या लँग्वेजला! '

' साहेब, ही भाषा पॉप्युलर होण्यासाठी काय करायला हवं असं तुम्हाला वाटतं ?'

' तू मला मुलीसारखी आहेस म्हणून विचारतो. तू मराठी माध्यमात शिकलीस की इंग्लिश ?'

' साहेब, माध्यम म्हणजे काय ते जरा सांगा ना प्लीज! '

' माध्यम म्हणजे इथे त्याचा अर्थ मीडियम असा घ्यायचा मुली. '

' मी तुमची मुलगी आहे तर मी तुम्हाला डॅडी म्हटलं तर चालेल का ? म्हणजे आपला इंटरव्ह्यू अधिक इन्फॉर्मल वाटेल. '

' तू मला बाबा म्हटलं तर जास्त चांगलं वाटेल. कारण बातम्या मराठी आहेत ना! '

' बाबा, हे फारच बोअरिंग आणि डाऊन माकेर्ट आहे. '

मुली तू कोणत्या शाळेत होतीस ?'

' मी ना काँव्हेन्टमध्ये होते. माझी मम्मी आणि पप्पा म्हणले की इंग्लीश मीडियम असलं की प्रोग्रेस होतो. '

' हो का! आणि तुझे पप्पा आणि मम्मी कोणत्या शाळेत जायचे ?'

' ते व्हिलेजमध्ये होते स्टडीला. '

' मग त्यांची प्रगती झाली नाही का ?'

' प्रगती म्हणजे प्रोग्रेस ना! आय फिल दे आर वेल सेटल्ड! '

' मुली मला सांग, तुझी मातृभाषा कोणती ?'

' मातृभाषा म्हणजे मदर टंग ना! माझी मराठी आहे मातृभाषा. '

' मग मुली तुला मराठी बोलायला आवडत नाही का ?'

' मम्मी आणि पप्पा अॅडव्हाईस करतात की फ्रेंच, जर्मन, जॅपनीज अशी एखादी फॉरीन लँग्वेज तू शिक. पण मी संस्कृत स्टडी करणार आहे. '

' का गं! '

' आई म्हणते आता यूएस आणि युरोपमध्ये या भाषेला ब्राईट फ्युचर आहे. मला मग तिकडे मायग्रेट होता येईल. '

' मुली, तू मराठी पण शिक. मराठी पण फॉरेन लँग्वेज होईल आणि तिला तिकडे सोनियाचे दिवस येतील असं दिसत.

' बाबा, थँक यू फॉर युवर अॅडव्हाईस. '

चकोर
