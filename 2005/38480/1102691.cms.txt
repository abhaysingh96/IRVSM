लेख: जाता जाता
सोडा साखर!
[ Sunday, May 08, 2005  07:52:06 pm]


' साखर खाणं सोडून द्यायला हवं! ' चकोरने सुस्कारा टाकत उद्गार काढले. ' का! डायबेटिस झाला वाटतं! ' मुलीने गंमतीने विचारलं.

' माझ्या गोड खाण्यावर तुझा इतका राग का ?' चकोरने हसत विचारलं.

' तुम्हीच म्हणालात ना, साखरेला रामराम ठोकायला हवा. ' मुलाने प्रश्ान् केला.

' लगेच तुम्ही निष्कर्ष काढून मोकळे झालात. ' चकोरने त्यांना टोकलं.

' इतकं होऊनही तुम्ही कारण सांगतच नाही. चहात साखर किती टाकायची ?'' शांतपणे चर्चा ऐकणाऱ्या चकोरीने विचारलं.

' आता तू पण यांंच्यात सामील झालीस वाटतं! मला काहीही झालेलं नाही. नेहमीप्रमाणे साखर टाक आणि खायला पण गोडाचाच शिरा कर. आठवड्याची सुरुवात गोडाने करावी. ' चकोरने भूमिका स्पष्ट करताना आपली फर्माइशही दिली.

' आठवड्याची सुरुवात खाण्याच्या विचारापेक्षा कामाच्या विचाराने करावी. सगळयांनी पटापट आटपा आणि उद्योगाला लागा. ' चकोरीने फर्मान काढलं.

' पण बाबा, तुम्ही एकदम साखरेवर कसे घसरलात ?' मुलाने विचारलं.

' साखर महाग झाल्याच्या बातम्या यायला लागल्या आहेत. वस्तूची किंमत वर गेली की मागणी खाली येते. ' चकोरने अकारण अर्थशास्त्र सांगितलं.

' तरी तुम्हाला शिरा हवा. ' चकोरीने सुनावलं.

या आक्रमणामुळे चकोर गप्प झाला.

' साखरेचे भाव कधी कमी होतात आणि कधी वाढतात हे कळतच नाही. ' चकोरीनेच गाडी रुळावर आणली.

' व्यापारी ठरवतात अशा गोष्टी. आपण फक्त बातम्या वाचायच्या. ' चकोरने सांगितलं.

' पण शरद पवार नेहमी आश्वासन देतात की साखरेच्या किमती वाढणार नाहीत. आता ते इथे नाहीत वाटतं! ' चकोरीने विचारलं.

' इथेच आहेत. कोणत्याही थंड हवेच्या देशी गेलेले नाहीत. ' चकोरने माहिती दिली.

' मग त्यांचं काय म्हणणं आहे ?' चकोरीचं कुतुहल वाढलं.

' गेल्या वर्षी पाऊस कमी झाल्यामुळे उसाचं उत्पन्न घटलं, म्हणून साखर कमी तयार झाली वगैरे तपशील ते देतील. पेपरवाल्यांनी छापलं तर ब्राझीलमध्ये साखरेची काय स्थिती आहे, हेही आपल्याला त्यांच्याकडून कळेल. ' चकोरने सांगितलं.

' मग साखर इंपोर्ट का नाही करत ?' ज्यु. चकोरने शंका काढली.

' जागतिक बाजारपेठेत साखर कशी महागली आहे वगैरे आपल्याला पवारसाहेब सांगतील. ' चकोरने शंकासमाधान केलं.

' गोड साखरेवर इतकी बोअर चर्चा तुम्ही का करता आहात डॅड ?' ज्यु. चकोरीने जांभई देत विचारलं.

' चर्चा बंद, शिऱ्याचा गंध दरवळायला लागला बघ! ' चकोरने जाहीर केलं.

चकोर
