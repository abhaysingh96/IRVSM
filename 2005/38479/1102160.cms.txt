संवाद
याहू ते हरि ओम
[ Saturday, May 07, 2005  06:42:37 pm]


... शम्मी कपूर नावाच्या बंडखोर नायकाच्या आगमनाबरोबर हिंदी सिनेमातील नायक उत्साहाने उसळणारा, धटिंगणपणे उड्या मारणारा, मदमस्त नाचून प्रणयाराधन करणारा बनला. इतका की सिनेमा थिएटरातील 35 मिमीच्या पडद्यावर जणू तो जवाँजोश मावत नव्हता!

इंटरनेट जगतात ' याहू ' वेबसाइट अवतीर्ण होण्यापूवीर् काही दशके या हू! या आपल्या आरोळीने तत्कालीन तरुणतरुणींना झपाटून टाकणाऱ्या जुन्या पिढीतील नामवंत नायक, अभिनेते व दिग्दर्शक शम्मी कपूर यांना महाराष्ट्र सरकारने नुकताच ' राज कपूर जीवन गौरव पुरस्कार ' दिला आहे. आता सत्तरीच्या पुढे गेलेल्या शम्मी कपूर यांचा जीवनप्रवास ' या हू! ' ते आता ' हरि ओम ' असा आहे. होय! शम्मीजी आता आध्यात्मिक बनले आहेत.

1950 ते 70 पर्यंतच्या काळातील ' जंटलमनली ' हिंदी चित्रपटांमध्ये शम्मी कपूरांनी ' या हू! ' व ' सुकू सुकू ' च्या आरोळया मारून वातावरण पार बदलून टाकले. त्यापूवीर्च्या चित्रपटांतील हीरो- हिरोइन हळुवार गाणी म्हणून मनधरणी, प्रेमयाचना करीत. शम्मी कपूर नावाच्या बंडखोर नायकाच्या आगमनाबरोबर हिंदी सिनेमातील नायक उत्साहाने उसळणारा, धटिंगणपणे उड्या मारणारा, मदमस्त नाचून प्रणयाराधन करणारा बनला. इतका की सिनेमा थिएटरातील 35 मिमीच्या पडद्यावर जणू तो जवाँजोश मावत नव्हता!

नामवंत अभिनेते पृथ्वीराज कपूर यांचा मधला मुलगा म्हणून मुंबईत 1931 मध्ये जन्मलेला देखणा शमशेरराज म्हणजे शम्मी. त्याने न्यू इरा स्कूलमधून मॅट्रिक झाल्यानंतर रुईया कॉलेजात शिक्षण घेतले व नंतर वडिलांच्याच ' पृथ्वी थिएटर्स ' मध्ये वयाच्या 17 व्या वषीर् ज्युनियर आटिर्स्ट म्हणून प्रवेश केला. वडील बडे निर्माते-अभिनेते असूनही वयाच्या 22 व्या वषीर् चित्रसृष्टीत पदार्पण करणाऱ्या शम्मीला वारशाचा लाभ मिळाला नाही. 1952 मध्ये शम्मीचा पहिला चित्रपट ' जीवन ज्योती ' आला. तो साफ पडला. त्यानंतरच्या 52 ते 57 या पाच वर्षांच्या काळात शम्मीचे 17 चित्रपट आले आणि लागोेपाठ ' फ्लॉप ' झाले. मधुबाला यांच्याबरोबरचा ' रेल का डिब्बा ', सुरय्यासोबतचा ' शमा परवाना ', नलिनी जयवंतसमवेतचा ' मेहबूबा ' या चित्रपटांचा त्यात समावेश होता. दरम्यान 55 पर्यंत वडील बंधू राज कपूर हीरो व दिग्दर्शक बनून प्रसिद्धीच्या झोतात आले होते. याच काळात गीता बाली यांच्याशी विवाह, पुत्र मिकी याचा जन्म या घटना शम्मींच्या जीवनात घडल्या. 59 मध्ये ' दिल देके देखो ' आणि 61 मधील ' जंगली ' या चित्रपटांनी शम्मींना स्टार बनविले. 60 ते 70 या दशकात ' राजकुमार ', ' जानवर ', ' अॅन इव्हनिंग इन पॅरिस ' हे शम्मींचे चित्रपट रसिकांनी डोक्यावर घेतले. ते स्टार बनले खरे, पण माकडचेष्टा करणारा अभिनेता अशीच त्यांची प्रतिमा बनली होती. पण ही प्रतिमा त्यांनी ' तिसरी मंझिल ' (1968), ' ब्रह्माचारी ' (1968) या चित्रपटांतील गंभीर अभिनयाने बदलून टाकली. ' ब्रह्माचारी ' ने त्यांना सवोर्त्कृष्ट अभिनेत्याचे ' फिल्मफेअर अॅवॉर्ड ' मिळवून दिले. पण 1965 साली गीता बालींच्या आकस्मिक निधनाने शम्मींवर दुदैर्वाचा घाला घातला. नंतरची चार वषेर् ते भरकटले. मदिरा व मदिराक्षीत शम्मी वाहवल्याची चर्चा फिल्म इंडस्ट्रीत चवीने चाले. नंतर शम्मींनी दुसरा विवाह नीलादेवींशी केला. नीलादेवींनी शम्मी कपूरांचे जीवन पुन्हा रुळावर आणले.

शम्मींच्या नंतरच्या करीअरमध्ये पुन्हा काही ' पडेल ' चित्रपट आले. लीना चंदावरकरसोबतचा ' प्रीतम ', बबिताबरोबरचा ' तुमसे अच्छा कौन है ' हे चित्रपट पडले. आता वय होत चालले होते. डोक्यावरील केस विरळ होत होते. याच काळात हिंदी चित्रसृष्टीत प्रवेश केलेल्या राजेश खन्नाने ' स्टारडम ' वर कब्जा केला. शम्मींचे ' जाने अनजाने ' ' जवाँ मोहब्बत ', ' पगला कहीं का ' हे नवे चित्रपट चालले नाहीत.

दिग्दर्शक बनण्याचा प्रयत्न शम्मींनी केला खरा, पण ' मनोरंजन ' व ' बंडलबाज ' हे चित्रपट बॉक्स ऑफिसवर आपटले. तरीही नाउमेद न होता त्यांनी दुय्यम भूमिका (ज्यांना फिल्म इंडस्ट्रीच्या भाषेत ' कॅरॅक्टर रोल्स ' म्हणतात) करायला सुरुवात केली. ' जमीर ', ' प्रेम रोग ', ' परवरिश ', ' रॉकी ' या चित्रपटांत ते ' कॅरॅक्टर रोल्स ' मध्ये दिसले. 1982 मधील ' विधाता ' या चित्रपटात केलेल्या भूमिकेबद्दल त्यांना ' बेस्ट सपोटिर्ंग अॅक्टर ' चे ' फिल्मफेअर अॅवॉर्ड ' मिळाले.

1987 साली शम्मी कम्प्युटरच्या, पाठोपाठ इंटरनेटच्या प्रेमात पडले! दिवसाचे सुमारे 12 तास कम्प्युटरच्या सहवासात आणि इंटरनेटवर सैर करण्यात घालवू लागले. 1995 मध्ये त्यांनी ' इंटरनेट यूजर्स क्लब ऑफ इंडिया ' नावाची संस्था काढली. तिचे ते अध्यक्ष बनले. इंटरनेटविषयी जागरुकता निर्माण करणे, इंटरनेटविषयक प्रश्ान् सोडविणे ही संस्थेची उद्दिष्टे आहेत. शम्मींना नंतर ' फिल्मफेअर लाइफटाइम अचिव्हमेंट अॅवॉर्ड ' मिळाले. आजमितीला जाहिराती / टीव्ही सीरियल्समध्ये भूमिका करणारे कपूर घराण्यातील शम्मी कपूर हे एकमेव सदस्य आहेत. काही वर्षांपूवीर् शम्मीजींची गाठ एका आध्यात्मिक गुरूशी पडली. दिवंगत हेड्डा खानबाबा हे त्यांचे गुरू. त्यांच्यामुळे शम्मीजींचे जीवन पार पालटले. ते आता दररोज पूजा करतात. पण आपल्या श्रद्धेचे, भक्तिभावाचे अवडंबर माजविणे त्यांना अजिबात आवडत नाही. स्पष्टवक्ते, सत्य भाषण आवडणारे, फटकळ (सणकू!) असे शम्मी कपूर यांचे व्यक्तिमत्त्व आहे. एकदा त्यांचे भाषण ऐकण्याची संधी मिळाली. ' इन्फमेर्शन टेक्नॉलॉजी ' वरील वेबसाइटचे उद्घाटन शम्मी कपूरांच्या हस्ते सहा-सात वर्षांपूवीर् पंचतारांकित हॉटेलात झाले. कंपनीच्या प्रवर्तकांनी शम्मींचे वर्णन ' फादर ऑफ इंडियन इंटरनेट ' असे केले. शम्मी नंतर भाषणाला उभे राहिले. पहिल्यांदाच त्यांनी सांगितले की हे त्यांचे वर्णन साफ चुकीचे असून, ते आपल्याला मुळीच आवडले नाही! शिवाय आधीच्या वक्त्याने इंटरनेटविषयी दिलेल्या गुंतागुंतीच्या तांत्रिक माहितीतली निम्मी तर आपल्या पार डोक्यावरून गेल्याचेही त्यांनी प्रामाणिकपणे सांगून टाकले. नंतर मात्र वीस मिनिटांच्या अस्खलित इंग्रजीत, अधुनमधून उत्तम हिंदीची पेरणी करीत त्यांनी जुन्या काळापासून ते आजतागायत दूरसंचार सेवा कशी विकसित होत गेली, याचा इतिहास सांगून श्रोत्यांना मंत्रमुग्ध केले. शम्मी कपूर नावाचे जातिवंत अभिनेते एक उत्तम वक्तेही असल्याचे त्या दिवशी अनेकांना कळाले!

विश्वास डिग्गीकर
