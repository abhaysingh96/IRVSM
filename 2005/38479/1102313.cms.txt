महामुंबई
'दिल को समझाने तेरी याद चली आयी है...'
[ Saturday, May 07, 2005  09:29:00 pm]


म. टा. विशेष प्रतिनिधी

मुंबई : एस. डी. बर्मननी ' देवदास ' मध्ये केवळ सतारीचे सूर व बासरीचा एकच आलाप वापरून दारूड्या देवदासचा ' मितवा नहीं आये ' म्हणून स्वरबद्ध केलेला आर्त विलाप... ' मिर्झा गालिब ' मध्ये गुलाम महमदच्या संगीतात गायलेले ' दिलें नादाँ तुझे हुआ क्या है '... ज्या गाण्याने सचिनदांना उत्कृष्ट संगीतकार म्हणून पहिला फिल्मफेअर पुरस्कार मिळवून दिला, ते ' टॅक्सी ड्रायव्हर ' मधील ' जायें तो जायें कहाँ '... खय्यामच्या संगीतातले ' फूटपाथ ' मधील ' शामें गम की कसम ' आणि अनिलदांच्या ' तराना ' तील ' सीनेमें सुलगते है अरमाँ ' - अशा कैक मखमली आवाजातील दर्दभऱ्या गीतांनी रसिकांच्या काळजावर कट्यार फिरवणारे गायक तलत महमूद यांची याद तरुण गायक गौरव बांगिया याने ' दिलको समझाने ' उत्कटतेने करून दिली...

तलत आपल्यातून गेल्याला येत्या 9 मे रोजी सात वषेर् पूर्ण होत आहेत. ' गझल सम्राट ', ' गझल गौरव ' म्हणून गौरवल्या गेलेल्या तलतला स्वरांजली वाहायला ' कीप अलाइव्ह ' ने उचितपणे तलतच्याच तरलतेने गाणाऱ्या गौरवची योजना केली होती. ' सा रे गा मा पा ' गीतस्पधेर्त सातत्याने दहा भागांत यशस्वी ठरलेला एकमेव विजेता असलेल्या या ' दिल्ली पुत्तर ' ने प्रचंड मेहनतीने साडेचार तासांत तलतची 40 गाणी रविवारी सादर करून, रसिकांनी ओसंडून गेलेल्या षण्मुखानंद सभागृहाकडून जोरदार दाद घेतली.

ज्या कलाकारावर विशेष कार्यक्रम सादर करायचा, त्याच्या वारसांचा मध्यंतरात सत्कार करण्याची ' कीप अलाइव्ह ' ची प्रथा आहे. पण ' आपल्या वडलांची गाणी कुणीच धड सादर करू शकणार नाही ' अशी भावना असलेले तलतचे पुत्र त्याच्या गाण्याच्या कुठल्याही कार्यक्रमाला जात नाहीत, असे उत्तर मिळाल्याने ' तलत महमूद : द गौरव ऑफ गझल्स ' या कार्यक्रमाच्या मध्यंतरात, मुंबईत ज्यांच्याकडून हिंदुस्थानी शास्त्रीय संगीत तो शिकत आहे, ते आचार्य अनंत वैद्यनाथन यांच्या हस्ते गौरवचा सत्कार करण्यात आला.

सरिता माधव व वृषाली पाटील या सहगायिकांसह गौरवने गायलेली युगलगीते, स्वत: सादर केलेली एकलगीते आणि अभिनयाच्या मोहात सापडल्याने संगीतकारांकडून 1955 नंतर काहीशा दुर्लक्षिल्याच गेलेल्या तलतच्या व्यक्तिगत व व्यावसायिक जीवनाचा मनोहर अय्यर यांनी ओघवत्या निवेदनातून काढलेला आलेख यामुळे ' गौरव ऑफ गझल्स ' हा कार्यक्रम संस्मरणीय झाला...
