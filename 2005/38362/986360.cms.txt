महामुंबई
'अतुल कुलकर्णी त्याच्या पिढीतील सवोर्त्तम अभिनेता'
[ Monday, January 10, 2005  10:06:04 pm]


म. टा. विशेष प्रतिनिधी

मुंबई : ' अतुल कुलकर्णी त्याच्या पिढीतील सवोर्त्तम अभिनेता आहे, अशी माझी ठाम धारणा आहे. म्हणूनच माझ्या ' मानसरोवर ' मधील विक्षिप्त रवी रायच्या भूमिकेसाठी मी त्याची निवड केली, असे लेखक/निर्माते/दिग्दर्शक अनुप कुरियन यांनी रविवारी पत्रकारांना सांगितले.

रवींद नाट्य मंदिरात सुरू असलेल्या ' मामि ' चित्रपट महोत्सवात कुरियन यांनी पत्रकारांशी संवाद साधला. ' मानसरोवर ' चा आधीच खूप बोलबाला झालेला असल्याने शनिवारी दुपारी ' रंगस्वर ' मध्ये झालेल्या ' शो ' ला प्रेक्षकांनी अमाप गदीर् केली होती. अगदी रंगमंचावरदेखील उत्सुक प्रेक्षक चढून बसले होते. ' मिडियाने ' प्रसिद्धी दिल्यामुळेच हा भरघोस प्रतिसाद लाभला, याबद्दल कुरियन यांनी प्रारंभीच पत्रकारांना धन्यवाद दिले. ' अतुलने केरळी शेतकरी रवी राय उत्तम साकारला असला तरी त्याचे उच्चारण मल्याळी वाटत नाही ; ते मराठी वळणाचेच वाटते ,' असा आक्षेप एका पत्रकाराने घेतला, त्यावर अनुपने तो आक्षेप मान्य केला. पण ' मल्याळी उच्चारणाकडे अधिक लक्ष देण्याऐवजी अनुपने भावदर्शनाकडेच जास्त लक्ष केंदित करावे, असे आमचे आधीच ठरले होते ' असेही त्याने सांगितले.

' मानसरोवराला तुम्ही माळती चंदन (नेहा दुबे) व रवी राय ( अतुल) यांच्यातील प्रेमकथा समजू शकता ; अथवा नायक रवीचा धाकटा भाऊ जॉर्ज (जाफर कराचीवाला) या दोन भावांची कहाणी समजू शकता किंवा ती जीवनातील विलक्षण योगायोगांची कहाणी समजू शकता. मी तो निर्णय प्रेक्षकांवरच सोपवला आहे! असेही कुरियन म्हणाले : ' मानसरोवर ' पुढील आठवड्यात इंग्लंडमध्ये प्रदशिर्त होत आहे. भारतातले आगमन अद्याप ठरलेले नाही, असेही त्यांनी सांगितले.

' आवार आशक्यो फिरे ' ( अनफिनिश्ड मॅटर्स) या बंगाली चित्रपटाचे निर्माते- दिग्दर्शक रवी ओझा व पटकथाकार मित्राणी भट्टाचार्य हेही या वार्तालापात सामील झाले होते. बंगाली सिनेमात उत्तमकुमार, सुचित्रा सेन यांसारख्या कलावंतांच्या, उत्तम बांधेसूद कथानकाच्या दजेर्दार चित्रपटांची परंपरा अलीकडील काळात खंडित झाल्यापासून दजेर्दार प्रेक्षकाने, सुमार निमिर्तीमूल्ये असणाऱ्या कमशिर्यल बंगाली चित्रपटांकडे पाठ फिरवली आहे. प्रेक्षकांना आकषिर्त करण्यासाठी उत्कृष्ट निमिर्तीमूल्ये व कल्पक कथा असलेला हा चित्रपट मी बनवला आहे. बंगाली मिडियाने त्याला उचलून धरले. प्रेक्षकांचा प्रतिसाद धिमा असला तरी तो वाढेल, असा विश्वास ओझांनी व्यक्त केला.
