अर्थव्यवहार
आता 'व्हीसीडी' नको; 'डीव्हीडी'च घ्या
[ Tuesday, May 10, 2005  08:46:42 pm]


' डीव्हीडी प्लेअर ' ची बाजारपेठ गेल्या वर्षभरात तब्बल 250 टक्क्यांनी वाढली आहे! यंदा, सन 2005 मध्ये 35 लाख ' डीव्हीडी (डिजिटल व्हिडिओ डिस्क) प्लेअर्स ' विकले जाण्याची अपेक्षा आहे.

' डीव्हीडी ' बाजारपेठेत एवढी तेजी येण्याचे कारण राष्ट्रीय पातळीवरील ' डीव्हीडी प्लेअर्स ' च्या ब्रॅण्ड्सच्या आकर्षक किमती. या ' डीव्हीडी प्लेअर्स ' च्या ' एण्ट्री लेव्हल ' किंमती 3990 रु.पासून आहेत. ' फिलिप्स ' चा डीव्हीडी प्लेअर्सच्या बाजारपेठेतील हिस्सा सुमारे 40 टक्के असून, गेल्या वर्षात 4,990 रु.च्या ' एण्ट्री लेव्हल प्राइस ' सह या कंपनीने किंमतयुद्ध सुरू केले आहे.

' डीव्हीडी प्लेअर ' वर सर्व प्रकारच्या डिस्क ( ' व्हीसीडी ' व ' एमपीथ्री ' ही) पाहता, ऐकता येत असल्यामुळे या उपकरणांची बाजारपेठ झपाट्याने वाढली आहे. यंदा ' एलजी ' ने 3990 रु. किमतीस एक ' एण्ट्री लेव्हल मॉडेल ' बाजारात आणले आहे.

अनेक चित्रपट वितरक कंपन्याही ' डीव्हीडी व्हिडिओ ' च्या किमती कमी करण्यासाठी आणि ' डीव्हीडी टायटल्स ' बाजारात मोठ्या प्रमाणावर उपलब्ध व्हाव्यात यासाठी योजना राबवीत आहेत.
