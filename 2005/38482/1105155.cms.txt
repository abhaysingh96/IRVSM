प्रगती फास्ट
'नायका'चा संघर्ष
[ Tuesday, May 10, 2005  04:47:24 pm]


निसर्गावर स्वामित्व गाजवण्याच्या इष्येर्ने झपाटलेला मनुष्य अन्य सजीवांच्या सुखदु:खाला कारणीभूत ठरतो. अशाच एका मुक्या पशूची कहाणी ' लांडगा ' या कादंबरीत वाचायला मिळते. कादंबरीचा ' नायक ' असलेल्या शूर लांडग्याची कहाणी फक्त प्राणीप्रेमींनाच नव्हे, तर कुणालाही आवडेल अशी आहे.

आयुष्यातील चढउतार, जीवनाला कलाटणी देणारे बरेवाईट प्रसंग फक्त माणसांच्याच नाही, तर पशुपक्ष्यांच्याही आयुष्यात येतात. मात्र अनेकदा निसर्गावर स्वामित्व गाजवण्याच्या इष्येर्ने झपाटलेला मनुष्यप्राणीच अन्य सजीवांच्या सुखदु:खाला कारणीभूत ठरतो.

देव हा सर्वशक्तिमान आणि संपूर्ण जगाचा नियंता असल्याची माणसाची धारणा असते. त्याचप्रमाणे पशुपक्ष्यांच्या दृष्टिकोनातून त्यांची ' नियती ' ठरवू पाहणारे मनुष्यप्राणी त्यांचे ' देव ' ठरतात. पशुपक्ष्यांचे हे ' देव ' प्रत्यक्षात ' भगवान ', ' हैवान ' आणि ' इन्सान ' यापैकी कुठल्या गटात मोडतात, यावर त्या मुक्या जनावरांचं जगणं-मरणं अवलंबून असतं. अशाच एका मुक्या पशूची कहाणी ' लांडगा ' या कादंबरीत वाचायला मिळते. जगप्रसिद्ध लेखक ' जॅक लंडन ' यांच्या ' व्हाइट फँग ' या कादंबरीचा ' लांडगा ' या नावाने अनंत सामंत यांनी मराठीत अनुवाद केला आहे. मात्र हा लांडगा ' गरीब बिच्चारा ' मुळीच नाही. कादंबरीचा ' नायक ' असलेल्या या शूर लांडग्याची कहाणी पशुपक्ष्यांबद्दल कुतूहल आणि आस्था असलेल्या प्राणीप्रेमींनाच नव्हे, तर कुणालाही आवडेल अशी आहे.

लांडगा आणि कुत्री यांच्या मिश्र विवाहातून जन्माला आलेल्या या पिल्लाच्या पांढऱ्याशुभ्र सुळ्यांमुळे त्याला ' व्हाइट फँग ' हे नाव मिळतं. बलिष्ठ लांडग्याची प्रचंड ताकद आणि कुत्र्याच्या प्रमाणबद्ध शरीराच्या वारसा लाभलेल्या व्हाइट फँगला बालपणापासूनच स्वत:चं अस्तित्व टिकवण्यासाठी अटीतटीचा संघर्ष करावा लागतो. त्याची नियती ठरविणारे ' देव ' हात धुऊन त्याच्या मागे लागतातच. त्यामुळे त्याच्यातला ' सैतान ' जागा होऊन थयथयाट करू लागतो. या यातनामय, भयंकर आयुष्यातील छळाची परिसीमा गाठली जाते, तेव्हा मात्र अकल्पितपणे त्याच्या सुटकेला धावून येतो एक ' लव्ह-गॉड! ' या लोभसवाण्या ' देवा ' च्या संगतीत ' व्हाइट फँग ' मधला सैतान शांत होतो का, या प्रश्नाचं उत्तरं लेखकाच्या शैलीदार कथनातून जाणून घेण्यातच खरी मजा.

गुन्हेगारी प्रवृत्तीचे विकृत मनोवृत्तीचे, निर्दय, क्रूर लोक मुळातच तसे असतात का ? की त्यांच्या आजूबाजूची परिस्थिती त्यांना तसे बनण्यास भाग पाडते ? हा अवघड प्रश्न ' लांडग्याच्या ' माध्यमातून मूळ लेखकाने वाचकांसमोर ठेवला आहे, हे या कादंबरीचे अत्यंत ठळक वैशिष्ट्य. त्यामुळेच ही कादंबरी निव्वळ एका लांडग्याची गोष्ट न राहता एका वेगळ्याच उंचीवर जाऊन पोहोचते.

कादंबरीत वर्णन केलेला काळ सन 1897 च्या आसपासचा आहे. या काळातील दैनंदिन जीवन, राहणी, चालीरितीचं प्रतिबिंब कादंबरीत यथार्थपणे उमटले आहे. फासेपारधी, रेड इंडियन्स, त्यांचे ' स्लेड डॉग्ज ' 1897 मध्ये सोन्याच्या खाणी सापडू लागल्या, तेव्हा आणि अलास्काच्या बर्फाळ प्रदेश तुडवत सोन्याच्या शोधात धावणारे वेडे यांची कथानकाच्या ओघात येणारी वर्णनं कुतूहल चाळवणारी आहेत. मूळ लेखकाची निरीक्षणशक्ती अत्यंत दांडगी असली पाहिजे त्याशिवाय या सगळ्या गोष्टींचे इतके हुबेहूब वर्णन त्याला करताच आलं नसतं.

कादंबरीतील ' क्लायमॅक्स ' च्या प्रसंगाचा होणारा ' गोड शेवट ' मात्र वासतववादी वाटत नाही. केवळ ' व्हाइट फँग ' ची नवजात पिल्ले आणि त्यांच्या आईसह कादंबरीचा शेवट केला असता तर तो दु:खाची झालर असूनही जास्त परिणामकारक, वास्तवाशी इमान राखणारा आणि एका वेगळ्या प्रकारे गोडच ठरला असता. मग मूळ लेखकाने असा शेवट का बरे केला नसावा ? ' निखळ गोड ' शेवट केला नाही तर कादंबरीच्या खपावर परिणाम होईल असे त्याला वाटले असावे का ? की खरोखरच त्याला असाच शेवट अभिप्रेत होता, हे प्रश्न मनात उद्भवतातच. एवढा ' शेवटाचा ' मुद्दा वगळला तर ' लांडगा ' वाचताना एक चांगलं पुस्तक वाचल्याचा आनंद मिळेल हे नक्की.
