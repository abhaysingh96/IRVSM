लेख: जाता जाता
चकोरीची खरेदी!
[ Tuesday, May 10, 2005  08:16:23 pm]


' तुमचं एटीएम कार्ड देऊन जा! ' चकोरीने सक्काळी फर्मान काढलं.

हे ऐकताच चकोरचा चेहरा पडला. त्याने ज्युनियर कंपनीकडे बघितलं. दोघेही गालातल्या गालात हसताहेत असा त्याला संशय आला.

त्याने चकोरीकडे नजर वळवली.

' कशाला ?' असा प्रश्न त्याच्या डोळ्यात होता.

पण त्याने काही विचारण्याच्या आतच चकोरीने त्याच्या मनातले भाव ओळखले.

आज अक्षय्य तृतीया आहे ना! काही खरेदी करावी म्हणते. ' तिने आपला बेत जाहीर केला.

' मग आज स्पेशल काय आहे ?' ज्यु. चकोरने उत्सुकतेने विचारलं.

अर्थात त्याला, इतक्या महत्त्वाच्या दिवशी, आई खायला वेगळं काय करणार आहे यात रस होता.

चकोरीने फक्त मान हलवली.

' आम्ही आज हा दिवस सेलिब्रेट करायला बाहेर जाणार आहोत. ' तिने जाहीर केलं. थोडक्यात , ' तुमचं तुम्ही बघा ' असा मेसेज तिने दिला.

' तुला असे कितीसे पैसे लागणार आहेत! तितके तर तुझ्या पर्समध्ये नेहमीच असतात. ' चकोरने तिला सांगितलं. ' त्यामुळे तू एटीमएममधून पैसे काढण्याची गरज नाही ' असं त्याला सुचवायचं होतं.

' आज थोडी खरेदी करावी असं म्हणते. साडेतीन मुहूर्तांपैकी एक आहे ना! ' तिने आपलं उद्दिष्ट जाहीर केलं.

' सोनं वाटतं! ' मुलीने विचारलं.

' तुझ्या लग्नासाठी आवश्यक तितकं आहे. आता आणखी घेण्याची गरज नाही. ' तिने हसत हसत मुलीची चेष्टा केली.

' मॉम! ' मुलीने चिडल्याचा आविर्भाव आणला.

' सुनेची काही सोय केली नाहीस वाटतं! ' चकोरने मुलाकडे बघत विचारलं.

' तिच्यासाठी हळूहळू डायमंड सेट घ्यायचा आहे. दरम्यान मी वापरीन तो. ' तिने जाहीर केलं.

' वॉव! ' मुलीला आश्चर्य लपवता आलं नाही.

' किती पडतात त्याला ?' चकोरच्या विचारण्यात भीती वाटत होती.

' घ्यावा तसे पडतात. पण लाखभराची सोय करायला हवी. ' चकोरीने शांतपणे सांगितलं.

चकोर दचकलाच आकडा ऐकून.

' इतके नाहीत अकाऊंटला. ' त्याने सांगितलं.

' सगळा सेट काही आज घेणार नाही. जमेल तसतसा घ्यायचा. ' तिने खुलासा केला.

चकोरी हे इतक्या सहजतेने सांगत होती की जणू काही तिने सगळं ठरवलं होतं. पण ज्यु. चकोरीला संशय आला की आईच्या मनात काहीतरी वेगळं आहे. तिने हळूच खुण करून विचारलं. चकोरीने तिला त्याच पद्धतीने सांगितलं की आज चकोरसाठी चांगले कपडे घेणार आहे. ज्युनियर्सनीही तिला कंपनी द्यावी असं तिने खुणावलं.

दोघांनी तिला ' मानलं तुला! ' असा इशारा केला.

चकोर
