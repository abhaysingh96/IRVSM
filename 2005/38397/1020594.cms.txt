महामुंबई: मुंबई टाइम्स
मोबाइलवर कॅरीबे व्हायरस
[ Monday, February 14, 2005  06:23:48 pm]


महेश पावसकर

मोबाइल हॅन्डसेटवर सध्या ' कॅरीबे ' नावाच्या व्हायरस धुमाकूळ घालतोय. त्यामुळे ' ब्ल्यूटूथ ' टेक्नोलॉजीचे मोबाइल वापरणाऱ्यांमध्ये भीतीचं वातावरण पसरलंय.

ब्ल्यूटूथ असलेल्या मोबाइल-धारकांकडे अचानक एक मेसेज येतो. ' तुम्हाला तुमचा मित्र ब्ल्यूटूथद्वारे मेसेज पाठवू इच्छितोय. ' असं या मेसेजमध्ये लिहिलेलं असतं. उत्सुकतेपोटी ' येस ' पर्याय स्वीकारल्यास कॅरीबे व्हायरस मोबाइलमध्ये शिरतो आणि मोबाइलची हार्ड डिस्क नेस्तनाबूत करून तात्काळ मोबाइल बंद पडतो.

गेल्या काही दिवसांत अनेक मोबाइलधारकांना याचा फटका बसला असून, कॅरीबेचा धोका वाढत चाललाय. यावर उपाय म्हणजे ब्ल्यूटूथच्या माध्यमातून आलेले मेसेज स्वीकारू नयेत. मेन्यूत ' कनेक्टिविटी ' मधील ब्ल्यूटूथ सीलेक्ट करायचं. त्यात ' माय फोन्स ' ( व्हिजिबिलिटी) ही साधारण ' शोन टू ऑल ' असते. ती बदलून ' हिडन ' ठेवायची. नंतर ब्लूटूथ ' ऑफ ' ठेवायचं. ही काळजी घेतल्यास कॅरीबे व्हायरसच्या संकटातून सुटता येईल.

यासंदर्भात सायबर क्राइम सेलचे इन्स्पेक्टर दीपक ढोले यांनी सांगितलं की, कायद्यानुसार व्हायरस पाठवणं गुन्हा असून त्यासाठी दंडाची शिक्षा होऊ शकते. अशा तक्रारी आल्यास आम्हाला संबंधितांवर कारवाई करता येतील, असंही ते म्हणाले.
