लेख: जाता जाता
पॉकेटमनी
[ Monday, February 14, 2005  08:46:31 pm]


' पुन्हा रेल्वेवाले आमच्या वॉलेटमध्ये हात घालणार! ' ज्यु. चकोरच्या बोलण्यात संताप होता.

' तुमच्या कसा काय! आमच्या! ' चकोरने बातमी आधीच वाचली असल्यामुळे त्याने दुरुस्ती केली.

' आमच्याच! आमचा पॉकेटमनी कमी होणार ना! ' ज्यु. चकोरीने आपल्या पिढीची बाजू मांडली.

' आजपर्यंत असं कधी झालं आहे का ? तुम्ही दोघं आईला मस्का मारून दर काही महिन्यांनी पॉकेटमनी वाढवून घेता. सरकारी कर्मचाऱ्यांचा डी. ए. वाढतो तसा. ' चकोरने आपल्याला गुपित कळलं आहे हे दाखवून दिलं.

' डॅड, आमची एक फ्रेंड आहे ना, तिला स्कुटी घेऊन दिली. स्टेशनपर्यंत जाण्यासाठी. ' तिने विषयांतर केलं.

' तिला सायकल द्या. तितकाच व्यायाम होईल रोज. ' चकोरीने आगीत तेल ओतलं.

' त्यापेक्षा तिला चालत जायला सांगा. रिक्षाचे आणि सायकलचे पैसे वाचतील आणि फिटनेसही वाढेल. ' भावाने बहिणीला चिडवलं.

' तू कोपऱ्यावर ब्रेड आणि बटर आणायलाही बाईकला किक मारणार आणि मला सांगणार एक्झरसाईज करायला. बाबा, याची बाईक आधी काढून घ्या! ' मुलीने तक्रार केली.

' मी सकाळी उठून जिममध्ये जातो तेव्हा तू झोपा काढत असतेस. ' दोघांमधलं भांडण वाढलं.

' हो, पण जिमची फी किती आहे ते सांगत नाहीस. ' ज्यु. चकोरी ऐकण्याच्या मन:स्थितीत नव्हती.

'' मला जिम ही नेसेसिटी आहे. ' ज्यु. चकोर बॅकफूटवर गेला.

' व्हॉट अबाऊट बाईक ?' बहिणीने भावाला बुनियादी सवाल टाकला.

' मोस्ट इसेंशियल कमोडिटी! ' चकोरने त्याला चिडवलं.

' यस, सगळ्या फ्रेंड्सना घेऊन भटकायला. ' ज्यु. चकोरी चिडलेली वाटली.

' मला एक सांगा, आपण कामं टाकून ही कसली चर्चा करतो आहोत. तुम्हाला काही उद्योग नाहीत काय ?' चकोरीने निर्णायक सुरात विचारलं.

' मुळात रेल्वेवाले तिकिटाचे रेट्स किती वाढवणार हे ठरल्याशिवाय आपण बोलतो आहोत. आणि सव्हिर्स चांगली द्यायची असेल तर रेट्स वर जाणारच. ' चकोरने मुद्दा मांडला.

' आधी सव्हिर्स सुधारा मग आमच्याकडून पैसे घ्या. ' ज्यु. चकोरने युक्तिवाद केला.

' पण पैसे नसतील तर सव्हिर्स कशी सुधारणार ?' ज्यु. चकोरीने पुन्हा ज्यु. चकोरला अडचणीत आणलं.

' त्या लालूंच्या राज्यात सगळे फुकट फिरतात. त्यांना तिकीट काढायला शिकवा. ' ज्यु. चकोर भयंकर संतापला.

' कोण शिकवणार ? लालूच ?' चकोरीने शांतपणे विचारलं.

' मला वाटतं, आणखी थोडं आपण थांबावं आणि बजेटनंतर डिस्कशनची सेकंड राऊंड करावी. ' चकोरने चर्चा गुंडाळली.
