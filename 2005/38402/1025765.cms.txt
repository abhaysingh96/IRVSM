'पेज थ्री'पासून पुढे...
[ Saturday, February 19, 2005  06:34:13 pm]


' चाँदनी बार ' मधल्या ' गोकुळ ' नंतर पुढच्या चित्रपटांमध्ये तुझ्या लायक भूमिका नाही, असं सांगणारा मधुर भांडारकर जेव्हा ' पेज थ्री ' या चित्रपटातील इन्स्पेक्टरची भूमिका उपेंदच करणार हे निर्मात्याला ठासून सांगतो, तेव्हा उपेंदमधल्या अभिनेत्याची ओळख चित्रपटप्रेमी रसिकांना होणार हे नक्कीच असतं !

मराठी दूरचित्रवाणी वाहिन्यांवर मालिकांचं पेव फुटलं असलं, तरीही ' त्याचा ' चेहरा मात्र मोजक्या वेळीच दिसतो. चित्रपट किंवा मालिकातील ' नायका ' ला शोभणारा त्याचा चेहरा नाही ; पण छोट्या, मोठ्या पडद्यावरचं त्याचं अस्तित्त्व अनेकदा नायकाच्या भूमिकेपेक्षाही सरस ठरलंय. काही दिवसांपूवीर्च प्रदशिर्त झालेल्या मधुर भांडारकरच्या ' पेज थ्री ' चित्रपटातला बेधडक इन्स्पेक्टर भोसले नुसताच भाव खाऊन जात नाही, तर उपेंद लिमये या गुणी कलाकाराला हिंदीतल्या ' चमचमत्या ' दुनियेतही भाव देऊन जातो.

पुण्यामध्ये अनेक वर्षं मराठी प्रायोगिक रंगभूमीवर ताकदीच्या भूमिका करणारा कलाकार, डेक्कनवरील ' कॅफे पॅरेडाईझ ' मध्ये गप्पात दंग झाला की, दोन-चार शिव्यांशिवाय वाक्य पूर्ण करू न शकणारा... मराठी मालिका, चित्रपटांमध्ये काहिशा आक्रमक भूमिकांतून छाप पाडणारा उपेंद मुंबईत पोहोचल्यावर मात्र बराच बदललाय. आपल्यापुरत्या आखून घेतलेल्या वर्तुळाच्या बाहेरील जगातही तो आता मोकळेपणानं वावरतोय हे ' पेज थ्री ' बघितलेल्या कोणाच्याही लक्षात येईल. ' चाँदनी बार ' मधल्या ' गोकुळ ' नंतर पुढच्या चित्रपटांमध्ये तुझ्या लायक भूमिका नाही, असं सांगणारा मधुर भांडारकर जेव्हा ' पेज थ्री ' या चित्रपटातील इन्स्पेक्टरची भूमिका उपेंदच करणार हे निर्मात्याला ठासून सांगतो, तेव्हा उपेंदमधल्या अभिनेत्याची ओळख चित्रपटप्रेमी रसिकांना होणार हे नक्कीच असतं !

इन्स्पेक्टर भोसले हा ' पेज थ्री ' मधील एक वल्ली आहे ; उपेंद लिमये प्रत्यक्षात आहे त्याच पद्धतीची, असं बिनधास्त कबूल करताना त्याचा आवाजातला भारदस्तपणा जाणवतो. मात्र, त्या वल्लीला पडद्यावर आकार देण्यासाठी मधुरच्या साथीनं खरंच अनेक क्लृप्त्यांचा वापर केल्याचंही तो मान्य करतो. त्यासाठी मेकअपविना काम करण्यापासून ते इन्स्पेक्टर भोसलेचं चालणं, वागणं, बोलणं आणि त्याचा पेहराव कसा असेल यांसारख्या अनेक गोष्टींवर मधुरशी सखोल चर्चा व्हायची. या भूमिकेसाठी मधुरने मला पूर्ण फ्रीडम दिल्याचा बराच उपयोग झाला. त्यामुळेच लिहिलेलं स्क्रिप्ट हातात नसूनही चित्रपटातील अनेक सीन्स खुलवल्याची आठवण उपेंद सांगतो. नव्या पिढीतल्या मुलीला थेट इंग्रजीतून जशास तसं उत्तर देणं असो वा गुन्हेगाराला कायद्याच्या चौकटीबाहेरची शिक्षा दिल्यावर ' क्राइम रिपोर्टर ' समवेतचा ' टेन्स ' सीन.... एकामागोमाग एक त्याच्या स्मृतीतल्या आठवणींना उजाळा मिळत जातो च्ङ्घश्ाह्व द्धड्ड 1 द्ग ह्लश्ा ड्ढद्ग द्बठ्ठ ह्लद्धद्ग ह्य 4 ह्यह्लद्गद्व ह्लश्ा ष्द्धड्डठ्ठद्दद्ग ह्लद्धद्ग ह्य 4 ह्यह्लद्गद्वज् हे चित्रपटात अतुलच्या तोंडी असलेलं वाक्यच इन्स्पेक्टरच्या भूमिकेचंही ' सार ' असल्याचं मत तो प्रांजळपणे व्यक्त करतो.

आजच्या स्पधेर्च्या जगात टिकायचं म्हणजे मुंबईला पर्याय नाही, हे सत्य स्वीकारून त्याने अनिश्चिततेच्या दुनियेत पाऊल ठेवलंय ; पण म्हणून कोणत्याही तडजोडी करायला त्याचा स्पष्ट नकार असतो. म्हणूनच पैसा, नाव, प्रसिद्धी अशा साऱ्या ' ग्लॅमर ' मिळवून देणाऱ्या दूरचित्रवाणीवरील दैनंदिन मालिका त्याला ' रोजगार हमी योजना ' वाटतात. मात्र, तरीही लोकांच्या आणि कलाकारांच्याही बदलत्या जीवनशैलीचा अविभाज्य भाग बनलेल्या या मालिकांमध्ये काम करण्याचं त्याला वावडं आहे, असंही नाही. फक्त ' दजेर्दार ' या लेबलखाली मोजक्या मालिकांमध्ये तो दिसतोच आहे.

' पेज थ्री ' नंतर उपेंदचं अभिनयाचं जग रुंदावलंय. चित्रपटासाठी अभिनंदन करणारे आणि आगामी प्रोजेक्टविषयी विचारणा करणारे फोन घेऊन तो दमलाय, तरीही हुरळून मात्र गेलेला नाही! कारण, कोणतीही गोष्ट ठरवून होत नसते, असं तो मानतो. किंबहुना, प्रत्येक यशानंतर ' काहीही प्लॅन न करण्याचा ' प्लॅन तो दरवेळी करतो. आयुष्यात येणारा एखादा ' टनिर्ंग पॉईंट ' ठरवून आणता येत नाही किंवा ठरवून चुकतही नाही.... उपेंदच्या गणिताचं हे समीकरण! ' स्क्रीन प्रेझेन्स ' बघूनच मला बऱ्याचदा आक्रमकतेची झालर असणाऱ्या भूमिका दिल्या जातात, ही त्याची कैफियत ऐकताना त्याच्यातला अभिनेता उत्तमोत्तम भूमिकांसाठी आसुसलेला असल्याची साक्ष पटते. म्हणूनच , ' पेज थ्री ' मधील इन्स्पेक्टर भोसलेला लोकप्रियता लाभली असली, तरी मुंबईत गोरेगावच्या घरातून सेटवर पोहोचताना त्याचा बसस्टॉप सुटलेला नाही.

रूपेरी दुनियेत ' स्टार ' म्हणून नाही, तर नासिर, ओम पुरी यांच्यासारखं ' अॅक्टर ' म्हणून ओळखलं जावं ही त्याची मनीषा! अमोल पालेकर, विनय आपटे, सचिन खेडेकर यांच्या सहवासामुळे बनलेल्या साच्यातून उपेंदच्या अभिनयाची जडण- घडण झाली आहे. त्याच्यातल्या अभिनेत्याचा प्रवास आता खरा सुरू झालाय....!
