विज्ञान-तंत्रज्ञान
याला 'ब्लॉगिंग' ऐसे नाव!
[ Saturday, February 19, 2005  11:01:34 pm]


' वेब मॅगझिन ' ला शक्य न होणाऱ्या दोन गोष्टी ' ब्लॉग ' मुळे होतात. पहिलं म्हणजे ते पूर्णत: खासगी असतात. ' ब्लॉग ' वर व्यक्त होणाऱ्या मतांना ' पर्सनल टच ' प्राप्त होतो. त्यामुळे वाचकांना लेखकाच्या हेतूबद्दल खात्री पटते. पुस्तकाची पानं ' पीडीएफ फॉर्म ' मध्ये रुपांतरीत करून त्यांची इंटरनेटवर थेट विक्री करता येऊ शकते.

जग जवळ आणणारं इंटरनेट हे सातत्यानं बदलत जाणारं माध्यम आहे. झटपट संवाद साधण्याचं माध्यम , ' ऑन लाइन ' माहितीचा खजिना , ' ई- कॉमर्स ' चं व्यासपीठ अशा अनेक माध्यमांतून आपला इंटरनेटशी घरोबा निर्माण झालाय. आता त्याचं ' क्रांतीकारी ' या सदरात मोडणारं आणखी एक रूप पुढे आलंय आणि ते झपाट्यानं पसरतंय. ते म्हणजे ' ब्लॉगिंग '.

थोडक्यात सांगायचं तर ' ब्लॉगिंग ' म्हणजे ' ऑन लाइन डायरी '. ' वेब ब्लॉग ' या पूर्ण नावाचा तो शॉर्टकट. इंटरनेटच्या मायाजालात आपले विचार व्यक्त करण्यासाठी, इतरांशी वैचारिक जवळीक साधण्यासाठी 1994 पासून हा प्रकार सुरू झाला. जॉन बार्गर याने 1998 मध्ये या पद्धतीला ' ब्लॉगिंग ' असं नाव दिलं. अवघ्या दशकभरातच त्यानं इंटरनेट वापरणाऱ्यांवर मोहिनी घातली आहे.

' ब्लॉग ' वर लिहिला जाणारा मजकूर केवळ त्या लेखकाची मतं आणि भावनाच उलगडतो असं नव्हे, तर इतरांना त्याच्या मतांवर वाद- प्रतिवाददेखील करता येतो. त्याचं छोटंस वाद-विवाद मंडळच तयार होतं. त्याच्यावर आपण स्वत:बद्दल, कुटुंब किंवा मित्रांबद्दल अथवा समस्या आणि विविध मुद्द्यांवरही लिहू शकतो. इतकंच नव्हे, तर तांत्रिक माहितीची देवाण- घेवाणही त्याच्यावरून होऊ शकते.

कला, साहित्य, व्यापार, सेलेब्रिटी, फूड, शिक्षण, कायदा, चित्रपट, बातम्या, फोटो, राजकारण, क्रीडा, पर्यटन, संस्कृती अशा अनेक विषयांना वाहिलेले ' ब्लॉग ' इंटरनेटवर उपलब्ध आहेत. ही अर्थातच ' पेड सव्हिर्स ' असली, तरी तिने जगाला आपल्या घराच्या ' ड्रॉइंग रूम ' इतकं लहान करून टाकलंय.

गेल्या वर्षी घडलेल्या दोन मोठ्या घटनांनंतर जगाला ' ब्लॉग ' च्या लोकप्रियतेची प्रचीती आली. पहिली घटना अमेरिकी अध्यक्षपदाची निवडणूक आणि दुसरी त्सुनामीचा प्रलय. अमेरिकेत जॉर्ज बुश सत्ता राखणार की जॉन केरी त्यांच्यावर मात करणार या चढाओढीत ' ब्लॉग ' वर विचारांचा काथ्याकुट झाला. या निवडणुकीतील इराकचं युद्ध, डळमळती अर्थव्यवस्था, दहशतवाद यापासून तर गर्भपातापर्यंतच्या प्रत्येक मुद्द्यावर ' ब्लॉगर्स ' नी ठामपणे मतं मांडली. त्यामुळे प्रचंड जनजागृती घडून आली, ही वस्तुस्थिती आहे.

त्सुनामीच्या प्रलयानंतर दक्षिण आशियात झालेली हानी जगानं टीव्हीवर पाहिली आणि तिची भरपाई करून देण्यासाठी आपल्या मदतीचा हात पुढे केला, तो ' ब्लॉग ' वर. त्सुनामीनंतरच्या सात दिवसांत तब्बल 10 लाख जणांनी ' त्सुनामी ब्लॉग ' ला हिट दिली.

पाश्चिमात्त्य देशांत तर ' ब्लॉगिंग ' नं पत्रकारितेचं रूप धारण केलंय. ' ब्लॉगर्स ' ला पत्रकार म्हणावं की नाही, याचा वाद संपुष्टात आला नसतानाही अमेरिकेत निवडणुकीच्या वेळी ' ब्लॉगर्स ' ना पत्रकारांप्रमाणे अधिस्वीकृती देण्यात आली. मुख्य प्रवाहातल्या बड्या वृत्तपत्रांनीही आपापल्या वार्ताहरांच्या नावे ' ब्लॉग ' सुरू केले आहेत.

' वेब मॅगझिन ' ला शक्य न होणाऱ्या दोन गोष्टी ' ब्लॉग ' मुळे होतात. पहिलं म्हणजे ते पूर्णत: खासगी असतात. ' ब्लॉग ' वर व्यक्त होणाऱ्या मतांना ' पर्सनल टच ' प्राप्त होतो. त्यामुळे वाचकांना लेखकाच्या हेतूबद्दल खात्री पटते. दुसरं म्हणजे लेखन प्रसिद्ध होण्यासाठी कुठल्याही प्रकाशकाकडे खेटे मारावे लागत नाहीत. लेखनाचे संपादन करावे लागत नाही. त्यामुळे संपादन करणाऱ्याचा दृष्टिकोन त्यात डोकावत नाही. या लेखनाचा जाहिरातींशी संबंध नसल्यामुळे जाहिरातदारांच्या प्रभावापासूनही ते दूर राहते. शिवाय जगभरातला वाचक प्रचंड संख्येने मिळतो. ही संख्या अनेकदा बड्या वृत्तपत्रांच्या रिडरशीपपेक्षाही जास्त असते. पुस्तकाची पानं ' पीडीएफ फॉर्म ' मध्ये रुपांतरीत करून त्यांची इंटरनेटवर थेट विक्री करता येऊ शकते.

ही सगळी कामं वेबसाइट किंवा ' चॅटिंग ' द्वारेही शक्य आहेत. मग ' ब्लॉग ' चं वेगळेपण काय ? वेबसाइटचं स्वरूप सर्वसाधारणपणे बदलत नाही. ते ' स्टॅटिक ' प्रकारात मोडतं. ती ' अपडेट ' होते तीही पानांच्या स्वरुपात. ' ब्लॉग ' प्रत्येक नोंदीनंतर ' अपडेट ' होतात. ङ्खद्धड्डह्ल ङ्घश्ाह्व स्द्गद्ग ढ्ढह्य ङ्खद्धड्डह्ल ङ्घश्ाह्व त्नद्गह्ल या तत्त्वाचा वापर करावयाचा झाल्यास वेबसाइट डिझाइन करण्यासाठी तांत्रिक माहिती आवश्यक आहे. ' ब्लॉग ' मध्ये त्याची तितकीशी निकड नाही. वेबसाइट कशी तयार करावी, याची माहिती असणारे वेबलॉग तयार करू शकतात. निव्वळ नोंद करून ' ब्लॉग ' च्या दुनियेत प्रवेश करता येतो. वेेबसाइटवर ' मेसेज बोर्ड ' किंवा ' गेस्ट बुक ' सारख्या सुविधा परस्परांशी संवाद साधण्यास उद्युक्त करतात. ' ब्लॉग ' या संवादावर प्रतिक्रियाही उपलब्ध करून देतो.

' ब्लॉग ' ची यादी 222. ष्ड्डद्व 2 श्ाह्मद्यस्त्र.ष्श्ाद्व किंवा 222. ह्ल 4 श्चद्गश्चड्डस्त्र.ष्श्ाद्व या बेवसाइटवर मिळू शकेल.
