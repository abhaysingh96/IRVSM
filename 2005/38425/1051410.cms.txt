लेख: जाता जाता
हत्तीराज्य!
[ Monday, March 14, 2005  08:52:02 pm]


' माणूस आणि हत्ती यांची लढाई सुरू झाली असं दिसतं! ' चकोरीने म. टा.वर नजर टाकली आणि भाष्य केलं.

' गेली अनेक वर्षं चालू आहे. तुला आज साक्षात्कार झाला. ' चकोरने शांतपणे तिला सांगितलं.

हा काय मामला आहे हे बघायला ज्यु. चकोरने पेपर पुढे ओढला.

' हत्तींनादेखील, आंबे, फणस आवडतात वाटतं! ' त्याने त्याच्या सवयीनुसार गजराजाच्या खाण्याचा मुद्दा लक्षात ठेवला.

' आणि तुझ्यासारखाच भातखाऊ असतो तो! ' चकोरीने मुलाच्या भातप्रेमावर बोट ठेवलं.

' त्यामुळेच तो इतका बुद्धिमान असतो. ' मुलाच्या या वाक्यावर ज्यु. चकोरी फसकन हसली. ' इतकं सोप्पं असतं तर सगळ्यांनी भातावरच ताव मारला असता. ' तिने कॉमेंट केली.

' पण भाताचं महत्त्व कळण्याइतका मेंदू काहींना नसतो हीच तर गडबड असते ना! ' भावाने बहिणीला चिडवलं.

यावर काय आर्ग्युमेंट करावं हे ज्यु. चकोरीला सुचेना.

' आज तू जाणार आहेस का महिला मंडळात ?' तिने विषय बदलण्यासाठी आईला विचारलं.

पण चकोरीला हत्ती हा विषय जास्त प्रिय असावा.

' महिला मंडळापेक्षा हत्तीबद्दल आणखी माहिती कुठे मिळते हे बघायला हवं. ' तिने आपल्या कार्यक्रमाचं सुतोवाच केलं.

' काय हो तुम्हाला काही सुचतं का! ' तिने चकोरला विचारलं.

' नेटवर बरीच माहिती मिळेल. ' चकोरने शांतपणे सांगितलं.

' आई मी तुला मदत करते. आपण नेटवर बसू थोड्या वेळानं. ' नेट म्हटल्यावर कन्येच्या अंगात उत्साह संचारला.

' तु तुझा अभ्यास कर! मी बघते नेट वगैरे. ' चकोरीने तिच्या उत्साहाला आवर घातला.

कन्येचा चेहरा पडला. ' आई मलापण कळेल हत्तीबद्दल. इंटरव्ह्यूमध्ये उपयोगी पडेल. ' तिने मार्ग शोधला.

' माणसाला हत्तीची हुषारी पूवीर्च जाणवली असणार. त्यामुळे त्याने त्याला गजाननाच्या रूपात बसवून हत्तीचा मान राखला. ' चकोरीने गणपतीचं मुख देवाला जोडण्यामागचं तर्कशास्त्र सांगितलं.

' पण याच माणसानं जंगलं तोडून हत्तीला बेघर केलं आणि हत्ती माणसांच्या जंगलात येऊ लागल्यावर त्याला हकलवण्यासाठी मोहिमा काढल्या. ' चकोरने फटकन सांगून टाकलं. त्याच्या बोलण्यावर तिघेही एकमेकाकडे बघू लागले.

' नशीब समजा, हत्तीच्या मेंदूत इतकीच समज आहे. उद्या जर, उत्क्रांती होत होत, त्याच्या लक्षात आलं की माणूस आपल्याला घाबरून सैरावैरा पळतो, तर माणसाला पळायला जंगलंही उरणार नाहीत. ' चकोरने व्यक्त केलेल्या या भीतीवर तर तिघांना काय बोलावं हेच सुचेना.

चकोर
