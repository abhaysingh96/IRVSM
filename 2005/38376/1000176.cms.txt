लेख: जाता जाता
अरबट-चरबट!
[ Monday, January 24, 2005  11:13:04 pm]


' वाचा दोघांनी आणि विचार करा जरा! ' चकोरीने म. टा. मुलांच्या पुढ्यात भिरकावला.

मुलांनी आज्ञाधारकपणे पेपर चाळला.

' काय ग आई, कोणती बातमी! ' ज्यु. चकोरीने जणू काही आपल्याला कळलंच नाही अशा आविर्भावात विचारलं.

' अरबट चरबट खाता तुम्ही. बंदी आणली पाहिजे तुमच्यावर! ' चकोरीने ठाम विधान केलं. ' मॉम, आम्ही नाही खात असलं काही. अगदीच भूक लागली तर... ' ज्यु. चकोरने बाजू मांडायचा प्रयत्न केला. ' पण तुम्हाला नेहमीच भूक लागलेली असते. जेवण झाल्यावरदेखील तुम्ही लगेच डबे शोधता! ' चकोरीने टोला हाणला.

' खायचं वय आहे त्यांचं. किती बोलतेस त्यांना! ' इतका वेळ गप्प असलेल्या चकोरने मध्यस्थी केली.

' मी त्यांच्या खाण्याविरुद्ध नाही. फक्त बाहेर काहीतरी खाऊ नका इतकंच सांगते आहे. ' चकोरीने समजुतीचा सूर लावला.

' तुझ्या हातच्या खाण्यावर उतारा म्हणून ते बाहेर हादडत असतील. ' चकोर तिला चिडवलं. ' उतारा कशाला! नेहमीच बाहेर चरा. माझं काही म्हणणं नाही. फक्त आजारी पडलात की रात्रभर मला जागरणं करायला लावू नका! ' चकोरीने एकदम निर्वाणीची भाषा केली.

तिच्या उद्गारावर एकदम सीन बदलला.

ज्यु. चकोरने पेपरात डोकं खुपसलं. कन्येने पित्याकडे ' आता काय करायचं ' अशा नजरेने बघितलं.

चकोरने हात झटकून ' तुमचं तुम्ही बघा! ' असा मूक सल्ला दिला.

' आई, बाबांच्या बोलण्यावर इतकी कशाला चिडतेस! आम्ही काही बोललो का तुला! ' मुलीने आईची समजूत काढण्याची मोहीम सुरू केली.

' बोलायला कशाला हवं! तुम्हाला वाटतं की मला काही उद्योग नाहीत म्हणून मी आपली तुमच्या जिभेचे चोचले पुरवते! ' चकोरीचा गुस्सा कमी झाला नव्हता.

' आई, अगं तुझ्या हाताची सर कशाला येणार नाहीच. पण आम्हाला कधीतरी वाटतं, की रस्त्यावर पाणीपुरी, रगडा पॅटिस हाणावा. चुरचुरीत, गरम डोसा खावा. ' मुलीने किल्ला लढवणं चालू ठेवलं.

' अगं तुमच्या वयात आम्हालाही असंच वाटायचं. फरक इतकाच की तुमचा महिन्याचा पॉकेट मनी हा आमचा वर्षाचा असे. ' चकोरी अचानक निवळली.

चकोरने जोरात सुस्कारा सोडला. वातावरण इतक्या लवकर निवळल्याबद्दल तो खुष झाला.

' आता काय बेत आहे ?' त्याने विचारलं.

त्याच्या मनचा भाव ओळखून चकोरी जोरात हसली.

' आज यांच्या आवडीचा पिझ्झा! ' तिने मुलांकडे बोट दाखवत घोषणा केली.

दोघांचे डोळे विस्फारले.

चकोर
