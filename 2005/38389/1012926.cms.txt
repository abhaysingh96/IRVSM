लेख: जाता जाता
राणे-राज्य
[ Sunday, February 06, 2005  07:07:03 pm]


' मी, मुंबईसुन नारायण राणे बोलतंय. आमचे सी.एम.साहेब आसत थय ?'

पलीकडून आवाज येतात.

' सर, मुंबईहून कोणीतरी आपलो सोयरो नारायण राणे उलयता. '

' हॅलो, राणे स्पीकिंग. '

' राणेसरकार, मी, नारायण राणे. तुमचा अभिनंदन करुचा असा. हय नाय तर थय राणेंचा सरकार इला. लय बरा वाटला आमका. '

काहीशा घाबरलेल्या आवाजात पलीकडून विचारणा होते.

' म्हंजे ?'

' समस्त राणेखानदानाक! '

सुटकेचा नि:श्वास टाकल्याचा आवाज येतो.

' आमच्या मालवणाक तुमका आमंत्रण देवूक येवचा असा पणजीत. '

' सावकाश ये. सध्या हांव बजेटच्या गडबडीन आसा. '

' माझ्या बजेटवरील भाषणांचा पुस्तक तयार असा. पाठवून देतंय. उपयोगच होईत तुमका. '

' नका, नका, ते विरोधी पक्षासाठी बुक आसा. '

इकडून हळूच आवाजात विचारणा होते.

' माका एक सांगा, राणेसरकार, थय राणेंक सी.एम. बनूक काय अडचण येऊक नाय! '

तिकडून आश्चर्याचा स्वर.

' अडचण, काय ना! '

' काय असा, हय कोणी राणे नावाचो मुख्यमंत्रीपदाच्या शर्यतीत उतरतो म्हटला की, सगळे जोशी, मुंडे एक होतत. मग ते ठाकरेंक सांगतत की आता महाराष्ट्राक वाचवूक फक्त ठाकरे आडनावाच्या माणसाची गरज असा. मग दोन ठाकरे सगळ्यांका सांगतत की माका सी.एम. बनण्यात रस असा. पण उमेदवार निवडून आणूचे पडतत ह्या त्यांका कोणी सांगना नाय. '

पलीकडून पुन्हा चक्रावल्या आवाजात विचारणा होते.

' कोण ठाकरे! '

इकडून त्याहून दुप्पट आवाजात आश्चर्य व्यक्त होते.

' तुमका ठाकरे ठाऊक नायत ?'

पलीकडून उत्तर येते.

' हांव, गांधी घराण्याक ओळखता. '

इकडून संमती दर्शवल्याचा आवाज.

' हां ताबी खरा असा. '

पुन्हा इकडून विचारणा होते.

' तुमका सी.एम. बनवूचो निर्णय इतक्या लवकर कसो झालो ?'

' त्याचे अशे झाले, आमचो रिमोट कंट्रोल आसा दिल्लीत. त्यांका कोणाक सी.एम. बनण्यात इंटरेस्ट ना. रिमोट कंट्रोलची सून येवपाची आसा, जावयाक काय घेणा नसा गोेवा स्टेटचे देणेघेणे नो. आणि हाव दिल्लीत मी सांगलंय महाराष्ट्रात, मुंबईत कोणत्याही राणेंशी माझे काय संबंध ना. '

इकडे सन्नाटा पसरतो आणि पलीकडे, हसल्याचा आणि फोन ठेवल्याचा आवाज येतो.

चकोर
