लेख: जाता जाता
'वाघ' संकटात!
[ Sunday, March 27, 2005  08:32:35 pm]


फोन वाजतो. ' हॅल्लो, कोण पाहिजे ?' खरखरीत आवाजात विचारणा होते.

' मी राज. वर्किंग प्रेसिडेंट आहेत का ?'

' मराठीत बोला. कार्यकारी अध्यक्ष म्हणतात त्यांना. '

' अस्सं होय! मला वाटलं त्यांना काम नसतं तेव्हा वर्किंग प्रेसिडेंट म्हणतात. बरं, मग कार्यकारी अध्यक्ष आहेत का ?' शांतपणे राज विचारतात.

' बघतो हां, इथे पीएंशी बोला. ' फोनमधून आवाज ऐकायला येतात.

' हॅल्लो, कोण पाहिजे ?' पुन्हा विचारणा.

' कार्यकारी अध्यक्ष आहेत काय ?' राज.

' वर्किंग प्रेसिडेंट म्हणतात त्यांना. काय काम आहे ?' रोखठोख जबाब मिळतो.

' जरा वाघांबद्दल बोलायचं होतं. ' राजच्या स्वरात नम्रता.

' वाघांबद्दल बोलायला त्यांना टाइम नाही. आता समर व्हेकेशनमध्ये ते त्यांचे फोटो काढायला जाणार आहेत. ' पीएंचं उत्तर.

' वाघांना समर व्हेकेशन असते वाटतं! ' राजच्या स्वरात मिस्किलपणा.

' नाय हो, वर्किंग प्रेसिडेंटना. ' पीएंचा खुलासा.

' बरं साहेब, जरा काम सांगा ना पटकन्. सुट्टीचं बुकिंग करायचं आहे फोनवर. दोन-तीन अभयारण्यांत फोन लावायचे आहेत. आता निरुपम नाहीत ना ही कामं करायला. ' पीएंची विनंती.

' वाघ संकटात आहे. जरा बोलायचं होतं. ' राज यांचा खुलासा.

' ते वाचलं वर्किंग प्रेसिडेंटसाहेबांनी पेपरात. मोहन आणि सुरेशजींना पार्लमेंटमध्ये आवाज उठवायला सांगितलं आहे. मीच मोबाईल लावून दिला होता सकाळी. ' पीएंचं स्पष्टीकरण आणि त्यानंतर ' मग काय सांगू साहेबांना! ' अशी पीएंची विचारणा.

' सांगा महाराष्ट्रातील वाघ संकटात आहेत. त्याबद्दल बोलायचं आहे. ' राजवाणी.

' पण साहेब, बातमीत आपल्या राज्यातील वाघांचा उल्लेख नव्हता. नाहीतर जिल्हा-प्रमुखांना कामाला लावलं असतं. ' पीएंची शंका.

' गेली काही वर्षं इथले वाघ सैरभैर झाले आहेत. वर्किंग प्रेसिडेंटच्या लक्षात आलं नाही वाटतं! ' राज यांचा खुलासा.

' बहुधा नाही. कारण मिटिंग नाही लावली कधी यासाठी. ' पुन्हा पीए.

' त्यांना सांगा की, आपल्या अभयारण्यातून अनेक वाघ राष्ट्रवादीच्या नाहीतर इतर कोणत्या तरी जंगलात जायच्या मार्गावर आहेत. आता आपली सत्ता नसल्यामुळे काही जमत नाही म्हणतात. मला खूप वाघ भेटतात. ' राज यांचं स्पष्टीकरण.

' साहेब, तुम्ही बोलता ते वाघ आपल्याला समजत नाहीत. तुम्ही वर्किंग प्रेसिडेंट-साहेबांशीच बोला. समर व्हेकेशनच्या सुट्टीवरून आल्यावर. ' पीएंची विनंती.

राज निराशेने फोन ठेवतो.

चकोर
