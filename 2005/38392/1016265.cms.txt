लेख: जाता जाता
देशाचे कोण!
[ Wednesday, February 09, 2005  08:12:13 pm]


' आज पहिल्या पानावर साहित्य संमेलन भरलेलं दिसतं आहे. ' चकोरने म. टा.वर नजर टाकत कॉमेंट केली. '

' पण संमेलनातल्या मारामाऱ्या, रुसवे फुगवे यांना वगळून. ' चकोरीने खवचटपणा केलाच.

पाकिस्तानची मॅच मुंबईत होणार की नाही इतका मर्यादित रस असणाऱ्या ज्युनियर कंपनीने आता आईबाबांच्या लाडक्या साहित्यबातम्यांकडे नजर वळवली.

' दोघेही ग्रेटच आहेत ना! ' मुलीने प्रशस्तीपत्र दिलं.

' तुला आज कळलं वाटतं! ' मुलाने तिला चिडवलं.

' सुर्वे यांच्या कविता वाचल्यापण आहेत आणि ऐकल्यादेखील. तू भंकस काहीतरी इंग्लिश पल्प फिक्शनवाचत असतोस. ' मुलीने हल्ला परतवला.

' कविता हा माझा एरिया नसला तरी पाटलांचं ' पानिपत ', ' झाडाझडती ' वगैरे नॉव्हेल्स मी वाचल्या आहेत. ' मुलाने आपलं वाचन किती अफाट आहे हे दाखवलं.

' तू वाचलंस पानिपत! ' चकोरच्या विचारण्यात आश्चर्य आणि अविश्वास होता.

' मी शेजवलकरांचं पानिपतवरचं संशोधनही वाचलं आहे. ' मुलाने वडिलांना धक्का दिला.

' मग कादंबरी जास्त चांगली की संशोधन! ' चकोरने विचारलं.

' इट डिपेंड्स! कोणाला काय आवडतं यावर अवलंबून असतं. ' मुलाने डिप्लोमॅटिक उत्तर दिलं.

' आणि महानायक! ' चकोरीने विचारलं.

' महानायक बाजारात आलं तेव्हाच वाचलं आणि त्यातील चुकांबद्दल रंगलेला वादही. ' मुलगा फॉर्मात होता.

' अभ्यास करायचा नसला की वेळ भरपूर मिळतो. ' ज्यु. चकोरीने चिमटा काढला.

' काहींना फक्त मैत्रिणींबरोबर खादडीगिरी करायला वेळ असतो. वाचणार केव्हा ?' मुलाने प्रत्युत्तर दिलं.

ज्यु. चकोरीने लगेच सुर्व्यांच्या कविता म्हणायला सुरुवात केली. ' तेव्हढंच पत्रात लिवा.... '

हल्लीची पिढी वाचत नाही, मराठी तर अजिबात नाही हे आपलं मत चुकीचं आहे, असे भाव चकोरीच्या चेहऱ्यावर पसरले. ' काय बाबा इतका विचार कसला करताहात ?' अचानक मुलीने विचारलं.

' या बातमीतलं एक वाक्य महत्त्वाचं आहे. ' चकोरने खुलासा केला.

' कोणतं ?' तिघांनी एका सुरात विचारलं.

' सुभाषचंद बोस केवळ बंगाली नेता म्हणून उरले याचे विश्ोषण महानायकमध्ये आढळते. ' चकोरने वाचून दाखवलं.

' मग! ' मुलाने विचारणा केली.

' शिवाजी महाराज, टिळक महाराष्ट्राचे, बोस बंगालचे, रणजिंतसिंह पंजाबचे, राणा प्रताप राजस्थानचे. देशाचे कोणीच नाहीत. ' चकोरच्या बोलण्यात हताशपणा होता.

चकोर
