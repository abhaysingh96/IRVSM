अर्थव्यवहार
ब्रॅण्ड्स विक्रीसाठी सेक्सचा वापर हिंदुस्तान लिव्हरला मदतकारक ठरेल?
[ Friday, May 13, 2005  08:11:06 pm]


विकास कुमार

गोष्ट जेव्हा हृदयाची असते तेव्हा ग्राहक सेक्सशी संबंधित जाहिरातीला हमखास वश होतो ( किंवा बळी पडतो). हिंदुस्तान लिव्हर लि.ने ( ' एचएलएल ') आपल्या ' लिरिल ' व ' क्वॉलिटी वॉल्स ' या दोन उत्पादनांच्या ( ' ब्रॅण्ड्स ' च्या) सुरू केलेल्या जाहिरातींच्या मोहिमांमध्ये माणसाच्या याच सर्वात मूलभूत सहजप्रवृत्तीला फुंकर घातली आहे.

पण या संदर्भात सेक्सचा नुसता उल्लेख केला तरी ' एचएलएल ' चे कार्यकारी संचालक ( आइसक्रीम) जे. एच. मेहता आपल्या खुचीर्तून एकदम उठून म्हणतात: त्या ( जाहिराती) ' सेक्स्युअॅलिटी ' बद्दलच्या नाहीत! त्या ' सेन्स्युअॅलिटी ' ची संबंधित आहेत! '

आइसक्रीम हा पदार्थ तसा मौजमजेच्या वर्गवारीत मोडतो. मेहता म्हणतात: ' क्वॉलिटी वॉल्स ' हा पदार्थ ' मल्टि सेन्सोरियल एक्स्पिरियन्स ' साठी असल्याचा संदेश जाहिरातींतून लोकांत पोहचविणे आवश्यक आहे.

मॅकॅन् एरिक्सनचे प्रेसिडेंट संतोष देसाई यांच्या मते अशी जाहिरात ही ग्राहकाच्या बदलत्या वृत्तीची निदर्शक आहे. ते म्हणतात ,' यापूवीर् लोक जे स्वीकारायला तयार नव्हते त्यासाठी आता तयार झाले आहेत याची ही जणू पावतीच आहे. आता आपण अशा काळाकडे चाललो आहोत जेथे निर्बंधांशिवाय आनंद साजरा करणे मान्य केले जाईल आणि सेक्स ही अशा आनंदाचीच एक अभिव्यक्ती आहे. '

आता ' लिरिल ' ची जाहिरात घ्या. तीत एक तरूण जोडपे खेळकरपणे परस्परांची छेडछाड करीत आहे. जाहिरात व तिच्यातील भावना पाहता ती एखाद्या ' कंडोम ' ची जाहिरात म्हणूनही सहज चालू शकेल. मग परंपरेपासून असे भलतीकडे जाणे का ?

' एचएलएल ' चे व्हाइस प्रेसिडेंट (स्किन केअर) अशोक वेंकटरमणी याबद्दल स्पष्टीकरण देताना सांगतात: ' लिरिल ' ब्रॅण्ड बाजारात पूवीर्सारखा खपत नव्हता. त्यातील प्रमुख घटक- लिंबू हा तर सगळीकडेच असलेला आणि सर्वसाधारण बनला होता. त्यामुळे बदलत्या काळात त्याचे वैशिष्ट्य असे उरलेले नव्हते. म्हणून ' प्रॉडक्ट मिक्स ' सुधारण्यात आले. या साबणात अलोय व्हेरा (कोरफडीचा अर्क) मिसळण्यात आला, ज्यामुळे त्वचा मुलायम बनते. या जाहिरातीतून मग संदेश दिला जातो ,'' इन अ मॅड र्वल्ड, व्हेअर यू आर रशिंग अगेन्स्ट टाइम , ' लिरिल ' कॅन स्पार्क इन्टिमेंट मोमेन्ट्स विथ युवर स्पाउस. '' लोवे लिंटासचे मुख्य कार्यकारी अधिकारी प्राणेश मिश्रा म्हणतात: ' लिरिल ' च्या जाहिराती या नेहमीच ताजेपणा, तरूण आणि तरुणाईच्या प्रतिमेच्या प्रचार करणाऱ्या राहिलेल्या आहेत. ही जाहिरात म्हणजे ताजेपणा, टवटवीचा आणखी एक आविष्कार आहे.

मात्र अशा प्रकारचे ' सेक्शुअली लोडेड कम्युनिकेशन ' हे ब्रॅण्डला लाभ मिळवून देण्याऐवजी नुकसानच अधिक करील, असेही अनेकांना वाटते.
