महामुंबई
स्वदेस इफेक्ट!
[ Sunday, January 16, 2005  12:05:32 am]


मुकेश माचकर

मुंबई : भारतातल्या ऊर्जा क्षेत्रात पायाभूत यंत्रणा उभारणाऱ्या इंजीनियर मंडळींचं मॅगझिन आहे ' पॉवर स्ट्रीम '... त्यांनी या वषीर्चा राष्ट्रीय जल विद्युत इंजीनियर पुरस्कार देण्यासाठी ' मोहन भार्गव ' ची निवड केली आहे... मोहन भार्गव हा कोणी हाडामांसाचा माणूस नाही... हा एका सिनेमाचा हीरो आहे... ' स्वदेस ' चा!...

सोनिया गांधी यांच्या अध्यक्षतेखालील राष्ट्रीय सल्लागार परिषदेला त्यांच्या ग्रामीण उद्धार कार्यक्रमासाठी ' स्वदेस ' हवा आहे... अशीच मागणी अनेक स्वयंसेवी संस्थांनी केली आहे...

सुरुवातीला, काय असेल हा प्रकार, म्हणून थोड्या शंकेने आणि ' स्वदेस ' द्वेष्ट्यांनी पहिल्या दिवसापासून प्रसारमाध्यमांतून चालवलेल्या अपप्रचारामुळे बहुसंख्य ' स्वदेस ' वासी जवळपास आठवडाभर सिनेमाकडे वळाले नव्हते. तेच आता मात्र या सिनेमाला गदीर् करू लागले आहेत, तो पाहून इतरांना जोरदार शिफारस करू लागले आहेत. व्यावसायिक हिंदी सिनेमाची भाषा बोलतानाच त्याच्या घिस्यापिट्या चौकटीत न अडकणाऱ्या या सिनेमानं, सिनेमाच्या कक्षेपलीकडचा ' स्वदेस इफेक्ट ' निर्माण केला आहे...

जागतिक बँकेचे महाराष्ट्रातील पाणी, जलसिंचन आणि जलनिस्सारण प्रकल्पाचे अधिकारी प्रवीण दराडे यांनी राज्यभरातल्या पंचायत समिती सदस्यांना, बीडीओंना आदेश काढला की, जिथे लागला असेल तिथे जाऊन हा सिनेमा बघा. ज्यांच्या भागात तो लागला नसेल, त्यांनी तिकडे प्रिंट मागवून विशेष शो करा '!

तिकडे दिल्लीत शरद पवारांबरोबर ' स्वदेस ' चा शो पाहिल्यानंतर रेणुका चौधरींनी ' हा सिनेमा तात्काळ तेलुगूत डब कर, आमच्या आंध्रातल्या प्रेक्षकांना तो पाहायला मिळाला पाहिजे ' असा हेका धरला आहे...

एखाद्या सिनेमाला प्रेक्षकांचं प्रेम मिळणं ही अगदी स्वाभाविक प्रतिक्रिया असते... पण, आशुतोष गोवारीकरला ' स्वदेस ' नं प्रेमाबरोबरच आदर मिळवून दिला आहे, जो अत्यंत दुमिर्ळ असा ' इफेक्ट ' आहे... आजवर लोक संतमहंतांच्या-देवादिकांच्या भूमिका केलेल्या नटांचं ' दर्शन ' वगैरे घ्यायचे, आता तरुणांबरोबरच आशुतोषच्या आईच्या वयाच्या स्त्रियाही त्याचं ' दर्शन ' घेऊ इच्छिताहेत, पायावर डोकं ठेवू इच्छिताहेत...
