लेख: जाता जाता
फॉरीनर्स
[ Tuesday, February 22, 2005  08:07:38 pm]


' भारतात इतके परदेशी लोक का येतात ?' चकोरीने म. टा.मधील आकड्यांचा खेळ वाचून विचारलं. काहीच संदर्भ नसल्यामुळे तिघांनी एकमेकाकडे बघितलं. चकोरने चटकन पेपर नजरेखालून घातला.

' दरवषीर् आकडा वाढतोच आहे. गुड साईन! ' मुलाने मत दिलं.

' फील गुड! ' मुलीने पुष्टी जोडली.

' संबंध काय या सगळ्याचा ?' चकोरीने विचारलं.

' जितके परदेशी जास्त तितकं फॉरीन एक्स्चेंज अधिक. ' मुलाने अर्थशास्त्र समजावून दिलं.

' पण आपला देश गरीब, रस्त्यावर स्नेक चार्मर्स, वाघ फिरतात अशी इमेज आहे ना जगभर! ' चकोरीने विचारलं.

' गॉन आर दोज डेज! आपण आता सुपर पॉवर बनणार आहोत. ' मुलाच्या बोलण्यात अभिमान होता.

' बना, बना! पण माझ्या प्रश्ानचं उत्तर द्या आधी. ' चकोरीने हट्ट सोडला नाही.

' अगं सोप्पं आहे. फिरायला येतात, बिझीनेससाठी येतात, काहीजण अभ्यासासाठी येतात. ' चकोरने तिला समजुतीच्या स्वरात सांगितलं.

' काहीजण पंतप्रधान बनायला येतात! ' मुलीने आईच्या सॉफ्ट कॉर्नरवर हल्ला केला.

' काहीजण निव्वळ समाजसेवेसाठी येतात हे विसरता तुम्ही. ' तिने दिलेला संदर्भ तिघांना कळला.

' तू चेष्टा कर, पण असं कोणी ठरवून पंतप्रधान बनत नाही. व्होटर्सचा विश्वास मिळवावा लागतो. ' ज्यु. चकोर आणि ज्यु. चकोरी यांच्यातील वादाची नांदी म्हटली गेली.

' पण कँपेन तर लाँच करता येते. आणि व्होटर्स कशाला भुलतील हे सांगता येत नाही. ' मुलीने वाद सुरू ठेवला.

' पण पीएम न बनताही तुम्ही क्रिटीसाईज करता हे चुकीचं आहे. ' त्याने उत्तर दिलं.

' क्लिअर मेजॉरिटी मिळाली की क्लेम करायचा. सिंपल! ' मुलीचं आर्ग्युमेंट!

' पण शी इज नो मोअर फॉरिनर. कोर्ट आणि व्होटर्स या दोन्ही फोरमवर निकाल लागला आहे. ' चकोरने चचेर्त उडी घेतली.

' वन्स फॉरिनर, ऑलवेज फॉरिनर! ' मुलगी आपल्या भूमिकेशी ठाम होती.

' हा दुटप्पीपणा झाला. कोणी इंडियन युएस, युकेमध्ये निवडून आला की त्याचं कौतुक, पण इतरांना मात्र वेगळा न्याय. ' चकोरीने आपल्या लाडक्या व्यक्तिमत्त्वाची बाजू मांडली.

' या डिस्कशनमध्ये मूळ मुद्दा बाजूला पडला. फॉरीनर्स भारतात का येतात ?' चकोरने गाडी ट्रॅकवर आणली.

' एखादा अपवाद वगळता सगळेजण बिझीनेस किंवा फिरायला येतात. ' चकोरीने निष्कर्ष काढला आणि ती चहा करायला उठली.

चकोर
