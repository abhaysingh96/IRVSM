सदर
फर्मास 'गोडवा'
[ Sunday, February 13, 2005  06:18:47 pm]


नावात काय आहे, असं म्हणण्यापेक्षा नावातच सर्व काही असं म्हणण्यासाठी ' गोडवा ' एक उत्तम उदाहरण आहे. नाव ' गोडवा ' असलं तरी इथे झणझणीत पदार्थही मिळतात. इथल्या सर्वच पदार्थांना स्वतंत्र चव आहे. या ठिकाणी पदार्थ खात असता घरीच बनवलेल्या पदार्थांचा आस्वाद घेत आहोत की काय, असंच जणू वाटतं. मराठमोळ्या पदार्थांपासून उडप्याच्या हॉटेलात मिळणारे बरेच पदार्थ इथे मिळतात.

' गोडवा ' चं मेन्यूकार्डच इतकं आकर्षक आहे की त्यामधला प्रत्येक पदार्थ खावा असंच वाटतं! ' भाजीच्या मळ्यात ' या विभागात पनीर टिक्का मसालापासून फ्लॉवर-ओला वाटाणा भाजीपर्यंत सर्व प्रकार मिळतात. ' दाल राइस ' मध्ये साधा भात, मसाले भात, ग्रीनपीज पुलाव, वरणभात, आंबट वरणभातच नव्हे तर गुजराथी, पंजाबी आणि सोलकढीही मिळते. या ठिकाणी मिळणाऱ्या चटणीमध्येही भरपूर व्हरायटी आहे. शिवाय ' पुरी-पराठा ' मध्ये पालक पुरी, मेथी पुरी, मसाला पुरी, रुमाली रोटी, आलू पराठा आणि मालवणी वडेदेखील आहेत. ' रायता ' मध्ये दही-कोशिंबीर, व्हेज. रायताबरोबर सॅलडचेही भरपूर प्रकारही आहेत. चायनीज पदार्थांसाठी असलेला ' चायना टाऊन ' हा ऑप्शन लाजवाबच आहे. अर्थात, यात विविध प्रकारचे सूप आहेत, हे वेगळं सांगायला नकोच.

' गोडवा ' चे मालक राजेश शिंदे सांगतात, की वेगळ्या व्यवसायात शिरण्यापेक्षा माहितीच्याच व्यवसायात विविधता आणून, इथल्या खवय्यांची भूक भागवायची असं मनाशी ठरवूनच ' गोडवा ' सुरू केलं. ' गोडवा ' चा आवाका लहान असला तरी एकदम योग्य ठिकाणी आल्याचं समाधान नक्कीच तुम्हाला इथे आल्यावर मिळेल. नेहमीच्या पदार्थांबरोबरच सणावारला घरी बनणारे पदार्थच नव्हे तर वेगवेगळ्या भाज्याही आमच्याकडे मिळतील. उदाहरणार्थ संक्रांतीला बनवली जाणारी ' भोगीची भाजी ', ' फणसाची भाजी ', ' केळफुलाची काळा वाटाणा घातलेली भाजी तसंच होळीला बनणारी ' कटाची आमटी ' आमच्याकडे तुम्हाला मिळेल. पुरणपोळ्या आमच्याकडे वर्षाचे बाराही महिने मिळतात. इथे रोजची भाजीही वेगवेगळी असते. त्यात बटाटा रस्सा भाजीपासून फ्लॉवर, वांगी, बटाट्याच्या काचऱ्याही मिळतात.

' गोडवा ' मध्ये मिळणारा ' आलू पराठा ' अजिबातच ' हेवी ' वगैरे नसतो. यासाठी आलं, लसूण, मिरची, कढीपत्ता यांची पेस्ट करून तिची फोडणी बटाट्याला दिली जाते. कणकेच्या गोळ्यात पुरण(बटाट्याची भाजी) भरून त्याची चक्क पुरणपोळीप्रमाणेच पोळी लाटली जाते. नंतर तयार होतो खमंग आलू पराठा. तसंच इथला मकाई टोस्टही उत्तमच आहे. याची खासियत म्हणजे हा दिसायला जरी नावडत्या रंगाचा असला तरी त्याची चव मात्र आवडणारी आहे. स्लाइस बेडवर उकडलेला बटाटा आणि मक्याचे दाणे यांच्या मिश्रणात मीठ, मसाला घालून त्याची भाजी ठेवली जाते. त्याला वरून भाजलेले पांढरे तीळ लावले जातात. हा पदार्थ घट्ट होण्यासाठी साधारण अर्धा तास फ्रिजमध्ये ठेवून, नंतर तो डीप फ्राय केला जातो. अशा प्रकारचे वेगवेगळे आणि चवदार असे खमंग पदार्थ आहेत. यात आवर्जून नाव घ्यावं लागेल ते केळफुलाच्या भाजीचं आणि उकडीच्या मोदकांचं. संकष्टी आणि अंगारकीला तर यांच्याकडच्या मोदकांना विशेष मागणी असते.

गोडवाचं आणखी एक वैशिष्ट्य म्हणजे इथे बरेचसे पदार्थ हाफ प्लेटही मिळतात. जसे वरण-भात, उसळी, भाज्या वगैरे. राजेश शिंदे यांचा घरचाच केटरिंगचा जुना व्यवसाय असल्याने त्याचा फायदा या गोडवासाठी नक्कीच झाल्याचं ते सांगतात. त्यामुळे हा गोडवा नावापुरता असा निश्चितच नाहीये. बोरिवलीच्या एल.टी. रोडवरील बाभई नाक्यावरच डाव्या बाजूला वळलं की हा गोडवा तुमच्याही दृष्टीस पडेल. मग इथला आस्वाद लवकरच घेणार ना ? ऑर्डर देण्यासाठी गोडवाचा नंबर - 56825622/9821686058 आहे.
