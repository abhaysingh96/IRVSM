संवाद
हमारी अमृता!
[ Saturday, April 02, 2005  10:19:07 pm]


... पुण्यातल्या साळुंके विहारपासून सुरू झालेल्या तिच्या फॅशन जगतातील प्रवासाचा पुढचा टप्पा थेट बँकॉकमध्ये आकाराला येईल. तोवर ' नो शॉर्टकट्स टू सक्सेस ' या उक्तीप्रमाणे अविरत मेहनतीचा ध्यास तिनं घेतलाय. त्या मार्गावरील ' एव्हरेस्ट ' सर करण्यासाठी अमृता सज्ज आहे. जागतिक स्तरावर मोहोर उठवणाऱ्या पुणेकरांमध्ये अमृताच्या रुपानं भर पडली आहे!

पेठी संस्कृती आणि पेन्शनरांचं शहर म्हणून हिणवल्या गेलेल्या पुण्यानं आयटी क्षेत्रातील बंगलोर, हैदराबाद या शहरांबरोबरची स्पर्धा असो वा जगभरातल्या पर्यटकांना आकषिर्त करणारा गणेशोत्सव असो, गेल्या काही वर्षांमध्ये अनेक रंग दाखवले. आधुनिकतेचा नवा मंत्रही पुणेकरांनी आचरणात आणल्याचं चित्र दृष्टीस पडतंय. म्हणूनच, रविवारी पाँड्स फेमिना मिस इंडिया स्पधेर्त पुण्याची अमृता थापर विजेती ठरली, तेव्हा पुणेकर सोडून तमाम देशवासीयांच्या भुवया आश्चर्यानं उंचावल्या!

फॅशन क्षेत्रात ' शाइन ' होऊ इच्छिणाऱ्यांची कर्मभूमी खरंतर मुंबई. त्यामुळं पुण्याच्या उपनगरात राहणाऱ्या अमृताची फेमिना मिस इंडिया युनिव्हर्सचा किताब पटकावण्याची कामगिरी स्वप्नवतच! सहा वर्षांपूवीर्च्या शहरातल्या मर्यादित फॅशन जगतामध्ये ' मिस पुणे ' सहित अन्य दोन पुरस्कार पटकावून तिनं सौंदर्याप्रमाणेच स्वत:च्या बुद्धीचीही चुणूक दाखवली होती. वाडिया कॉलेजमधून कॉमर्सची पदवी घेणाऱ्या अमृतासमोर ' रोल मॉडेल ' म्हणून सर्वप्रथम तिच्या आईचंच नाव येतं. स्कूल ऑफ फॅशन टेक्नॉलॉजीमधून ' फॅशन डिझायनिंगचा ' कोर्स पूर्ण केल्यानंतर रविवारी मिळालेल्या सवोर्च्च यशापर्यंतच्या वाटचालीत आईचा पाठिंबाच मोलाचा होता, असं तिला वाटतं.

' मिस इंडिया ' चा मुकुट मिरवणारा तिचा चेहरा जगभरात पोचला असला, तरीही अत्युच्च शिखरापेक्षा, तेथे पोचण्यापर्यंतच्या प्रवासातील अनुभव तिला जीवन समृद्ध करणारे वाटतात. किताब मिळवल्यावर सर्व स्तरांतून प्रसिद्धीची वलयं तिच्याभोवती गुंफली जात असली, तरी ' असं होईलच ' याची तिलाही खात्री नव्हती. त्यामुळेच ' क्राऊन ' पर्यंत पोचता आलं नाही तर.... स्पधेर्तील अनुभवाची शिदोरी बरोबर घ्यायची आणि आपल्या वाटेनं चालत राहायचं, असं तिनं ठरवलं होतं. वडिलांच्या लष्करी पार्श्वभूमीमुळे बालपणापासून गिरवलेले शिस्तीचे धडे तिच्या व्यक्तिमत्त्वात रुजले होतेच!

गल्लीपासून दिल्लीपर्यंत अनेक सौंदर्य स्पर्धा होतात. त्यात विजेतेपदाची माळ गळ्यात पडणाऱ्या प्रत्येक स्पर्धकाचं शारीरिक सौंदर्यच बघितलं जातं असं नाही. विशेषत: फेमिना मिस इंडिया सारख्या स्पर्धांमध्ये सुंदरतेची तपासणी इतर निकषांवरही केली जाते. त्यामुळेच तल्लख उत्तरं देणाऱ्या अमृताचं वर्णन ' शेड डिफरंट फ्रॉम अदर्स ' असंच करायला हवं. स्पधेर्त परीक्षकांनी विचारलेल्या प्रश्ानंना तिनं अतिशय सडेतोड, मात्र मुद्देेसूद उत्तरं देऊन ' ब्यूटी विथ बेन ' चा नमुनाच पेश केला. ( सौंदर्य स्पर्धा म्हणजे केवळ ' ग्लॅमर ' आणि दिखाऊपणा, असा समज करून त्याकडे नेहमी बोटं दाखवणाऱ्यांवर अमृताच्या उत्तरांमुळे बोटं लपवण्याची वेळ आली असेल!) आयुष्यातील एखाद्या गोष्टीकडे पाहण्याचा तुमचा दृष्टिकोन आशावादी असेल, तर ' रुटीन लाइफ ' उत्साहवर्धक होण्यासाठी तो पुरेसा ठरतो, हे तिचं उत्तर कोणत्याही क्षेत्रात चपखल बसेल, असंच आहे.

चित्रपट क्षेत्रात अलगद पाऊल ठेवता यावं याकरता सध्या फॅशन आणि मॉडेलिंग क्षेत्रांच्या पायऱ्यांचा सर्रास वापर होतोय. सौंदर्य स्पर्धांमधून सुश्मिता सेन, ऐश्वर्या राय आणि प्रियांका चोप्रा यांच्यानंतर बॉलिवूडमध्ये ठसा उमटवण्यासाठी प्रयत्न करणार का, चित्रपटांच्या येणाऱ्या ऑफर्सविषयी काय निर्णय घेणार... अशा प्रकारच्या अनेक प्रश्ानंची सरबत्ती ' क्राऊन ' पटकावल्यावर तिच्यावर होणं क्रमप्राप्त होतं. चित्रपटांच्या रुपेरी पडद्यापासून मात्र सध्या तिनं चार हात लांबच राहायचं ठरवलंय. कारण, चित्रपटांमधून प्रसिद्धीचे झोत मिळवण्यापेक्षा तिला ' इंटरनॅशनल ' स्तरावर देशाचं प्रतिनिधित्व करण्यात जास्त रस आहे. ' मिस युनिव्हर्स ' चा बहुमान पुनश्च एकदा भारतात आणण्यासाठी ती उत्सुक आहे. त्यामुळे ' लाइट्स, कॅमेरा, अॅक्शन ' पेक्षा शारीरिक तंदुरुस्ती आणि मानसिक आरोग्य टिकवून जागतिक स्पधेर्त उतरण्याचं तिचं मुख्य ध्येय आहे.

देशातील ' सौंदर्यसम्राज्ञी ' चा शोध घेणाऱ्या स्पधेर्त अंतिम यशापर्यंत पोचूनही ' अमृता थापर ' म्हणून तिच्यात कोणताच फरक पडलेला नाही. मात्र, जागतिक स्तरावरील स्पधेर्त सहभागी होण्याची संधी मिळाल्यानं जबाबदारी वाढल्याची जाणीव झाल्याचं ती मान्य करते. त्यामुळे बँकॉक येथे जूनमध्ये होणाऱ्या स्पधेर्साठी फारच कमी अवधी राहिला असल्यानं सध्या तिनं सर्व लक्ष त्यावरचं केंदित केलयं. सुश्मिता सेननं एका तपापूवीर् ' मिस युनिव्हर्स ' चा किताब पटकावून देशाच्या लौकिकात भर टाकली होती. हा लौकिक पुन्हा मिळवण्याची कर्तबगारी तिला करून दाखवायची आहे.

पंकज उदासच्या ' जानेमन ' पासून सध्या तरुणांमध्ये सर्वाधिक लोकप्रिय ठरलेल्या सुखबिंदरच्या ' बल्ले बल्ले ' अशा आठ म्युझिक अल्बममध्ये काम करणाऱ्या अमृताचं ' र्फस्ट लव्ह ' मात्र फॅशन डिझायनिंगचं! त्यामुळेच ' क्राऊन ' पर्यंतची मजल मारण्यात त्या क्षेत्रातला कोर्स अमृताला वाटसरूसारखा उपयोगी पडल्याचं स्पष्टच आहे. या सौंदर्यसम्राज्ञीला फॅशनव्यतिरिक्त इतर क्षेत्रांचं वावडं नाही. ड्रेस डिझायनिंग आणि स्केचिंग हे तिच्या फावल्या वेळातले छंद. तर स्विमिंग आणि ट्रेकिंगचीही तिला आवड आहे.

मार्चचा अखेरचा रविवार अमृता थापर या तेवीस वषीर्य तरुणीच्या आयुष्याला कलाटणी देणारा ठरला. पुण्यातल्या साळुंके विहारपासून सुरू झालेल्या तिच्या फॅशन जगतातील प्रवासाचा पुढचा टप्पा थेट बँकॉकमध्ये साकारेल. तोवर ' नो शॉर्टकट्स टू सक्सेस ' या उक्तीप्रमाणे अविरत मेहनतीचा ध्यास तिनं घेतलाय. ' मिस युनिव्हर्स ' चा किताब पटकावणं अवघड असलं, तरी असंभव नक्कीच नाही. त्या मार्गावरील ' एव्हरेस्ट ' सर करण्यासाठी अमृता सज्ज आहे. जागतिक स्तरावर मोहोर उठवणाऱ्या पुणेकरांमध्ये अमृताच्या रुपानं भर पडली असून ' अमृताहूनी गोड.. ' चा शोध सध्या थबकलाय!
