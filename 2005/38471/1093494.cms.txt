अर्थव्यवहार
टीव्ही उत्पादकांचा ग्राहकाभ्यास!
[ Friday, April 29, 2005  08:43:20 pm]


एक काळ होता, फ्रिज म्हटला म्हणजे गोदरेज, टीव्ही म्हणजे सोनी, ओनिडा हे ठरलेलं असायचं. कहाँ गये वो लोग ? ते ' लोग ' इथेच आहेत. सध्या चलती आहे ती मात्र ' एलजी ' च्या लोगोची! फ्रिज 27 टक्के, रंगीत टीव्ही 25 टक्के, धुलाई यंत्रे 34 टक्के, मायक्रोवेव ओव्हन 41 टक्के. एकूण खपात ' एलजी ' चा सध्या इतका हिस्सा आहे. 1990 मध्ये ' लकी गोल्ड स्टार ' ही भारतात अपयशी ठरलेली कंपनी, भारतीय भागीदारीची अट गेल्यावर ' एलजी ' या नावाने एकदम जोमाने वाटचाल करू लागली. कानामागून आली आणि सुरुवातीच्या 150 कोटी रु.च्या उलाढालीवरून 2004 पर्यंत तिने 6500 कोटींपर्यंत मजल मारली.

जाहिरातीबरोबरच ' ग्राहकाभ्यास ' हेही कंपनीच्या यशाचं एक गमक आहे म्हणे. उदाहरणार्थ, केरळकडे लोकांना ' टीव्ही ' चा छप्परफाड आवाज लागतो म्हणून ' एलजी ' आपल्या ' टीव्ही ' त त्या क्षमतेचा स्पीकर देते. पंजाबी मसाल्यांचा रंगावर परिणाम होतो, म्हणून तिथे भडक रंगाचे रेफिजरेटर्स ठेवले जातात वगैरे. शिवाय जिथे ' सैतान ' पोहोचत नाही (आठवा ओनिडाची जाहिरात!) तिथे ' एलजी ' पोहोचते. उत्तर प्रदेशातील एका छोट्या शहरात ' एलजी ' ने आपला एक लाख रु. किमतीचा टीव्ही चक्क पंधरा दिवसांच्या प्रयोगावर वापरायला दिला. आता, एकदा घरात घुसलेल्या टीव्हीला बाहेर काढणं कुणाच्या (म्हणजे घरातल्या पोरांच्या) बापाला तरी शक्य आहे का ? परिणामी त्या छोट्या शहरात मोठ्या टीव्हींची साथ आली. कंपनीने त्यासाठी परवडेल अशी हप्तेबंदी करून दिली. आज ' एलजी ' चा 65 टक्के खप मोठ्या शहरांबाहेर आहे. रुद्बद्घद्गज्ह्य त्नश्ाश्ास्त्र असं उगाच नाही म्हणत!

आणखी एक - वाणीला रावजी, बहेनजीचं वळण लागलेल्या या देशात जनता ' एलजी ' लाही सहजपणे ' हाँजी ' म्हणून जात असल्याची शक्यताही नाकारता येत नाही!

सुधीर सुखठणकर
