महामुंबई
छे छे, निव्वळ बाता!
[ Wednesday, April 20, 2005  12:07:18 am]


म. टा. प्रतिनिधी

मुंबई : ' जगातील सर्वात सुंदर स्त्री ' म्हणून गौरवलेली ऐश्वर्या राय भारताचे नाव हॉलिवुडमध्ये रोशन करणार अशी खात्री करून देणाऱ्या बातम्यांमधील बनेलपणा आणि खोटेपणा उघड करून ' तहलका ' या वृत्तपत्राने खळबळ उडवून दिली आहे. ऐश्वर्याची तथाकथित आंतरराष्ट्रीय यशोगाथा म्हणजे जनसंपर्क मोहिमेमार्फत मारल्या गेलेल्या निखालस खोट्या बाता आहेत, असे ' तहलका ' च्या ताज्या अंकातील कव्हर स्टोरीत स्पष्ट झाले आहे.

ऐश्वर्याने आतापर्यंत ' टाइम ' च्या मुखपृष्ठावर झळकणे , ' सीबीएस ' चा ' सिक्स्टी मिनट्स ', ' गुडमॉनिर्ंग अमेरिका ', ' डेव्हिड लेटरमन शो ' अशा गाजलेल्या टीव्ही कार्यक्रमांमध्ये तिला पाचारण केले जाणे, कान महोत्सवाला परीक्षक म्हणून हजेरी लावणे, मादाम तुसॉच्या मेण-संग्रहालयात पुतळारूपाने अजरामर होणे, असे अनेक खरेखुरे पराक्रम गाजवले.

मात्र, त्या बळावर ती हॉलिवुड पादाक्रांत करून ज्युलिया रॉबर्ट्सची जागा घ्यायला निघाल्याच्या प्रसिद्धीचा जो धडाका सुरू आहे, तो अतिशय चतुराईने रचलेल्या प्रसिद्धीमोहिमेचा भाग असून त्यात एखादा टक्काही तथ्यांश नाही, असे ' तहलका ' ने उघडकीस आणले आहे.

ऐश्वर्याच्या या ' इंटरनॅशनल फालमफोकी ' ची सुरुवात झाली 2003 साली. जेम्स बाँडच्या आगामी सिनेमात बाँडगर्ल म्हणून ऐश्वर्याचा विचार सुरू असल्याची ' माहिती ' ऐश्वर्याच्या सेक्रेटरीने एका वृत्तसंस्थेला दिली होती. तिची अर्थातच मोठी बातमी झाली. ऐश्वर्याच्या पदरात ' हॅलो ' या ब्रिटिश प्रकाशनात मुलाखतीचा मानही पडला आणि ' असे काहीही घडलेले नाही, जिथे अजून पुढच्या बाँडपटाची कथासुद्धा ठरलेली नाही, तिथे बाँडगर्ल कुठून पक्की होणार ' हा बाँड सिंडिकेटचा खुलासा त्या गदारोळात विरूनही गेला. मेरिल स्ट्रीपबरोबरचा ' केऑस ' आणि रोलँड जॉफीच्या ' सिंग्युलॅरिटी ' चीही तीच गत आहे. सिनेमावाले म्हणतात, अजून काही ठरलेले नाही. ऐश्वर्याचा कंपू सांगतो, अॅशची निवड झालीच.

गुरिंदर चढ्ढाच्या ' ब्राइड अँड प्रेज्युडिस ' च्या अमेरिकेतील यशाची कथा ऐश्वर्यानेच एका मुलाखतीत सांगितली होती. या सिनेमाने अडीच कोटी डॉलरचा धंदा केल्याचे तिने सांगितले, तेव्हा प्रत्यक्षात या सिनेमाची कमाई होती 60 लाख डॉलर (सिनेमानिमिर्तीचा खर्च 70 लाख डॉलर होता.) आणि हा सिनेमा अमेरिकेत 3272 व्या क्रमांकावर होता!

ऐश्वर्या ऑस्कर द्यायला जाणार हा थापांमधला ' मास्टरपीस ' होता. ऑस्करच्या सर्व प्रेझेन्टर मंडळींची नावे तीन महिने आधीच ठरतात. त्यात कधीही ऐश्वर्याचा नामोल्लेखही झाला नसताना गुरिंदर चढ्ढाच्या कँपमधून ही बातमी पेरण्यात आली होती, तीही ' ब्राइड... ' ची बॉक्स ऑफिसवरची पिटाई रोखण्यासाठी.

आता ऐश्वर्याच्या हाती (खरोखरच) आहेत ते, सी ग्रेड इंग्रजी कामुकपट बनवणाऱ्या जगमोहन मुंध्रांचा ' प्रव्होक्ड ', हॉलिवुडमध्ये एका रेडिओ कार्यक्रमाचा अनुभव असलेल्या झकारी कॉफिनचा ' विंडफॉल, द भोपाल स्टोरी ', मायकल डग्लस आणि सहारा वन यांची निमिर्ती असलेला, लोणावळ्याच्या अॅम्बी व्हॅलीत चित्रित होणार असलेला ' रेसिंग द मॉन्सून ', आयमॅक्स फॉरमॅटमधला ' ताजमहल ' आणि गुरिंदर चढ्ढाचा नवरा पॉल बजेर्स याचा दिग्दर्शनक्षेत्रातले पदार्पण साजरे करणारा ' द मिस्ट्रेस ऑफ स्पायसेस '! यातला एकही सिनेमा ' हॉलिवुडपट ' या बिरुदाच्या जवळपासही जाणारा नाही!
