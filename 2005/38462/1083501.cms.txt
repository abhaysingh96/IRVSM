लेख: जाता जाता
सुदर्शनचक्र!
[ Wednesday, April 20, 2005  07:54:22 pm]


सायंशाखा भरली आहे. चार ज्येष्ठ व वयोवृद्ध स्वयंसेवक रिंगण करून बसले आहेत. त्यांच्या बोलण्यात गुरुजी, बाळासाहेब, रज्जुभय्या अशी नावं वारंवार येत आहेत. मध्येच देवल, अप्पा, ज्ञानप्रबोधिनी असेही संदर्भ ओझरते निघतात.

' काय चाललं आहे काही समजत नाही. ' एकाचं मनोगत.

' कशाबद्दल ?' दुसऱ्याच्या विचारण्यात सावधपणा.

' अटलजी आणि लालजींचं! ' पहिल्याचा खुलासा.

' शाखेवर हा विषय मला वाटतं घेऊ नये. ' तिसऱ्याचं मत.

' का नको ?' पहिल्याचा सवाल.

' राजकारण आपल्याला र्वज्य आहे. ' चौथ्याचं स्पष्टीकरण.

' आपल्याला म्हणजे कोणाला! ' उलटा प्रश्ान्.

' संघाला! ' उत्तरात त्रोटकपणा.

' पण आपले स्वयंसेवक राजकारणात जातात. सवोर्च्च पदं मिळवतात. आणि विजयादशमी संचलनात संपूर्ण गणवेशात येऊन आपल्याला उपदेश करतात. ते चालतं वाटतं! ' बोलणाऱ्याच्या स्वरात कडवटपणा.

' तसं करणं म्हणजे सत्ता मिळाली तरी आपल्या अजेंडापासून दूर गेलेलो नाही असा मेसेज ते देतात. ' एकजण भूमिका मांडतो.

' पण आपला अजेंडा प्रत्यक्षात कधी येणार! ' विचारणाऱ्याच्या स्वरात वैफल्य.

' पूर्ण सत्ता आल्याशिवाय शक्य नाही असं गेल्या वषीर्च्या संचलनात स्पष्ट केलंय ना! ' एकजण आठवण करून देतो.

' ती कधी येणार! ' विचारणारा सुस्कारा टाक तो.

' नजिकच्या काळात तरी ते शक्य दिसत नाही. ' एकाची भविष्यवाणी.

' इतकं निराश होण्याची गरज नाही. तरुण पिढी हे स्वप्न साकार करेल. ' एकाचा आशावाद.

' म्हणजे कोण! प्रमोद वगैरे! ' विचारणारा पूर्ण निराश झालेला.

' पण सध्या अचानक सगळेजण का सुदर्शनचक्रं सोडायला लागले आहेत ?' एकाची विषय बदलण्यासाठी विचारणा.

' बहुधा सुदर्शनचक्रांचा कारखाना निघाला असावा! ' या वाक्यावर बाकीचे तिघे हसावं की हसू नये या विचारात पडतात.

' पण भगवान कृष्णानं शिशुपालाला मारून सुदर्शनचक्र परत घेतलं होतं. आता सुटलं की मीडियावाले ते ताब्यात घेतात आणि गरागरा फिरवत बसतात. त्यात किती जणांचा शिरच्छेद होतो हे आपण बघतोच आहोत. ' एकजण सगळ्यांना वस्तुस्थितीची जाणीव करून देतो.

' राजकारणात कधी ' सु ' दर्शन घडतं का ?' एक थेट प्रश्ान्.

' हल्ली ते कोठेच घडत नाही. ' बोलणारा पूर्ण निराश झालेला.

' सीईटी आणि व्हेकेशन बॅचेसमुळे युवापिढी शाखेवर येईल असं वाटत नाही. चला आपण प्रार्थना म्हणू या. ' सगळेजण उठतात.

चकोर
