महामुंबई
मेनी हॅपी रिर्टन्स ऑफ द डे!
[ Friday, April 01, 2005  11:55:15 pm]


मामंजी

मुंबई : एक फेब्रुवारीलाच एक एसएमएस येतो... ' हा मेसेज आजपासून बरोब्बर 59 दिवसांनी वाच... ' उत्सुकता चाळवलेला माणूस अर्थातच त्यात पुढे काय आहे, ते पाहू लागतो... दोन-तीन कोऱ्या ओळींनंतर पुढची ओळ उमटते... ' मला ठाऊक होतंच की तुझी उत्सुकता तुला स्वस्थ बसू देणार नाही... ठीक आहे... तू ' एप्रिल फूल ' च्या 59 दिवस आधीच ' फूल ' ठरला आहेस... हॅपी एप्रिल फूल्स डे!!! '...

' एप्रिल फूल ' बनवणाऱ्यांनी मोहीम उघडली होती ती इतकी आधीपासून... एकाने दोनचार दिवस आधीच लिहिले होते , ' एके दिवशी तुझी योग्यता सर्व जगाला कळेलच... सगळं जग तुझा उदोउदो करील, तुझ्या व्यक्तिमत्त्वाचा गौरव करील, तुझे विचार, तुझ्या कल्पना, तुझे अस्तित्त्वच त्या दिवशी साजरे होईल.... तो दिवस ओळखला जाईल... ' एप्रिल फूल्स डे ' या नावाने!!!.... ' या धमाल मोहिमेची यथोचित ' सांगता ' शुक्रवारी एकमेकांना हसतखेळत मूर्ख बनवणाऱ्या एसएमएस संदेशांच्या जोरदार देवाणघेवाणीने झाली... या मूर्खदिनाच्या पूर्वसंध्येलाच चतुर एसएमएस सुरू झाले. एक असा होता , ' तुम्ही किती सुंदर आहात, किती सार्मथ्यवान आहात, किती स्मार्ट आहात, किती ग्रेट आहात, असे कोणी तुम्हाला 31 मार्चलाच सांगू लागले तर... तर ताबडतोब त्याचा गळा दाबा... एक दिवस आधीच तुम्हाला मूर्ख बनवतो म्हणजे काय ?!!!'

प्रत्यक्ष एप्रिल फूलच्या दिवशी एका एसएमएसमध्ये दोस्तीची फसवी दुहाई दिली होती , ' ऐसा दोस्ताना हमारा, मैं कश्ती तू किनारा, मैं धनुष तू तीर, मैं मटर तू पनीर, मैं हॉट तू कूल... मैं एप्रिल तू... फूल!!! ' ' एका एसएमएसमध्ये फक्त एकाखाली एक ' येडा ' हा एकच शब्द वेगवेगळ्या प्रकारांनी लिहिला होता, बटण दाबून खाली खाली खाली गेल्यावर शेवटची ओळ होती , ' काय को दबाता है, येडा है क्या ?'

एकाने नेहमीप्रमाणे खोटा मस्का चोपडत एसएमएस सुरू केला होता , ' या भूतलावरचा सर्वात देखणा चेहरा तुझाच आहे, तू अत्यंत श्ाेष्ठ व्यक्तिमत्त्व आहेस, जितीजागती दंतकथा आहेस, स्टार आहेस... छे छे! मी तुला मूर्ख बनवत नाहीये... हे असलं ' क्रॅप ' वाचून तूच स्वत:ला मूर्ख बनवतोयस!!! ' आणखी एकाने तर सरळसरळ शरणागतीच पत्करणारा एसएमएस सर्वांना- त्यांचे येण्याआधी- पाठवून टाकला , ' नका नका... मला मूर्ख बनवण्याचा विचारसुद्धा मनात आणू नका... फसाल... अहो, सार्वकालिक , ' सार्वमासिक ' मूर्खाला तुम्ही ' एप्रिल फूल ' बनवू शकत नाही!!! '

' एप्रिल फूल ' बनलेल्यांना ' दिलासा ' देणाऱ्यांचीही कमतरता नव्हती. एकाने लिहिलं , ' लोकांनी आज तुला ' फूल ' बनवलं असेल म्हणून उदास नको होऊस! मी सांगतो तुला, तू खूप कलर ' फूल ' आहेस, पॉवर ' फूल ' आहेस, ब्युटि ' फूल ' आहेस!!! '

... आता ही बातमी तुम्ही इथपर्यंत वाचलीच आहेत, तर ' एप्रिल फूल ' च्या सवोर्त्कृष्ट ' शुभेच्छा ' तुम्हालाही... ' मेनी हॅपी रिर्टन्स ऑफ द डे!!! '
