लेख: जाता जाता
नसलेल्या विजेसाठी...
[ Tuesday, March 15, 2005  08:52:47 pm]


मौजे बुदुकमध्ये रविवारची सकाळ. ग्रुप ग्रामपंचायतीच्या कार्यालयात गावकरी जमले आहेत.

' आपल्या बोर्डाने नवीन कोरी अँबेसेडर घेतलीय. ती आणणारेत सगळ्यांना दाखवायला. '

' अरं येड्या, गाडी नाय, नवीन ब्रँड अँबेसेडर येणारेय. कोणतरी पोरगाय मुंबईचा. '

' अस्सं, तो काय लायटी आणणारय वाटतं! '

' तसं नाय, पण आपल्याला तो बोर्डाच्या आयडिया सांगणारय, पारेेषण, वितरण हे पण समजावून सांगणारय. '

' त्यामुळे लाईट लागणार काय लगेच ?'

' लायटी कशां लागतील लगेच ? पण बलब का पेटत नाय!, पंप का चालत नाय! बिल जादा का येतं! या समद्या सवालांचा जबाब भेटंल बघ. '

' पण हे सांगायला पोरगा कशाला ? आपले इंजिनीयरसाहेब कितीतरी वेळी हेच सांगून गेले बघ. '

' आरं, पण साहेबांनी सांगणं वेगळं, पोरानं सांगितलं की जास्त गोड लागेल बघ. साहेबाचा खर्च आपल्या बोडक्यावर बसतो, पोराचा खर्च बोर्ड करल. आपल्याला त्रास नाय. '

' पण हे सांगायला पोरगा कशाला ? त्यापेक्षा ती वैशालीबाय आणि अवधूतबाबाला आवताण देयाचं. लय बेश्ट जोडी हाय बघ. एकदम फेमस. '

' पण ती आता जुनी झाली. ह्यो पोरगा तेंचेपेक्षा बेश्ट गातो म्हणे. '

' तो गाणी म्हणणारेय की बोलणारेय ?'

' तुझी काय इच्छा हाय ?'

' गायला तर टाईमपास होईल. रविवारच्या सकाळी भाषण कोण ऐकणार त्याचं! '

' आरं, सॉलीड पॉप्युलरय बघ ह्यो पोरगा. '

इतक्यात मंडळाची जीप धुरळा उडवत येते. त्यातून अभियंते आणि मंडळाचा अँबेसेडर उतरतो.

मंडळाचे अभियंते बोलायला सुरूवात करतात.

कोपऱ्यातून आवाज येतो ' गाणंं ऐकवा, गाणं ऐकवा '.

' ते गाणं ऐकवायला आलेले नाहीत. ते आता तुम्हाला मंडळाच्या योजनांची माहिती करून देणार आहेत. '

यावर सगळीकडे गडबड उडते. गोंधळात कोणाचंच म्हणणं कोणाला कळत नाही.

थोड्या वेळाने शांतता पसरते. अँबेसेडर बोलायला उभा राहतो.

' आज आपलं राज्य विजेच्या भीषण संकटाला तोंड देतं आहे. अशा वेळी सगळ्यांनी वीज जपून वापरायला हवी. '

कोपऱ्यातून पुन्हा आवाज येतो ' गाणं, गाणं.... '

' मी गाणं गाऊन विजेचा प्रश्ान् सुटणार नाही. '

' पण आमचा टाईमपास व्हईल ना... ' कोपऱ्यातून आवाज येतो.

' ठीक आहे, मी गातो. '

' वीज आली हो अंगणी, लक्ष्मी आली हो दारी... '

' हे नकोय, रीमिक्स म्हणा. '

ब्रँड अँबेसेडर मटकन खाली बसतो.

चकोर
