महाराष्ट्र
नाशिक केंदातून 'सत्यमेव जयते' अव्वल
[ Tuesday, March 15, 2005  09:20:48 pm]


म. टा. प्रतिनिधी

नाशिक : राज्य सरकारच्या सांस्कृतिक कला संचालनालयातर्फे नाशिक केंदावर घेण्यात आलेल्या राज्य बालनाट्य महोत्सवात नाशिक येथील ज्ञानवधिर्नी विद्याप्रसारक मंडळाच्या डे केअर सेंटरचे ' सत्यमेव जयते ' अव्वल ठरले तर सिन्नर येथील चांडक कन्या विद्यालयाच्या ' अग्निपंख ' ला दुसरे पारितोषिक जाहीर करण्यात आले आहे.

येथील कालिदास कलामंदिरात 14 फेब्रुवारी ते 4 मार्चदरम्यान या महोत्सवात जिल्ह्यातून 39 संस्थांनी बालनाट्ये सादर केली. त्यात प्रथम ठरलेल्या ' सत्यमेव जयते ' ला तीन हजार रुपयांचे तर द्वितीय ठरलेल्या ' अगिनिपंख ' ला दोन हजार रुपयांचे पारितोषिक देण्यात येणार आहे.

' सत्यमेव जयते ' या नाटकासाठी सुरेश भुसारे यांना दिग्दर्शनाचे प्रथम पारितोषिक तर द्वितीय पारितोषिक ' अग्निपंख ' चे दिग्दर्शक नितीन ठाकरे यांना मिळाले.

पंचवटीतील दीपक सांस्कृतिक विकास मंडळाच्या ' पल्लवीचा प्रवास ' या नाटकातील प्रकाश योजनेसाठी हेमंत देशपांडे यांना प्रथम पारितोषिक तर राजहंस माळी ( अग्निपंख) यांना प्रकाश योजनेचे द्वितीय पारितोषिक जाहीर झाले. नाशिकरोड या पुरुषोत्तम इंग्लिश स्कूलने सादर केलेल्या ' भटक्याचे भविष्य ' या नाटकासाठी स्वप्निल कट्यारे आणि समीर कुलकणीर् यांना उत्कृष्ट नेपथ्याचा प्रथम तर नाशिकच्याच युवा विहार सांस्कृतिक कला व क्रीडा मंडळाच्या ' प्लॅटफॉर्म ' या नाटकासाठी सुयोग रत्नपारखी यांना द्वितीय क्रमांक देण्यात आला.

पुरुष अभिनयातले प्रथम पारितोषिक ' सत्यमेव जयते ' त सतीशची भूमिका करणाऱ्या मुकुल बारुळकर याने तर ' अग्निपंख ' मध्ये प्रज्ञाची भूमिका करणाऱ्या नुपुर कंकरेजा हिने स्त्री अभिनयातले प्रथम पारितोषिक पटकावले. अभिनयातील पारितोषिके मिळविणारी मुले : प्रसाद बाबलकर ( प्लॅटफॉर्म), आशुतोष गुंजाळ (येरे येरे पावसा, व्ही. जे. तांत्रिक हायस्कूल, नांदगाव), किरण त्र्यंबके (ब्रूस ली वर्सेस चालीर्, ब. ना. सारडा विद्यालय, सिन्नर), आशिष पाटील (...आणि भिंत मोडली, नवरचना विद्यालय, नाशिक), पार्थ चोकसी (मोरूची गोष्ट, रंगुबाई जुन्नरे इंग्लिश स्कूल, नाशिक)

अभिनयातील पारितोषिके मिळविणाऱ्या मुली : श्रेया पाटील ( अक्षरांच्या संगे, रतनबेन जयराम चौहान बिटको र्गल्स हायस्कूल, नाशिक रोड), भारती अवसरकर (आई, महात्मा गांधी विद्यालय, इगतपुरी), शीतल लोंढे ( भटक्याचे भविष्य, संजीवनी प्राथमिक विद्यालय, सिन्नर), मयूरी मंडलीक ( सत्यमेव जयते), शिवानी भडांगे ( केल्याने होते आहे रे, नवीन मराठी शाळा, नाशिक रोड.)
