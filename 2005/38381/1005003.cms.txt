महाराष्ट्र
'न' नाट्याचा, 'स' संमेलनाच्या 'सावळ्या गोंधळा'चा!
[ Saturday, January 29, 2005  10:43:19 pm]


अमित भंडारी

डोंबिवली ( पु. भा. भावे नाट्यनगरी) : नांदीपूवीर् त्सुनामीग्रस्त आणि मांढरदेवी दुर्घटनेत बळी पडलेल्यांना श्रद्धांजली वाहून नाट्यसंमेलनाला सुरुवात झाली खरी ; पण ' सावळ्या गोंधळा ' चा सिलसिला अ. भा. मराठी नाट्यसंमेलनाच्या 85 व्या वर्षातही कायम राहिला. नाट्यसंमेलन ग्लॅमरस करण्याच्या नादात ' नाट्यसंमेलनाध्यक्ष ' मात्र उपेक्षितच राहिले.

हातात ' वॉकीटॉकी ' घेऊन पळापळ करणाऱ्या कार्यर्कत्यांच्या रूपात ' इव्हेंट मॅनेजमेंट ' करणाऱ्या मुलामुलींचा ताफाच अधिक लक्ष वेधून घेत होता ; पण ' इव्हेंट मॅनेजमेंट ' कंपनीच्या हातात सूत्रे देऊनही नाट्यसंमेलनाच्या ' सक्सेस रेट ' चं समीकरण पुरतं कोलमडल्याचं चित्र दिसत होते. रूपेरी आणि छोट्या पडद्यावर चमकणाऱ्या सिताऱ्यांना पाहण्यासाठी उफाळून आलेल्या गदीर्ला आटोक्यात आणण्यासाठी ' एनएसई ' या इव्हेंट मॅनेजमेंट कंपनीचे कार्यकतेर् झटत होते. काळ्या टीशर्टमध्ये झळकणाऱ्या या कार्यर्कत्यांच्या हाती आता नाट्यसंमेलनाची महत्त्वपूर्व सुरक्षेची जबाबदारी सुपूर्द केली होती, असे चित्र प्रथमच संमेलनात दिसत होते.

सकाळी दिंडीत सहभागी झालेल्या आणि कल्याण-डोंबिवली परिसरातील सांस्कृतिकतेचे दर्शन घडवणाऱ्या कलाकारांना मात्र मुख्य सभामंडपात प्रवेश दिला नाही, त्यामुळे नाट्यसंमेलनाच्या सुरुवातीच्या सत्रात काहीशी नाराजी पसरली होती. मुळात नाट्यसंमेलनासाठी तिकिट काढून यावं, अशी आयोजकांची भूमिका असल्यामुळे हा गोंधळ झाल्याचे उघड होते. आजपर्यंत झालेल्या नाट्यसंमेलनातलं तिकिट काढून पेक्षकांनी हजेरी लावलेलं हे पहिलंच संमेलन! गदीर् लक्षणीय असली तरी तिकिटांच्या दरात कालांतराने घट करण्यात आल्याची कुजबूज संमेलननगरीत कानांवर पडत राहिली.

सभामंडपाच्या एका बाजूस ' क्लोज सकिर्ट ' कॅमेऱ्याचे आयोजन करण्यात आले होते. मात्र ' बिग बी ' हजर राहूनही ' बिग स्क्रीन ' ची सोय नसल्यामुळे व्यासपीठावरील मान्यवर करंगळीच्या आकाराचे दिसत होते. त्यामुळे ' बिग बी ' असो वा शरद पवार यांच्या अंधुक हालचालींवर असंख्य पेक्षकांना समाधान मानावे लागले.

स्वागतसमारंभाच्या भाषणांचा कार्यक्रम काहीसा लांबला. ठीक साडेदहा वाजता अमिताभ बच्चन यांचं आगमन झालं. पावणे अकरा वाजता रंगमंचावर येऊनही नंतर राजकारण्यांची ' कलात्मक ' भाषणबाजी ऐकण्याचं भाग्य अमिताभ बच्चन यांच्या नशिबी असल्याचं त्यांच्या मनगटावरील घडाळ्याकडे पाहण्याच्या देहबोलीतून स्पष्ट होत होतं. हारतुऱ्यांनी भाषणबाजीच्या गदारोळात दस्तुरखुद्द बच्चन यांच्याकडे माईकवर बोलण्याची वेळ पावणेदोन तासांनी आली. आपल्या बिनतोड शैलीतील संवादेफेकीची मोहिनी टाकण्यापूवीर्च रसिकांनी टाळ्यांच्या गजरात अभिनयसम्राटाचं स्वागत केलं. विशेष म्हणजे त्यांचं आगमन झाल्याच्या केवळ घोषणेने उपस्थितांनी उभं राहून टाळ्या वाजवून मानवंदना दिली.

' ग्लॅमर ' चे वलय नाट्यसंमेलनाला लाभल्यामुळे खरंतर अधिक पेक्षक आकषिर्त झाल्याचे व्यासपीठावरून सूचकपणे सांगितले जात असताना मात्र ' सन्मानमूतीर् ' आणि नाट्यसंमेलनाच्या अध्यक्षांवर सगळ्यांच्या शेवटी भाषण करण्याची वेळ ओढवली खरी, पण भाषण सुरू असताना शहेनशहा अमिताभ बच्चन यांनी व्यासपीठावरून काढता पाय घेतला. त्यामुळे प्रेक्षकांनीही भाषणाकडे पाठ फिरवली. भाषण लांबल्याची चिन्हे दिसू लागताच नाट्यसंमेलनाध्यांकडे ' भाषण आवरते ' घेण्याबाबतच्या चिठ्ठ्यांचा सपाटाच सुरू झाला.

नाट्यसंमेलनाच्या आस्थेपोटी अमिताभने आपण येथे आल्याचे सांगितले खरे ; आणि पुढील वषीर्च्या निमंत्रणास पात्र आहोत असे वाटले आणि बोलावणं धाडलं तर आपण नक्की उपस्थित राहू, असं आश्वासन देणाऱ्या बच्चनसाहेबांना आपण उगाचच आश्वासन दिलंय, असं वाटू नये बसं इतकंच!
