संवाद
 गेले द्यायचे राहून ...
[ Saturday, January 29, 2005  06:16:14 pm]


' महासागर ' नाटकातली ' चंपू ' मला भेटली तेव्हा मी 22 वर्षांची होते. चंपू - चाळिशीची, तीन मुलींची आई, चाळीत राहणारी, एकत्र कुटुंबात राबणारी- मला पूर्णपणे अपरिचित! मी उच्चभ्रू मध्यमवगीर्य कुटुंबात वाढलेली, इंग्रजी शाळेत शिकलेली, मॉडर्न कपडे घालणारी कुमारिका. मी चंपू साकार करू शकेन, असं वाटतच नव्हतं ; पण तालमी करत गेलो तशी ही स्त्री मला थोडीथोडी कळत गेली. रंगमंचावर ' कायापालट ' कसा होऊ शकतो हे ' चंपू ' साकारताना अनुभवलं. माझी ' चंपू ' पोक काढून चालायची - जणू काही जगाचं ओझं आपल्या नाजूक खांद्यावर वाहतेय! ओंगळ नेसणं, करकचून बांधलेला अंबाडा, कावलेली तरीही शहाणी, प्रेमळ चंपू. चंपूमध्ये आणि, त्या वयातल्या माझ्यामध्ये असलेला निरागसपणा, एकरूप होऊन रंगमंचावर या पात्राला ' खरेपणा ' द्यायचा. ' महासागर ' चे लगभग 500 प्रयोग केले आणि जणावायला लागलं की , ' चंपू ' मध्ये मी अडकायला लागेल. रस्त्यावर चालताना जेव्हा नकळत माझे खांदे वाकायला लागले, तेव्हा हा धोका लक्षात आला - जड अंत:करणाने मी ' महासागर ' चा आणि ' चंपू ' चा निरोप घेतला. तरीही चंपू माझ्यात कुठेतरी कामयची वस्तीला आलीए हे नाकारता येणार नाही.

तब्बल एक तपानंतर ' ध्यानी-मनी ' मधली ' शालन पाठक ' भेटली. बाह्यरूपाने ' चंपू ' सारखी ; पण चंपूपेक्षा अगदी निराळी. शालन पाठक , - वय 45, तिचं जग तिचा चौदा वर्षांचा काल्पनिक मुलगा, मोहित, याच्याभोवती फिरणारं. अबोल नवऱ्यासोबत राहणारी, अतीव बडबडी शालन. हौशी, हसरी शालन. मानसिक संतुलन गमावलेली शालन. ही स्त्री मला तिच्या हसण्यातून मिळाली. पहिल्या अंकातली शालन वठवताना मला खास मौज यायची- माझी ही शालन जराशी श्ा 1 द्गह्मश्चद्यड्ड 4 द्गस्त्र होती. हावभाव, लकबी, एकटक बघत राहाणं, मध्येच हसणं - सगळच जरा जास्त करायचे मी. अंकाच्या शेवटी जेव्हा शालनचं काल्पनिक जग प्रेक्षकांसमोर उलगडलं जायचं, तेव्हा माझ्या या श्ा 1 द्गह्म - ड्डष्ह्लद्बठ्ठद्द चाही उलगडा व्हायचा! मग दुसऱ्या अंकातल्या शालनचा आणि प्रेक्षकांचा एक लाजवाब सुसंवाद साधला जायचा - विस्मयाचा, सहानुभूतीचा, संवेदनांचा!

' ध्यानी - मनी ' चेही जवळपास 500 प्रयोग झाले ; पण यावेळी मी ' शालन ' ला माझ्यापासून हवी तेवढी लांब ठेवण्यात यशस्वी झाले. नाहीतर ' शालन पाठक ' माझ्या मानगुटावर बसली असती! प्रेक्षकांना वाटायचा, तेवढा त्रास मला शालनच्या बाबतीत झाला नाही हे खरं! प्रयोगानंतर मात्र मी खूप दमायचे - श्चद्ध 4 ह्यद्बष्ड्डद्य ह्यह्लह्मड्डद्बठ्ठ मुळे - कारण हे पात्र साकारताना मी प्रचंड द्गठ्ठद्गह्मद्द 4 वापरायचे.

माझ्या दोन आवडत्या, सकारात्मक भूमिका महणजे श्वस्त्रह्वष्ड्डह्लद्बठ्ठद्द क्रद्बह्लड्ड मधली ' रीटा ' आणि ' सवत माझी लाडकी ' मधली ' सीमा '. ' रीटा ' ही माझी सर्वात उत्तम वठलेली आणि माझ्यावर सर्वाधिक प्रभाव पाडणारी भूमिका! 26 वर्षांची ब्रिटिश ' हेअरड्रेसर ' - असंस्कृत, अशिक्षित, तोंडाळ ' रीटा ' - ज्ञानप्राप्तीसाठी कॉलेजात नाव नोंदवते. तिच्यात होत जाणारं परिवर्तन आणि अखेरीस शिक्षणामुळे, निवड करण्याचं स्वातंत्र्य मिळालेली ' रीटा '. ' रीटा ' मला संहितेतच सापडली. तिचे हाव-भाव, बोलणं, सगळच मिळत गेलं - या काळात मी ' रीटा ' आणि ' रीटा ' मी झाले होते. कॉन्व्हेन्टमध्ये शिकण्याचा फायदा मला इथे झाला. ' चंपू ' सारखा ' कायापालट ' इथे नव्हता. इथे मी रीटा ' जगले '.

रीटाचा वावर मला भारी आवडायचा. जसा शालनचा एक वावर होता. सहज, सवयीचा वाटणारा - पण त्या ' सहजते ' साठी मी प्रचंड मेहनत घेतली होती. ' रीटा ' चा वावर सहज होता आणि तो मला सहज जमलाही. ' रीटा ' ने माझ्या मनात कायमचं घर केलय. आजही श्वस्त्रह्वष्ड्डह्लद्बठ्ठद्द क्रद्बह्लड्ड मधले संवाद मला मुखोद्गत आहेत.

' सवत माझी लाडकी ' मधली ' सीमा ' - रसिकांच्या पसंतीस उतरलेली लाडाची भूमिका. ' सीमा ' करता मी ' फिट्ट ' होते. ' सीमा ' चं वय, दिसणं, वागणं, वावरणं - सर्व काही माझंही होतं. कुठलाही आधार न होता ही भूमिका मी केली (ना लकब, ना आवाजात बदल, ना विगदेखील) - इतकी स्वाभाविक होती ' सीमा ' माझ्याकरता! ' सीमा ' ची सकारात्मक वृत्ती या भूमिकेचा प्लस पॉइंट होता. ' सवत माझी लाडकी ' आजही कालबाह्य झालेला नाही. काही कलाकृती सदाबहार असतात हेच खरं!

अलीकडे एक वेगळीच व्यक्तिरेखा मला भेटली - ' अधुरी एक कहाणी ' तली ' कल्याणी पटवर्धन ', प्रचंड ताकदीची, मोठ्या आवाक्याची कल्याणी एक आव्हान आहे माझ्यापुढे. वरवर ती कितीही ग्लॅमरस दिली, तरी माझ्यासाठी ' कल्याणी ', ' खरी ' साकारणं महत्त्वाचं आहे. खलनायकी, महत्त्वाकांक्षी, चाणाक्ष कल्याणी मला खूप द्बठ्ठह्लद्गह्मद्गह्यह्लद्बठ्ठद्द वाटते. ही बाई साकारण्यात एक गंमत आहे खास!

चंपू, शालन, रीटा, सीमा, कल्याणी व्यतिरिक्त अनेक उत्तम भूमिका वठविण्याचं भाग्य मला लाभलय , - ' आईचं घर उन्हाचं ' मधली ' क्षमा ', ' अकस्मात ' मधली ' अरुंधती ,' ' नागमंडळ ' ची ' राणी ', ' प्रेमपत्र ' मधली ' उमिर्ला '... यादी मोठी आहे! या प्रत्येक स्त्रीने मला काही दिलंय, जे आज माझं केवढं संचित बनलय. मी त्यांची उतराई आहे... माझ्यापाशी आता कळ्या आणि थोडी ओली पाने, गेले द्यायचे राहून , ' तुमचे ' नक्षत्रांचे देणे.
