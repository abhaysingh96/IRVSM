महामुंबई
शब्दांवर हुकूमत गाजवणारा नाटककार
[ Friday, January 21, 2005  01:21:19 am]


म. टा. सांस्कृतिक प्रतिनिधी

मुंबई : शब्दांवर हुकूमत गाजवणारा नाटककार म्हणून प्रदीप दळवी यांची ओळख मराठी रंगभूमीवर राहील. ' सहा रंगांचे इंदधनुष्य ', ' जबरदस्त ', ' माताजी ', ' वासूची सासू ', ' रिस्की टर्न ', ' आज धंदा बंद आहे ', ' झाले मोकळे आभाळ ', '23 जानेवारी ' ही नाटके त्यांनी लिहिली. तथापि ' वासूची सासू ' आणि ' मी नथुराम गोडसे बोलतोय! ' हीच त्यांची नाटके अत्यंत लोकप्रिय झाली.

रंगभूमी प्रयोग परिनिरीक्षण मंडळाने तब्बल 18 वषेर् परवानगी नाकारलेले ' मी नथुराम... ' अखेर 2001 साली रंगमंचावर आले. मात्र, दळवी यांनी या नाटकासंदर्भात दिलेल्या वादग्रस्त मुलाखतीमुळे नाटकाच्या प्रयोगावरच बंदी आली. पुढे या बंदीविरुद्ध दळवी निर्मात्याच्या खांद्याला खांदा लावून न्यायालयीन लढाई लढले. कोर्टाने ' नथुराम ' वरची बंदी उठवली, पण पुढे दळवी यांनी मूळ निर्मात्याने नाटक करण्यालाच आक्षेप घेऊन त्यावर बंदी आणण्याची मागणी केली. ' मी नथुराम... ' हे नाटक प्रचंड गाजले. यादरम्यान दळवी यांनी शिवसेनाप्रमुख बाळासाहेब ठाकरे यांच्या जीवनावर ' पसायदान ' नावाचे नाटक लिहून आणखी एक वाद ओढवून घेतला. हे नाटक तालमीतच बंद पडले.

दळवी यांनी 15 कादंबऱ्यांचे लेखन केले. या कादंबऱ्याही अतिशय लोकप्रिय झाल्या. ' रक्तरेखा ' ही त्यांची सर्वात लोकप्रिय ठरलेली कादंबरी. या कादंबऱ्यांचे प्रकाशक ' मनोरमा प्रकाशन ' चे अनिल फडके म्हणाले की , ' प्रदीप दळवींनी नथुराम गोडसे यांच्या जीवनावर लिहिलेली ' सागरा अगस्ती आला ' ही कादंबरी प्रकाशनाच्या वाटेवर आहे. ' मी नथुराम... ' या नाटकाच्या बंदीपासून नांदीपर्यंतचा इतिहास सांगणारं ' नथुरामचे ते दिवस ' हे त्यांचं पुस्तकही हायकोर्टाच्या निकालपत्रासह आम्ही प्रकाशित करणार आहोत. '

' मी नथुराम... ' चे दिग्दर्शक विनय आपटे म्हणाले , ' राजकीय सजगता असणारा, प्रगल्भ बुद्धिमत्तेचा पण कधी कधी विचित्र वागणारा नाटककार मित्र गेला. तो सातत्याने लिहीत राहिला असता तर त्याने व्यावसायिक रंगभूमीसाठी उत्तम नाटके दिली असती. '
