लेख: जाता जाता
एसी लोकलला विरोध
[ Tuesday, March 01, 2005  08:56:12 pm]


' बाबा, आता तुम्हाला अशा लोकलमधून प्रवास करायला मिळेल. ' ज्यु. चकोरने पित्याच्या पुढ्यात म. टा. सरकवत सांगितलं.

चकोरने सकाळीच पेपर नीट वाचला होता, पण छोटीशी चकाचक एसी लोकल त्याच्या नजरेतून सुटली होती.

' मी रिटायर होइपर्यंत तरी हे दिवस येतील असं वाटत नाही. ' चकोरने सहजतेने सांगून टाकलं.

' तोपर्यंत यांच्या गाड्या येतीलच. काय रे आम्हा म्हाताऱ्यांना न्याल ना तुमच्या बरोबर ?' चकोरीने हसत हसत विचारलं.

' सीनियर सिटिझन्सना प्रायॉरिटी! ' मुलीने पॉलिसी स्टेटमेंट केलं.

' असं आता म्हणते आहे. नंतर काय करतील देव जाणे! ' आई गंमतीने म्हणते आहे की गंभीरपणे हे ज्युनियर कंपनीला कळलं नाही.

' कशाला मुलांना चिडवतेस. तेव्हाचं तेव्हा बघू! ' चकोरने गाडी ट्रॅकवर आणण्याचा प्रयत्न केला.

' अशी लोकल खरोखर येईल काय हो ?' चकोरीने चकोरचा मूड ओळखला.

हे युरोपात वगैरे ठीक आहे. तिथे मुळात गदीर् कमी असते. आपल्यासारखी शेवटच्या लोकललाही दारात लटकण्याची पाळी त्यांच्यावर येत नाही. ' चकोरने वस्तुस्थिती मांडली.

' मुंबईत दर मिनिटाला गाडी सोडली तर हे शक्य होईल. ' मुलीने मत मांडलं.

चकोरने नकाराथीर् जोरात मान हलवली.

' डॅड, तुम्ही फार निगेटिव विचार करता. ' मुलाने तक्रार केली.

' मी तुमच्यापेक्षा जास्त विचार करतो. ' चकोरने त्रोटक उत्तर दिलं.

' का अशी गाडी शक्य नाही ?' चकोरीने विचारलं.

' मुंबईत जितक्या सोयी जास्त पुरवाल तितके लोंढे वाढत जातील. कारण अनेक राज्यांत इतकं भीषण चित्र आहे. ' चकोरने मांडलेल्या या मुद्यावर तिघेही विचारात पडले.

' मग आम्हाला अशा फॅ सिलीटीज मिळायलाच नकोत का ?' मुलीच्या विचारण्यात किंचित संताप होता.

' मिळायला हव्यात ; कारण या शहराची ती गरज आहे. पण इतर ठिकाणी निदान साध्या लोकल्स तरी चालवा. अगदी पुण्यातही लोकांना टू व्हीलर बाळगायला लागते. बिहार, यूपीमध्ये तर बघायलाच नको. ' चकोरने आथिर्क विचार मांडला.

' त्यांनी कष्ट करावेत आणि आपला प्रोग्रेस साधावा. ' मुलाने तक्रारीचा सूर लावला.

' मला वाटतं, अशा गाडीला बहुतेक मुंबईकर विरोध करतील. ' चकोरीने शांतपणे सांगितलं.

' का ?' तिघांनी कोरसमध्ये विचारलं.

' बहुतेकांना गाडीतून थंुकायची, बाहेर रिकाम्या प्लॅस्टिकच्या बाटल्या, पिशव्या, निर्माल्य भिरकवायची सवय जडली आहे. अशी लोकल या बेशिस्त मंडळींना कशी आवडेल ?' चकोरीच्या आर्ग्युमेंटवर तिघेही एकमेकाकडे बघायला लागले.

चकोर
