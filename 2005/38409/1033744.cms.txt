खास बात
बजेट 'चॅनलाइज्ड'
[ Saturday, February 26, 2005  08:12:19 pm]


देशावर शासन करणाऱ्या सरकारच्या जमाखर्चांचा, प्राधान्यांचा, भविष्यातील कार्यक्रमांचा ताळेबंद आणि एकवषीर्य योजना असलेला अर्थसंकल्प ( अर्थात अंदाजपत्रक उर्फ बजेट) केंदीय अर्थमंत्री पी. चिदंबरम सोमवारी सादर करतील. ते केवळ संसदेपुरतेच नसेल. देशभरातील टेलिव्हिजन संचांवर, विविध वृत्तवाहिन्यांवर कोट्यवधी प्रेक्षक ते ' लाइव्ह ' पाहात असतील. पूवीर् साक्षर, सुशिक्षित वर्गालाच समजण्यापुरते मर्यादित असलेले हे बजेट आता टेलिव्हिजन चॅनल्सच्या अलीकडच्या काळात झपाट्याने झालेल्या प्रसारामुळे सर्वसामान्यांपर्यंत पोहोचले आहे. प्रत्येकाला बजेटमधील सर्व तपशील समजण्याची आवश्यकताही नसते. पण, त्यातील तरतुदींचा आपल्या जीवनावर साधारणत: काय परिणाम होईल, कोणत्या वस्तू महागतील, कोणत्या वस्तूंचे भाव तुलनेने उतरतील याचा अंदाज सर्वसामान्य माणूस आता बांधू शकतो. ही आहे ' चॅनलवरील बजेट ' ची किमया!

सोमवारी बजेटचे भाषण केंदीय अर्थमंत्री पी. चिदंबरम वाचून संपवतील व त्यानंतर सुरू होणार आहे खरा ' मीडिया हाइप ' अर्थात इलेक्ट्रॉनिक माध्यमातून चर्चासत्रांची, विश्लेषणांची रेलचेल. आथिर्क विषयांना वाहिलेल्या ' सीएनबीसी-टीव्ही एटीन ' या चोवीस तास चालणाऱ्या इंग्रजी वृत्तवाहिनीने ' विल द ड्रीम टीम डिलिव्हर ?' असा विषय घेऊन कार्यक्रमांची आखणी केली आहे. या वाहिनीवर बजेटविषयक कार्यक्रम देशातील व्यापारविषयक निष्णात निवेदक सादर करतील. आथिर्क क्षेत्रातील नामांकित तज्ज्ञांचे विश्लेषण, विविध उद्योगांतील आघाडीच्या उद्योजकांची मते, प्रतिक्रिया, ख्यातनाम अर्थतज्ज्ञांशी सखोल चर्चा याद्वारे प्रेक्षकांना बजेटच्या सर्व पैलूंचे दर्शन घडवण्याचा या वाहिनीचा दरवषीर्प्रमाणेच प्रयत्न असेल. अर्थमंत्री आपल्या भाषणातून जे विविध प्रश्ान् मांडतील व अर्थव्यवस्थेच्या विविध क्षेत्रांचा विचार करतील त्यावर या कार्यक्रमांतून प्रकाशझोत टाकण्यात येणार आहे. अर्थातच या सर्व चर्चा इंग्रजीतून असतील.

अर्थतज्ज्ञ व पत्रकार डॉ. प्रणय रॉय यांनी प्रवतिर्त केलेल्या ' एनडीटीव्ही ट्वेंटी फोर सेव्हन ' ( इंग्रजी) , ' एनडीटीव्ही इंडिया ' ( हिंदी) आणि अलीकडेच सुरू झालेल्या ' एनडीटीव्ही प्रॉफिट ' या तीन वृत्तवाहिन्याही ' सीएनबीसी-टीव्ही एटीन ' शी जोरदार स्पर्धा करणार आहेत. तिन्ही वाहिन्यांवर तज्ज्ञांचे भरगच्च कार्यक्रम असतील. अर्थव्यवस्थेतील विविध घटकांचे, वर्गांचे, हितसंबंधांचे प्रतिनिधित्व करणारी मते अशा चर्चासत्रांमधून व्यक्त होतील.

विविध इंग्रजी, हिंदी व मराठी वृत्तवाहिन्यांमध्ये असलेल्या जबर स्पधेर्मुळे याही वेळी प्रेक्षकांना उत्तमोत्तम कार्यक्रमांची मेजवानी मिळणार आहे. ' एनडीटीव्ही इंडिया ' च्या स्पधेर्तील ' स्टार न्यूज ', ' सबसे तेज ' असल्याचा दावा करणारे ' आजतक ', ' झी ' या वाहिनी समूहांनीही विश्लेषणाचे, प्रतिक्रिया, मतमतांतरांचे भरगच्च कार्यक्रम आखले आहेत. ' आप को रखे आगे ' ची घोषणा करणाऱ्या ' स्टार न्यूज ' वर उद्योगपती डॉ. कुमारमंगलम बिर्ला हे 28 रोजी रात्री आठपासून बजेट विश्लेषणाच्या कार्यक्रमात सहयजमानाच्या ( ' को होस्ट ') भूमिकेत दिसतील. आतापर्यंत विविध टेलिव्हिजन चॅनल्सवर नामवंत उद्योगपती हे पाहुणे (गेस्ट) म्हणून विविध कार्यक्रमांत दिसले आहेत. यंदा पहिल्यांदाच एक उद्योगपती चर्चा कार्यक्रमाच्या सूत्रसंचालनात सहभागी दिसतील. डॉ. बिर्ला प्रेक्षकांनी विचारलेल्या प्रश्ानंना उत्तरे देऊन अर्थसंकल्पाच्या सामान्य माणसाच्या जीवनावर होणाऱ्या परिणामांची चर्चा करतील. ' क्योंकी ये मेरी जेब है ' या शीर्षकाने चचेर्चे हे कार्यक्रम सादर होणार आहेत.

' झी ' समूहातील ' झी बिझनेस ' या हिंदी भाषेतील पहिल्याच ' बिझनेस चॅनल ' ने मात्र आपल्या कार्यक्रमांसाठी एक अभूतपूर्व ' अँकर पर्सन ' मिळविला आहे. ते आहेत पाच वेळा बजेट सादर करणारे माजी केंदीय अर्थमंत्री यशवंत सिन्हा!

' ई टीव्ही मराठी ' आपल्या हैदराबाद येथील स्टुडिओतून अर्थसंकल्पाच्या विश्लेषणाचा कार्यक्रम प्रसारित करणार असून, याच वेळी मुंबई व दिल्ली येथून विविध तज्ज्ञांचा चचेर्तील सहभाग, देशाच्या विविध भागांतून वेगवेगळ्या क्षेत्रातील तज्ज्ञांच्या प्रतिक्रिया प्रसारित करणार आहे. अशाच स्वरूपातील कार्यक्रम मराठीतून कार्यक्रम सादर करणाऱ्या अन्य वाहिन्यांवरूनही प्रेक्षकांना दिसतील.

चॅनलवरील बजेट हे असे दिमाखदार, झगमगाटाने, मतमतांतरांनी भरलेले असेल यात शंका नाही. पण त्याचा परिणाम काय होणार ? पु. ल. देशपांड्यांचा अंतू बर्वा कोकणातल्या आपल्या खेड्यात वीज आल्यामुळे दारिद्य मात्र उठून दिसेल, असे म्हणतो. चॅनलवरील बजेटमध्येही तसेच काहीसे होण्याची शक्यता आहे. प्रचंड लोकसंख्या, झपाट्याने वाढणारी बेरोजगारी, वाढती महागाई, पायाभूत सोयींचा अभाव, साधनसंपत्तीचा तुटवडा आदी विषय चर्चांच्या आवर्तनांतून पुन्हा घुसळून बाहेर येतील. तरीही चॅनलवाल्यांनी साक्षरतेच्या मर्यादा ओलांडून सर्वसामान्यांना त्यांच्या भाषेत बजेट समजावून सांगण्याची चालविलेली धडपड हे निश्चितच एक सकारात्मक पाऊल आहे.
