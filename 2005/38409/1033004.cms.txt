महामुंबई
अभिजीत, अमित... दोघेही खरे 'आयडॉल'
[ Saturday, February 26, 2005  12:09:48 am]


म. टा. प्रतिनिधी

मुंबई : त्या दोघांपैकी कुणातरी एकाच्याच शिरावर ' इंडियन आयडॉल ' चा ताज चढणार आहे... एक कोटीच्या काँट्रॅक्टने कुणातरी एकाचेच आयुष्य पालटणार आहे... तो ' स्टारडम ' च्या प्रकाशात पोहोचणार आहे... दुसऱ्याला मात्र प्रसिद्धीच्या शिखरावरून एका क्षणात अंधारात पायउतार व्हावे लागणार आहे... त्यांच्यापैकी कोण ' इंडियन आयडॉल ' होणार यावरून त्यांच्या चाहत्यांमध्ये अघोषित ' एसएमएसयुद्ध ' सुरू आहे... ' आपला ' माणूस जिंकवा, असे संदेश जाताहेत, माणसे त्यांच्यावरून हमरीतुमरीवर येताहेत... पण, अभिजीत सावंत आणि अमित साना हे दोन स्पर्धक, सामान्य मध्यमवगीर्य कुटुंबांतून पुढे आलेली ही विनम्र तरुण मुले मात्र एकमेकांशी खऱ्या जिव्हाळ्याने, आपुलकीने वागताहेत... त्यांच्या इतरांशी वागण्यातही कुठेही ' स्टारडम ' चे गालबोट लागलेले नाही... स्पधेर्तला ' इंडियन आयडॉल ' कोणीही ठरो... हे दोघेही ' आयडॉल ' ठरावेत, असे ' आयडियल ' वागताहेत...

... ताडदेवच्या ' व्हेलॉसिटी पब ' मध्ये बॉलिवूडच्या स्टार मंडळींसोबत त्यांना पाहताना हे अप्रूप जाणवते. अजूनही दोघांचे थोडेसे गांगरणे पाहिल्यावर ' अरे, हे तर आपल्यातलेच कोणी तरी आहेत की! ' असे चटकन जाणवते.

मात्र , ' मराठी मुंबई ' त ' आमचा अभिजीत... व्होट फॉर अभिजीत, व्होट फॉर मराठी, अरे, इंडियन आयडॉल बनणार कोण, अभिजीत सावंत आणखी कोण ?' असे नारे सुरू असताना ' व्हेलॉसिटी पब ' च्या अमराठी वातावरणात मात्र ' अमित सानाकोही एसएमएस करना... अरे क्या मराठी, मराठी, मराठी... अभिजीत नही अमित ही इंडियन आयडॉल बनेगा... ' अशी जोरदार चर्चा चालू आहे. त्यातच परीक्षकांचा झुकणारा कौल, पक्षपातीपणावरून उठणारे मोहोळ या सगळ्या प्रोफेशनल मगजमारीत आयडॉलचा किताब शेवटी कोणाला मिळणार हे जनता जनार्दनाच्या कौलावरूनच ठरेल.
