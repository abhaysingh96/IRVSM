अर्थव्यवहार
फ्रँचायजिंग र्वक्स!
[ Monday, January 03, 2005  11:34:33 pm]


जन्माने मुंबईकर, माझगावकर असलेल्या झफर जानमोहम्मद खान यांचे वडील एका छोट्या हार्डवेअर व्यापारी कंपनीत व्यवस्थापक होते. नाशिकच्या ' बॉईज टाउन ' बोडिर्ंग स्कूलमधील शिक्षणानंतर झफरनी जे. जे. स्कूलमधून 1973 साली ' जी. डी. आर्ट-कमशिर्यल ' पदवी मिळविली. ' इंडिया बुक हाउस ' मध्ये आटिर्स्ट म्हणून वर्षभर नोकरी केल्यानंतर गोदरेजचे माजी आर्ट डायरेक्टर चक्रवतीर् यांनी झफरना आपल्या डिझाइन स्टुडिओत भरपूर काम, मोबदला दिला. ' जेजे ' त असताना रिबेका ज्यो सॅम्सन नावाच्या विद्याथिर्नीशी झफर यांची ओळख, ओळखीचे रुपांतर प्रेमात, कालांतराने ( बऱ्याच संकटांनंतर) विवाहात झाले.

जन्माने ज्यू असलेल्या (पण मराठी भाषा उत्तम येणाऱ्या) रिबेकांना नंतर चक्रवतीर्ंच्याच स्टुडिओत आटिर्स्ट म्हणून नोकरी मिळाली. 1977 मध्ये झफर, रिबेका यांनी स्वत:चा व्यवसाय सुरू करण्याचे ठरविले. झफरनी ' अॅटलांटिस आयएमसी ' ( इंटिग्रेटेड माकेर्टिंग कम्युनिकेशन) ही प्रोप्रायटरी फर्म नोंदणीकृत केली. झफर व रिबेका विवाहबद्ध झाले. रिबेकांच्या बचतीचे, त्यांच्या वडिलांनी दिलेले पैसे गुंतवून ताडदेवच्या एअर कंडीशन्ड माकेर्टमध्ये ऑफिसची 160 चौरस फूट जागा खरेदी केली. ' मेडिकल प्रॉडक्ट्स ऑफ इंडिया ' नावाच्या कंपनीचे पॅकेजिंग डिझाइनचे, आर्टवर्कचे, नंतर ' प्रिस्क्रिप्शन डिझाइन स्टुडिओ ' चे, अशी कामे मिळाली. ' क्रिएटिव्ह माकेर्टिंग ' नावाच्या कंपनीने ' अॅटलांटिस ' साठी व्यवसायाची दारे खऱ्या अर्थाने खुली केली. वर्धमान समूहाच्या प्रेस लेआउट, होडिर्ंग लेआउट डिझाइनचे काम , ' वर्धमान ' मार्फ त ' कचिन्स ' च्या डिझायनिंगचे, रिलायन्सच्या ' विमल ' सूटिंग व शटिर्ंगच्या ' डीलर कॅम्पेन ' च्या जाहिरातींच्या डिझाइनचे काम मिळाले. ' मेल्स्टार ' कडून अॅडव्हर्टायजिंग डिझाइन, संपूर्ण अॅडव्हर्टायजिंग कॅम्पेनचे काम ' अॅटलांटिस ' ला मिळाले. झफरनी आता माकेर्टिंग मॅनेजमेंट, फ्रँचायजिंगचा अभ्यास सुरू केला. 1980 ते 2001 या काळात झेनिथ कम्प्युटर्स, झेनिथ पाइप्स, हेक्सावेअर, जयभारत सिल्क मिल, जेटकिंग इन्फोट्रेन, इंटरनॅशनल स्कूल फॉर कम्प्युटर टेक्नॉलॉजी, डेटाप्रो, डायटल इलेक्ट्रॉनिक्स, एबीसी कन्सल्टंट्स, सिस्टम्स माकेर्टिंग, बॉम्बे मशिन इंजिनीअर्स हे क्लायंट मिळाले.

झफरनी 1991 मध्ये माकेर्टिंग व फ्रँचाइज कन्सल्टंटचा व्यवसाय सुरू करून छोट्या कंपन्यांना व्यावसायिक सल्ला पुरवून व्यवसायवृद्धीतून त्यांना मोठ्या बनविण्याचे प्रयोग सुरू केले. त्यांना घसघशीत यश मिळाले! झफरनी 2002 ते 2004 या काळात ' डिजिटल अकॅडमी फिल्म स्कूल ' आणि ' टेलिमॅन इन्स्टिट्यूट ऑफ मोबाइल टेक्नॉलॉजी, रिपेअर्स अॅण्ड मेन्टेनन्स ' या प्रशिक्षणसंस्थांना अॅडव्हार्टायजिंग, बिझनेस प्लॅन, कोर्सवेअर, माकेर्टिंग प्लॅन, ट्रेनिंग प्रोग्रॅम आणि फ्रँचायजी शोधून त्यांची नियुक्ती करणे, सपोर्ट टू फ्रँचायजी नेटवर्क ऑन अ डे टू डे बेसिस, अशा सर्व सेवा पुरविण्याच्या ' कम्प्लीट टर्न की प्रॉजेक्ट ' चे करार केले.

ठाण्यात मुख्यालय, प्रशिक्षणसंस्था असलेल्या ' टेलिमॅन ' ने सप्टेंबर 2004 मध्ये ' अॅटलांटिस ' च्या बिझनेस प्लॅननुसार व्यवसायाला प्रारंभ केल्यानंतर अपेक्षेच्या चौपट प्रतिसाद मिळाला. दोन महिन्यात 100 विद्याथीर् मिळाले.

दरमहा 30 ते 60 विद्याथीर् प्रवेश घेतात. ' टेलिमॅन ' साठी दादर पूर्व, अंधेरी पश्चिम, वाशी (नवी मुंबई), पनवेल येथे मिळून चार फ्रँचायजी ' अॅटलांटिस ' ने नियुक्त केले असून, त्यांनी प्रत्येकी 12 लाखांची गुंतवणूक केली आहे.

फेब्रुवारीत त्यांच्या व्यवसायाचा शुभारंभ होणार आहे. ' डिजिटल अकॅडमी ' ला कोलकाता येथे एक फ्रँचायजी मिळाला असून, त्याच्या व्यवसायाचे उद्घाटन येत्या 29 जानेवारीला होणार आहे. चित्रपट व टेलिव्हिजन कार्यक्रमांसाठी स्पेशल इफेक्ट्स, एडिटिंग यांचे प्रशिक्षण ' अकॅडमी ' चे फ्रँचायजी देणार आहेत.

' अॅटलांटिस ' चे आता अंधेरी पूवेर्त चांदिवली रोडवर ओबेरॉय इस्टेटमध्ये स्वत:चे कार्यालय असून , 222. ड्डह्लद्यड्डठ्ठह्लद्बह्यद्बद्वष्.ष्श्ाद्व ही वेबसाइटही आहे.

झफर व रिबेका यांच्यासह ' अॅटलांटिस ' मध्ये आता दोन फ्रँचाइज मॅनेजर व एका आर्ट डायरेक्टरसह सहाजण काम करतात.

प्रारंभिक गुंतवणूक: 21 हजार रु.

आताची उलाढाल: 46 लाख रु.
