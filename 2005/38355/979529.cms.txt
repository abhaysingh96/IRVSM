लेख: जाता जाता
न्यू इयर रिझोल्युशन!
[ Monday, January 03, 2005  09:52:29 pm]


' बाबा, यंदा तुमचं न्यू इयर रिझोल्युशन सांगितलं नाहीत. ' ज्यु. चकोरी फार आस्थेने चौकशी करत असली तरी तिच्या चेहऱ्यावरचे भाव वेगळे आहेत, असं चकोरला वाटलं.

' आज तुला बंधूराज भेटले नाहीत वाटतं! ' त्याने विचारलं.

' तो भेटला, पण म्हणाला की बाबांचा काय आहे ते विचार. ' तिने चकोरला आपल्या मनात काय आहे, याचा थांगपत्ता लागू दिला नाही.

' तू ओळख बघू, काय असेल तो! ' आता त्याने तिची फिरकी घेण्याचं ठरवलं.

' आईचं ऐकायचं. ' तिने सांगितलं.

' लग्न झाल्यापासून तेच करतो आहे. ' त्याने चकोरीकडे बघत सांगितलं.

' मग आईचं अजिबात ऐकायचं नाही. ' मुलीने दुसरा पर्याय दिला.

' त्यासाठी हिम्मत लागते. ती माझ्यात नाही. ' चकोरने प्रांजळ कबुली दिली.

चकोरी चहा करता करता हे सारं ऐकत होती. प्रत्येक वाक्यागणिक तिच्या मुखावरचे भाव बदलत होते. बाबांच्या या प्रतिक्रियांवर कन्या खिदळत होती.

' आईला आज्ञा सोडायची. ' आता तीपण जरा स्वातंत्र्य घ्यायला लागली.

यावर चकोरने, काही न बोलता असं काही तोंड केलं की मुलीला हसू आवरेना.

आता मात्र चकोरीला राहवेना.

' काय गं, आज तुला काही उद्योग दिसत नाही. कॉलेजला नाही जायचं! ' तिने किंचित रागावून विचारलं.

' कॉलेजला खूप दांड्या मारायच्या असं तिचं रिझोल्युशन असेल. ' चकोरने संधी साधली.

' त्यासाठी रिझोल्युशन कशाला करायला हवं! ते न करताही लेक्चर्स बंक करत असेल. ' चकोरीने आता लेकीला सापळ्यात पकडलं. ' नाही हां मॉम! ' तिने बाजू मांडायचा प्रयत्न केला. ' काही सांगू नकोस. तुमचं फोनवरचं बोलणं मी ऐकत नाही असं वाटलं तुला! ' तिने मुलीला विचारलं.

या संवादाकडे आपलं लक्ष नाही असं दाखवत चकोर ' म. टा. ' वाचत बसला. पण कन्येला कळलं की बाबा हे एन्जॉय करताहेत.

' तिला मोबाईल घेऊन द्यायला हवा. म्हणजे तू यांचे सुसंवाद टॅक करणार नाहीस. ' चकोरने आपण सहज सूचना करत आहोत, अशा टोनमध्ये सांगितलं.

' काही गरज नाही. नसता वायफळ खर्च आणि सतत मित्रमैत्रिणींशी गॉसिप! ' चकोरीने निर्णय देऊन टाकला.

मुलीने ' आता पुरे ना! ' अशा अर्थाने बाबांकडे बघून हात जोडले.

जा, वत्सा, तुला माफ केलं अशा थाटात त्याने हात वर केले.

आपलं लक्ष नाही असं दाखवणारी चकोरी हसली.

चकोर
