लेख: जाता जाता
मंत्र्यांची नाटककंपनी!
[ Tuesday, February 01, 2005  08:24:29 pm]


राज्य मंत्रिमंडळाची बैठक सुरू झाली. मुख्य सचिवांनी अजेंड्यावरचा पहिला विषय वाचला. ' मंत्रिमंडळाची नाटक कंपनी स्थापन करणे. ' सूचक : आर. आर. पाटील.

' तिजोरी रिकामी आहे. पहिल्या नाटकाचं नाव ' पैशाचं सोंग आणता येत नाही ' असं ठेवा. ' अर्थमंत्री जयंत पाटील यांच्या स्वरात पूर्ण उपरोध होता.

' आबांवरचा राग गेलेला दिसत नाही. ' विलासराव पतंगरावांच्या कानात कुजबुजले.

' हा प्रस्ताव उच्चवणीर्यांसाठी आणि मध्यमवगीर्यांना खूष करण्यासाठी आहे. ' चंदकांत हांडोरे यांनी सांगून टाकले.

' त्यापेक्षा आमच्या मतदारसंघात चौफुला आहे. तेथील आमच्या कलाकारांना उत्तेजन मिळेल असं बघा! ' अजितदादांनी भूमिका घेतली.

' आरं, पण नाटकात काम कोण करणार! अडवा आणि जिरवामध्ये जीव जायची वेळ आली हाय! ' पतंगरावांचा तारस्वरात आक्षेप ' विलासरावांना इथेही हीरोची भूमिका मिळणार म्हणून संताप झाला वाटतं! ' वळसे-पाटील पुटपुटले.

' या वेळी विदर्भाचा बॅकलॉग भरून काढायलाच हवा! ' कायम हीरोच्या थाटात वावरणाऱ्या अनिल देशमुखांनी मागणी केली. ' प्रस्तावात नमूद केलं आहे की, रिहर्सलसाठी मंत्रालयात स्वतंत्र रूमची व्यवस्था करण्यात यावी. ' मुख्यमंत्र्यांनी विषय बदलला.

' हितं स्टाफला बसायला जागा नाही, रिहर्सल कसली करता! ' पतंगरावांनी खडा सवाल केला.

' राणे किंवा मुंडे यांना रोल देऊया. ' विरोधकांत फूट पाडणारी सूचना क्वचित बोलणाऱ्या मोहिते-पाटलांनी केली.

' डायरेक्टर कोण ?' कोपऱ्यात बसलेल्या भुजबळांनी शांतपणे विचारलं.

' आघाडीचे नियम लक्षात घेऊन, हीरो ज्या पक्षाचा त्या पक्षाचा डायरेक्टर नसावा. ' गणेश नाईक यांनी मागणी केली. ' शरद पवार! ' विलासरावांचं स्वगत.

' पुन्हा हीरोलाच जादा फुटेज. ' - भुजबळ.

' पण ही कंपनी कल्चरल डिपार्टमेंटच्या अंडर हवी! इथे ती सीएम ऑफिसशी अॅटॅच दाखवली आहे. ' अशोक चव्हाण यांना संताप आवरत नव्हता.

' हा काय हिंदी पिक्चर आहे काय! फक्त हीरोची दादागिरी चालायला. ' अकलुजच्या दादांना कल्चरल डिपार्टमेंट सोडून बरीच वर्षं झाली असली, तरी त्यांचा टच कायम होता.

' आपण ठराव मांडणाऱ्या आबांचं मत विचारातच घेत नाही. ' सी. एम.नी आबांकडे बोट दाखवत मुद्दा मांडला.

' माझं इतकंच सांगणं आहे की आपण रोज जी नाटकं करतो ती नीट करावीत. म्हणजे, मध्यंतरी अजितदादांनी, माझ्या ' मावा ' खाण्याबद्दल जे डायलॉग मारले ते डायरेक्टरच्या परवानगीशिवाय होते. अशी अॅडिशन घेऊ नये... ' आबांनी खुलासा केला आणि सगळ्यांनीच निश्वास सोडला.
