क्रीडा
महाराष्ट्राच्या संघांची आगेकूच
[ Tuesday, January 25, 2005  09:56:47 pm]


लोधिवली : 66 व्या राष्ट्रीय टेबल टेनिस सांघिक स्पधेर्त महाराष्ट्राने आपले सामने लीलया जिंकून आगेकूच चालूच ठेवली आहे. रिलायन्स पुरस्कृत 5 ।। लाख रुपये बक्षिस रकमेच्या या स्पधेर्त पुरुषांच्या ' इ ' गटात महाराष्ट्र ' अ ' संघाने तर ' ह ' गटात महाराष्ट्र ' ब ' संघाने प्रतिर्स्पध्यांवर सहज मात केली. महाराष्ट्राच्या महिला ' अ ' संघाने ' ग ' गटात आपला सामना आरामात जिंकला. महाराष्ट्राच्या महिला ' ब ' संघाने उत्तरप्रदेशला 3-0 असे नमवले.

या सांघिक स्पधेर्च्या पहिल्या दिवसाप्रमाणे दुसऱ्या दिवशीही महाराष्ट्रानेच वर्चस्व गाजवले.

' इ ' गटात महाराष्ट्र ' अ ' संघाने झारखंड संघावर 3-0 अशी मात केली. अनिकेत कोपरकरने आपल्या चौफेर खेळाच्या जोरावर झारखंडच्या स्वर्णकारला 11-1, 11-2, 11-6 अशी मात केली. तर भूषण ठाकूरने पार्थसारथीचा 11-5, 11-6, 11-9 असा पराभव केला. ओंकार तोरगळकरने निरजकुमारचा 11- 6, 11-6, 11-2 असा फडशा पाडला.

सामन्यांचे निकाल

पुरुष : ' ह ' गट : महाराष्ट्र ' ब ' विजयी वि. त्रिपुरा 3-0 ( आदित्य महागावकर विजयी वि. नसिम अली 11-7, 11-6,11- 6. स्वप्नील नामजोशी विजयी वि. बिशन रॉय 11-6, 11-6, 12-10. रोहित चौधरी विजयी वि. देवनाथ 11-6,11-2,11- 6.)

महिला : ' फ ' गट : महाराष्ट्र ' ब ' विजयी वि. उत्तर प्रदेश 3-0 ( जान्हवी देशपांडे विजयी वि. विनिता सिंग 11-6,11-6,11-6. मानशी भागवत विजयी वि. शालीनी शर्मा 11-6,12-10,6- 11,12-10. आष्लेषा बोडस विजयी वि. सिता वर्मा 11-6,6-11,11-6,11-6.)

' ग ' गट : महाराष्ट्र ' अ ' विजयी वि. छत्तीसगढ 3-0 ( मधुरिका पाटकर विजयी वि. श्ाीवास्तव 15-13,11-6,11-6. शाल्मली म्हात्रे विजयी वि. रचिता सावळे 11-6,11-6,11-3. प्राजक्ता टिपले विजयी वि. राखी जैन 11-4,11-6,11-6.)
