अर्थव्यवहार
'कमोडिटी एक्स्चेंज'ची दुनिया
[ Friday, April 22, 2005  08:01:53 pm]


' माझ्या डिमॅट खात्यात 500 किलो गहू जमा आहेत किंवा 100 ग्रॅम सोने जमा आहे ', असे कुणी म्हटले तर समोरील व्यक्ती भुवया उंचावून पाहील यात शंका नाही ; पण आता ' कमोडिटी एक्स्चेंज ' च्या स्थापनेनंतर हे सहज शक्य झाले आहे. शेअर्सचे ' डिमॅट ' (' डिमटेरियलायजेशन ') आता सर्वांना परिचित आहेच. ज्याप्रमाणे शेअर्सचे खरेदी/विक्री व्यवहार मुंबई स्टॉक एक्स्चेंज, नॅशनल स्टॉक एक्स्चेंज येथे होतात, तसे वर उल्लेख केलेल्या गहू, सोने इत्यादी वस्तूंचे खरेदी/विक्री व्यवहार ' डेरिव्हेटिव ' स्वरूपात हृष्टष्ठश्वङ्ग म्हणजेच ' नॅशनल कमोडिटी अँड डेरिव्हेटिव एक्स्चेंज ' तसेच रूष्टङ्ग म्हणजे ' मल्टि कमोडिटी एक्स्चेंज ' येथे होतात. या दोन कमोडिटी एक्स्चेंजेसमध्ये ज्या वस्तूंचे व्यवहार होतात त्या वस्तूंना ' कमोडिटी ' असे संबोधले जाते. ढोबळपणे ' कमोडिटी ' म्हणजे कच्च्या स्वरूपातील वस्तू. उदाहरणच द्यायचे झाले तर चणे ही ' कमोडिटी आहे ' पण चण्याचे पीठ मात्र ' कमोडिटी ' नाही. कारण ते तयार स्वरूपातील मालाच्या (द्घद्बठ्ठद्बह्यद्धद्गस्त्र द्दश्ाश्ास्त्रह्य) स्वरूपात मोडते. सध्या जरी मोजक्याच कमोडिटींचे व्यवहार येथे होत असले तरी ' डिमॅट ' प्रणालीमुळे त्यात लवकरच खूप वाढ होईल यात शंका नाही.

तत्त्वत: स्टॉक एक्स्चेंज व कमोडिटी एक्स्चेंज यामध्ये बरेच साम्य असले, तरी कार्यपद्धती मात्र काहीशी भिन्न आहे. मुख्य म्हणजे अन्य स्टॉक एक्स्चेंजेसचे दलाल थेट ' कमोडिटी एक्स्चेंज ' मध्ये व्यवहार करू शकत नाहीत. त्यासाठी त्यांना कमोडिटी एक्स्चेंजचे स्वतंत्रपणे सभासद होणे जरूरीचे असते. कमोडिटी एक्स्चेंजचे दलाल , ' सीडीएसएल ' ( डिपॉझिटरी) , ' डीपी ' ( डिपॉजिटरी पाटिर्सिपंट), वेअर हाउस (इश्युुअर) , ' आरटीए ' ( रजिस्ट्रार अॅण्ड ट्रान्स्फर एजंट) आणि गुंतवणूकदार हे सर्व घटक येथेही असतात.

कमोडिटींच्या ' डिमॅट ' यंत्रणेत सध्या साखर, गहू, तेलबिया, सोने, कापूस, जिरे, ज्यूट, चणे इत्यादी कमोडिटींचे ' डिमॅट ' होण्याची सोय आहे. कमोडिटी एक्स्चेंजचे मान्यताप्राप्त दलाल , ' डीपी ' तसेच वेअर हाउस असतात. डिमॅट शेअर्सच्याबाबतीत कंपनी इश््युुअर असते, तर कमोडिटी डिमॅटमध्ये वेअर हाउस हे ' इश््युुअर ' म्हटले जातात. उदाहरण म्हणून कमोडिटीचे ' डिमॅट ' कसे होते ते पाहू. समजा रामभाऊंकडे 100 ग्रॅम सोने आहे. त्यांनी सर्वप्रथम सोने घेऊन कमोडिटी एक्सचेंजच्या मान्यताप्राप्त वेअर हाउसमध्ये जाऊन (सोने, चांदी इत्यादी कमोडिटीजच्या बाबतीत वेअर हाउसच्या जागी व्हॉल्ट असते). तेथील निरीक्षकाकडून (ड्डह्यह्यड्ड 4 द्गह्म) मालाची ( सोन्याची) शुद्धता, वजन प्रमाणित करून घ्यायचे. त्यानंतर ' वेअर हाउस ' तफेर् रामभाऊंना योग्य ती पावती दिली जाते. या पावतीत मालाचा सर्व तपशील असतो. त्या पावतीची दुसरी प्रत वेअर हाउस ' आरटीए ' कडे पाठवते. वेअर हाउसचा कम्प्युटरद्वारे ' आरटीए ' शी संपर्क असतो. त्याअन्वये ही माहिती ' आरटीए ' कडे पाठविली जाते. ' आरटीए ' उपरोक्त सोन्यासाठी एक स्वतंत्र ' आयझिमन ' क्रमांक देतो. लगेचच सीडीएसल आपल्या ' डेटाबेस ' मध्ये तो ' आयझिन ' सामील करून घेते. त्यानंतर ' आरटीए ' कॉपोर्रेट अॅक्शन करून ' डिमॅट ' खात्यात संबंधित कमोडिटीचे युनिट्स रामभाऊंच्या डिमॅट खात्यात जमा करते. ज्याप्रमाणे शेअर्सचे आयझिन क्रमांक बव्हंशी ढ्ढहृश्व या अक्षरांनी सुरू होतात, त्याप्रमाणे कमोडिटीचे आयझिन ढ्ढहृष्ट या अक्षरांनी सुरू होतात.

वेअर हाउसमध्ये जमा करण्यात येणाऱ्या प्रत्येक कमोडिटीला समाप्तीची तारीख (द्ग 3 श्चद्बह्म 4 स्त्रड्डह्लद्ग) मूल्यमापकाकडून ( ड्डह्यह्यड्ड 4 द्गह्म) देण्यात येते. कमोडिटीचा नाशिवंतपणा इत्यादी बाबी विचारात घेऊन समाप्तीची तारीख ठरविली जाते.

त्यामुळे चार डिमॅट खातेदारांनी एकाच प्रकारची कमोडिटी वेअर हाऊसमध्ये डिमॅट करण्यासाठी सुपूर्द केली तरी दर्जानुसार, तसेच समाप्तीची तारीख यानुसार त्यांना चार वेगळे ' आयझिन ' दिले जाण्याची शक्यता असते. डिमॅट करण्याच्या या प्रक्रियेत वेअर हाउसचे चाजेर्स, मूल्यमापकाचे (ड्डह्यह्यड्ड 4 द्गह्म) चाजेर्स इत्यादी आकार ' डिमॅट ' खातेदाराला भरावे लागतात.

शेअर्सच्या डिमॅट खात्यात किमान एक शेअर ठेवण्याची सोय असते. मात्र ' कमोडिटी डिमॅट ' खात्यात विशिष्ट युनिट्समध्येच कमोडिटी ठेवाव्या लागतात. महत्त्वाची बाब म्हणजे शेअर्सच्या डिमॅट खात्यात कमोडिटीचे ' आयझिन ' धारण करण्यास परवानगी नाही. कमोडिटी धारण करण्यासाठी स्वतंत्र डिमॅट खाते उघडणे जरूरीचे आहे.

याचे कारण म्हणजे कमोडिटी व्यवहारांवर ' फॉर्वर्ड माकेर्ट कमिशन ' चे नियंत्रण असते. तसेच कमोडिटी या ' सेक्युरिटीज ' गटात मोडत नाहीत. वेअर हाउसमधून कमोडिटीची डिलिव्हरी घेणे ( ह्मद्गद्वड्डह्लद्गह्मद्बड्डद्यद्बह्यड्डह्लद्बश्ाठ्ठ) तसेच कमोडिटी एक्स्चेंजेसमध्ये झालेले व्यवहार डिमॅट यंत्रणेमार्फत कसे पूर्ण केले जातात ते यथावकाश पुढे वेगळ्या लेखात पाहू.

चंदशेखर ठाकूर
