महाराष्ट्र
'एचएएल'च्या नफ्यात 5.1 टक्क्यांची वाढ
[ Wednesday, April 13, 2005  04:54:54 pm]


म. टा. प्रतिनिधी

नाशिक : ' हिंदुस्तान एरोनॉटिक्स लिमिटेड ' च्या नफ्यात (एचएएल) 2004-05 या आथिर्क वर्षात 5.1 टक्क्यांची वाढ झाली आहे. कंपनीने 4425 कोटी रुपयांची एकूण विक्री केली करून या आथिर्क वर्षात 630 कोटी रुपयांचा नफा कमावला.

नुकतीच ' सुखॉय- 30 एमकेआय ' या लढाऊ विमानांची पहिली तुकडी ' एचएएल ' ने भारतीय हवाई दलास सुपूर्द केली आहे. याशिवाय दोन आसनी ' जग्वार ' प्रकल्पासाठी लागणाऱ्या साधन सामुग्रीचा पुरवठा व देशीकरणातून साध्य केलेली ' एव्हीऑनिक्स ' व कम्प्युटर प्रणालीचे नूतनीकरण करून कंपनीने एक महत्त्वाचा टप्पा गाठला आहे. त्याचप्रमाणे ' धुव ' हेलिकॉप्टर्सची निमिर्ती करून देशातील विविध खासगी क्षेत्रांना पुरवठा केला आहे. धुवसाठी वापरली जाणारी प्रक्षेपण प्रणाली संकलित करण्याच्यादृष्टीनेही कंपनीची वाटचाल सुरू आहे.

हवाई क्षेत्रातील उत्पादनात ' एचएएल ' ने प्राप्त केलेल्या नावलौकिकामुळे आंतरराष्ट्रीय क्षेत्रातील प्रसिद्ध विमान उत्पादक कंपन्यांनी विमाननिमिर्तीसाठी या कंपनीशी भागीदारी करण्यात रस दाखविला आहे. फ्रान्सच्या ' एअर बस ' कंपनीने ' एअर बस ए- 323' या प्रवासी विमानांच्या फॉरवर्ड पॅसेंजर डोअर्ससाठी वापरण्यात येणाऱ्या एक हजार शिपसेटस तयार करण्यास ' एचएएल ' ला सांगितले आहे.

या व्यतिरिक्त युरो-कॉप्टरने त्यांच्या हेलिकॉप्टर्ससाठी लागणाऱ्या सुट्या भागांचा पुरवठा आणि जुळणीचे काम, बेल हेलिकॉप्टरसाठी लागणाऱ्या टेल रोटर ब्लेडची निमिर्ती आणि जुळणी अशी कामेही आंतरराष्ट्रीय बाजारातून ' एचएएल ' ला मिळालेली आहेत. इस्त्रायलला ध्रुव हेलिकॉप्टरची विक्री करून एचएएलने निर्यात क्षेत्रातही पदार्पण केले आहे.

' एचएएल ' मध्ये री-इंजिन्ड केलेल्या चिंतन या हेलिकॉप्टरने 25150 फुटांवर उतरून जगातील अतिउंच ठिकाणावरचे लँडिंग करण्याचा विश्वविक्रम प्रस्थापित केला आहे. तर ध्रुव या हेलिकॉप्टरने 27 हजार फूट उंचीवर उड्डाण करून नवीन उच्चांक घडविला आहे. ' सुखॉय- 30 एमकेआय ' आणि ' हॉक अॅडव्हान्सड जेट ट्रेनर ' यांची निमिर्ती, तसेच ' तेजस ' या हलक्या वजनाच्या लढाऊ विमानाची निमिर्ती आणि ' जेट ट्रेनर्स ' ची पुननिर्मिर्ती व प्रमाणीकरण व त्यांचे साखळी उत्पादन करण्याचे उद्दिष्ट आगामी वर्षात ' एचएएल ' ने ठेवले आहे.
