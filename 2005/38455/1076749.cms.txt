महामुंबई: मुंबई टाइम्स
'अ' का जादू चलेगा क्या?
[ Wednesday, April 13, 2005  06:20:07 pm]


मुंबई टाइम्स टीम

स्टार प्लस आणि ' क ' हे आद्याक्षर हा यशाचा फॉर्म्युला झालाय, तसा काहीसा झी मराठी म्हणजे पूर्वाश्रमीच्या अल्फाच्या बाबतीत घडतोय. या अल्फाच्या नावातील आद्याक्षराचा अ हरवून गेला, पण हा ' अ ' फॅक्टर ' सक्सेस रेट ' च्या दृष्टीने महत्त्वपूर्ण असल्यामुळे नवनव्या मालिकांचं आद्याक्षर ' अ ' पासून ठेवण्याचा ट्रेण्ड झीमध्ये कायम राहिलाय. ' आ ' भाळमाया , ' अ ' वंतिका , ' अ ' धुरी एक कहाणी , ' आ ' मने सामने ' ' आ ' पलं अल्फा अॅवॉर्ड यांच्या यशानंतर आता प्रतिमा कुलकणीर् दिग्दशिर्त ' अंकुर ' ही मालिका आजपासून प्रसारित होणार आहे.

प्रतिमा कुलकर्णींचा स्पेशल टच असणारी ही मालिका असली तरी ती एका वेगळ्या धाटणीची आहे. स्त्री हा या मालिकेचा केंदबिंदू असला, तरी जाणूनबुजून प्रश्नांच्या गर्तेत अडकत प्रेक्षकांची सहानुभूती मिळवून टीआरपीचा आकडा उंचावण्याचं ' गिमिक ' या मालिकेत नसेल.

प्रत्येक घरातील स्त्रीच्या अस्तित्वाचा प्रश्न या ' वीकली ' मालिकेतून मांडण्यात येईल. स्त्रीमुक्तीची कुठेही नारेबाजी न करता दररोजच्या जगण्यात येणाऱ्या छोट्या छोट्या गोष्टींमधून तो मुद्दा प्रकर्षाने मांडण्याचा प्रयत्न यात आहे. नोकरी करणाऱ्या स्त्रियांच्या प्रश्नांसोबत हाऊसवाइफ असणाऱ्यांच्या इच्छा, आकांक्षांचा मुद्दा हे याचं सूत्र आहे. मध्यमवगीर्य स्त्रीला नवश्रीमंत अथवा उच्चमध्यमवगीर्य होताना मोजावी लागणारी स्वत्वाची किंमत आणि छोट्या छोट्या गोष्टींमधून जाणवणारा पुरुषी वर्चस्वाचा मुद्दा आहे, पण त्याकडे बघण्याचा रूटिन आणि गृहित धरणाऱ्या अॅटिट्यूडवर मामिर्क भाष्य करण्यासोबत तीव्र ' स्पॉटलाइट ' टाकण्याचं काम अंकुर करेल, असं मत दिग्दशिर्का प्रतिमा कुलकणीर् यांनी सांगितलं.

या मालिकेत कविता लाड, तुषार दळवी, शैलेश दातार, सीमा देशमुख, राधिका इंगळे अशी रूटिन नसलेली गँग आहे. सध्याची स्त्रीप्रधान मालिकांची उणीव ' झी मराठी ' ने ' अंकुर ' सारख्या मालिकेतून भरून काढत आहेत, पण ' वीकली ' आणि प्रतिमा कुलकणीर् या समीकरणामुळे त्या मालिकेच्या दर्जाचा प्रश्न नाही. प्रतिमा कुलकणीर्ंच्या ' प्रपंच ', '405, आनंदवन ', ' झोका ' अशा मालिकांमधल्या ' व्यक्तिरेखा ' आपल्या घरातल्याच वाटतात. तोच सिलसिला ' अंकुर ' मध्ये कायम राहील, अशी आशा बाळगायला हरकत नाही आणि ' अ ' चा हा फॉर्म्युला हिट ठरतोय का, हेही थोड्याच दिवसांत स्पष्ट होईल.
