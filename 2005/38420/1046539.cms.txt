अर्थव्यवहार
'वाय फाय टेक्नॉलॉजी' झपाट्याने पसरणार आहे
[ Wednesday, March 09, 2005  08:55:05 pm]


प्रश्न्: वायरलेस तंत्रज्ञान ' वायर्ड सव्हिर्स ' पेक्षा का व कितपत महाग आहे ?

सादरीवाला: होय. सेल्युलर वायरलेस टेक्नॉलॉजीच्या वापराचे दर तुलनेने अधिक आहेत, कारण त्यासाठी पायाभूत सोयी उभारण्यासाठी करावी लागणारी अधिक खर्चाची गुंतवणूक. वायरलेस टेक्नॉलॉजी जगात 1967 पासून वापरात आहे, पण तिचा सर्वत्रच प्रसार झालेला नाही. ' सेल फोन टेक्नॉलॉजी ' आणि ' वाय फाय टेक्नॉलॉजी ' (' वाय फाय ' म्हणजे ' वायरलेस फिडेलिटी ') यात फरक आहे. ' वाय फाय टेक्नॉलॉजी ' ही ' सेल फोन टेक्नॉलॉजी ' च्या तुलनेत खूपच गतिमान असून, तीत अतिजलद इंटरनेट सेवा पुरविण्याची क्षमता आहे.

' वाय फाय टेक्नॉलॉजी ' चा भारतात प्रसार कितपत झाला आहे ?

फारच तुरळक. हॉस्पिटॅलिटी (हॉटेल) उद्योगात, तोही ठराविक पंचतारांकित व त्याहून अधिक सुखसोयी पुरविणाऱ्या काही हॉटेलांत ' वाय फाय ' आता कोठे पोहचले आहे. अमेरिकेत मात्र हॉटेल्स, रिसॉर्ट्स, विमानतळे, इस्पितळे यामध्ये ' वाय फाय ' तंत्रज्ञान सार्वत्रिक वापरले जात आहे. तेथील विद्यापीठे, अन्य शिक्षणसंस्था यामध्येही ' वाय फाय ' चा वापर झपाट्याने होत आहे. अमेरिकेतील अनेक शहरे ' वाय फाय एनेबल्ड ' बनली आहेत.

' वाय फाय टेक्नॉलॉजी ' चे भवितव्य काय असेल ? ' वायर्ड सव्हिर्स ' कालांतराने संपुष्टात येईल का ?

अर्थातच. ज्या कंपन्यांकडे आधीची ' वायर्ड सव्हिर्स ' आहे, त्या क्रमश: तिचे रुपांतर ' वाय फाय ' मध्ये करीत आहेत. अशी अनेक मोठी कामे आम्ही अमेरिकेत केली आहेत. ' वाय फाय टेक्नॉलॉजी ' येत्या दहा ते पंधरा वर्षांत जगातील सर्व प्रगत देशांत ' वायर्ड सव्हिर्स ' ची जागा घेईल. विशिष्ट परिसरात (या परिसराची व्यापी बरीच वाढविली जाऊ शकते) व्हॉइस मेल व इंटरनेट सेवा अतिजलद वेगाने पुरविण्यासाठी उभारलेली (अॅण्टेना व अन्य आवश्यक सामग्रीसह) यंत्रणा, असे ' वाय फाय ' चे वर्णन करता येईल.

भारतात गुंतवणुकीच्या आपल्या कोणत्या योजना आहेत ?

आम्ही संधींच्या (बिझनेस मॉडेल्स) शोधात आहोत. हॉटेल उद्योग, आरोग्य सेवा (मोठी इस्पितळे) , ' फ्रीट फॉरवडिर्ंग ' ( मालवाहतूक) या व्यवसायांत संधी दिसताहेत.

भारतात ' वाय फाय एनेबल्ड ' सेवेचे दर केव्हा खाली येतील ? ते सामान्य माणसापर्यंत केव्हा पोहचण्याची शक्यता आहे ?

का नाही पोहचणार ? मोबाइल फोन नाही का भारतात झपाट्याने पसरले ? ' वाय फाय ' तंत्रज्ञान जसजसे विकसित होत जाईल तसतशी इन्फ्रास्ट्रक्चरवरील खर्चात बचत (कमी हार्डवेअरची गरज) होईल व बाजारपेठ विस्तारत जाईल तसतसे या सेवेचा ' एण्ड युजर ' साठी दरही घटत जाईल.

भारतात इंटरनेटचा वेग फारच कमी असतो. तीत ' वाय फाय एनेबल्ड ' सेवेमुळे काय फरक पडू शकेल ?

त्याचे कारण भारतात ' डायलॉग मेथडॉलॉजी ' वर आधारित 56 के मॉडेम इंटरनेटसाठी वापरले जातात. खरी ब्रॉडबॅण्ड गती ' डीएसएल ' वरच (डिजिटल सव्हिर्स लाइन) मिळू शकते. ' डीएसएल ' व ' वाय फाय ' चा मेळ घातला गेल्यानंतर इथेही इंटरनेट अमेरिकेच्या स्पीडने चालू शकेल.

मुलाखत: विश्वास डिग्गीकर
