संवाद
सुवर्णयुगाची चाहूल
[ Saturday, March 19, 2005  10:24:53 pm]


साडी, धोतर, बांगड्या यांच्या चक्रात गोते खाऊ लागलेला मराठी सिनेमा अचानक ' श्वास ' च्या रुपाने ऑस्करवारी करून आला आणि मराठी सिनेमाला गतवैभव मिळणार असा आशावाद व्यक्त होऊ लागला. तो बळकट व्हावा अशा घटनाही घडू लागल्या आहेत.

' मटा सन्मान ' साठी स्पधेर्त आलेल्या मराठी चित्रपटांची संख्या यंदा 37 होती. पाच वर्षांतील हा उच्चांक. विविध वृत्तपत्रांतील समीक्षकांनी पहिले परीक्षण केले. त्यानंतर अंतिम स्पधेर्त आठ सिनेमे होते आणि त्यातील सवोर्त्कृष्टतेची स्पर्धा ' देवराई ', ' मानिनी ' आणि ' सावरखेड एक गाव ' या सिनेमांमध्ये होती. त्यात ' देवराई ' ने बाजी मारल्याचे तर आता सर्वांनाच माहीत आहे.

सुमित्रा भावे आणि सुनील सुकथनकर यांचा ' देवराई ' हा सिनेमा म्हणजे स्किझोफेनिक व्यक्तींकडे कसे पाहावे, याचा आदर्श वस्तुपाठ आहे. स्किझोफ्रेनियाची माहिती देणारा, शुभाथीर्-शुभंकराच्या संकल्पना स्पष्ट करणारा ढोबळ माहितीपट दिग्दर्शकद्वयाने नातेसंबंध, बदलत्या कुटुंबरचना, बदलता सामाजिक संदर्भ आणि आधुनिक विज्ञानाला उमजलेला ' मायेच्या इलाजा ' चा अर्थ अशा वेगळ्या उंचीवर नेला. व्यक्तिरेखांची मांडणी, त्यांच्या संवादांचा पोत आणि विषयाचे सातत्याने राखलेले भान यामुळे सिनेमा भरकटला तर नाहीच उलट पाहणाऱ्या प्रत्येक रसिकाला मुळापासून हलवू शकला.

राजीव पाटील दिग्दशिर्त ' सावरखेड एक गाव ' आणि कांचन अधिकारी दिग्दशिर्त ' मानिनी ' या सिनेमांनी वेगळे विषय घेऊन धाडस दाखवले. ' सावरखेड ' ची निमिर्तीमूल्येही दजेर्दार होती. एरवी तांत्रिक आघाडीवर मार खाणारा मराठी सिनेमा, या आघाडीवर मेहनत घेत असल्याचे चित्र या सिनेमांमुळे दिसले. ' अगंबाई अरेच्चा ', ' सातच्या आत घरात ' आणि ' उत्तरायण ' या सिनेमांनी तंत्राची श्रीमंती दाखवली.

अनुभवविश्व समृद्ध करण्यात सिनेमाचा वाटा मोठा असतो. ती सहज प्रक्रिया असते. जितकी निमिर्ती उत्तम तितका त्या समृद्धीचा पोत गहिरा. मराठी रसिकांच्या सुदैवाने अधिकाधिक गहिऱ्या अनुभवांची मांडणी करणारा ' श्वास ' पाठोपाठ ' देवराई ' असे सिनेमे आल्यामुळे मराठी सिनेमांच्या अस्तित्वापुढचा प्रश्न दूर झाला आहे. आता लढाई आहे ती हे अस्तित्व ठळक जाणवून देण्याची. गेल्या दशकात चंदकांत कुलकणीर् यांचा ' बिनधास्त ' व ' भेट ', महेश कोठारेंचा ' चिमणी पाखरं ' आणि ' पछाडलेला ', सुमित्रा भावे-सुनील सुकथनकर जोडीचे ' दहावी फ ' आणि ' वास्तुपुरुष ' या सिनेमांनी हळुहळू बांधत आणलेली मराठी सिनेमाची मोट ' श्वास ' आणि ' देवराई ' ने चांगल्यापैकी सांधली.

हे चित्र सुखद खचितच! पण संख्या वाढली म्हणजे गुणवत्ताही वाढली, असे नाही. सांगली, सातारा, इचलकरंजी, इस्लामपूर, बाशीर् येथे आणि गावोगाव जत्रेत चालणारे मराठी सिनेमे आणि शहरी प्रेक्षकांना आवडणाऱ्या सिनेमांची तुलनाच होऊ शकत नाही. ग्रामीण भागातील त्या सिनेमांची अर्थगणिते सर्वस्वी भिन्न आहेत. ते चित्रपट शहरात चालत नाहीत म्हणजे त्यांची गुणवत्ता उणी आहे, असेही नव्हे.

यंदा मराठी सिनेमांची संख्या साठाच्या घरात जाईल. म्हणजे पुढच्या वषीर् ' मटा सन्मान ' साठी आणखी चुरस वाढणार आणि रसिकांनाही वेगळे मराठी सिनेमे पाहण्याची संधी मिळणार. या चचेर्चा

अर्थ गतप्रभ मराठी चित्रपटसृष्टीला नवतेचे दान देणाऱ्या हरहुन्नरी दिग्दर्शकांची संख्या वाढली आहे. त्यामुळे आता या माध्यमावर पेम करणाऱ्या निर्मांत्याची संख्याही वाढली, तर हे समीकरण वैभव मिळवून देण्यास उपकारक ठरेल.

मराठी चित्रपटांचा प्रवास हा तीन मार्गांवरून चालला आहे. ' श्वास ' व ' देवराई ' हा अव्वल दर्जाच्या सिनेमांचा एक मार्ग. ' अगंबाई अरेच्चा ' व ' नवरा माझा नवसाचा ' हा दुसरा आणि ' काळूबाईच्या नावानं चांगभलं ' व ' सुहासिनीची सत्त्वपरीक्षा ' हा अतिपारंपरिक. या पैकी कोणत्या मार्गावरून पुढचा मराठी सिनेमा मार्गक्रमण करणार यावर आता जी ' चांगल्या ' चित्रपटांची एक लाट आली आहे, ती कितपत टिकते आणि उत्तम सिनेमांचे सुवर्णयुग कधी अवतरते, हे ठरणार आहे.
