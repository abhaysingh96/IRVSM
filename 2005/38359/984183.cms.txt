लेख: एअरमेल
रंग उतरतोय 'लाल बावट्या'चा...
[ Friday, January 07, 2005  10:50:13 pm]


शतकपूतीर्पासून केवळ सहा वर्षं दूर असलेली ' आयबीएम ' ही कोका-कोला आणि मायक्रोसॉफ्ट या कंपन्यांच्या खालोखाल जगातली तिसरी सर्वांत मोठी कंपनी आहे. चीनमधल्या दर चारपैकी एक कम्प्युटर ' लिनोवो ' चा असतो. ' पीसी ' च्या व्यवसायातून ' आयबीएम ' ला फारसं काही हाती लागत नव्हतं. त्यामुळे हा व्यवसाय विकण्याचा निर्णय झाला. ही सुवर्णसंधी सोडेल तो चिनी कसला ? ' लिनोवो ' ने ताबडतोब खरेदी करत ' आयबीएम ' च्या ' थिंक पॅड ' आणि ' थिंक सेंटर ' वर आपलं नाव कोरलंय. अगदी शब्दश: नसली, तरी या उत्पादनांवर आता ' लिनोवो ' ची मालकी असेल. या उत्पादनांवर आता ' आयबीएम मेड बाय चायना ' अशी अक्षरं पाहायला मिळतील.

डेव्हिड आणि गोलिएथ! शेकडो वर्षांपूवीर् त्यांच्यात निर्णायक संघर्ष झाला. अस्तित्वासाठीचा संघर्ष. इस्त्रायल आणि पॅलेस्टिनमधल्या पारंपरिक शत्रुत्वाचं, प्रतिर्स्पध्यांचा खऱ्या अर्थानं खातमा करण्याचं ते प्रतीक ठरले होते.

भांडवलशाही अमेरिका आणि साम्यवादी चीन यांच्यातला संघर्षही असाच पुरातन. या दोन्ही महासत्ता. वेगवेगळ्या धुवांवर उभ्या असलेल्या. जणू आधुनिक डेव्हिड आणि गोलिएथच. जगातली एकेक बाजारपेठ काबीज करत निघालेली अमेरिका आणि स्वत:च्याच कोषात गुरफटून बाहेरचं वारंही घरात येऊ नये, यासाठी दारं बंद करून बसलेला चीन. पण जागतिकीकरणाचा रेटा इतका प्रबळ ठरला की त्याने या बंद दारावर धडकामागून धडका देऊन ते उघडायला भाग पाडलंय.

' बिग ब्ल्यू ' नावानं ओळखल्या जाणाऱ्या ' आयबीएम ' या कम्प्युटर क्षेत्रातल्या दादा कंपनीनं आपला ' पर्सनल कम्प्युटर ' चा व्यवसाय चीनच्या ' लिनोवो ' कंपनीला विकलाय. ' कापोर्रेट जगतातली नित्याचीच घटना ' इतक्या ढोबळपणे या घडामोडीकडे पाहता येणार नाही. आजवर ' जगाची कार्यशाळा ' असलेल्या चीनला आता स्वत:ची ओळख ' जगभरात व्यवसाय करणारा देश ' म्हणून प्रस्थापित करायचीय. त्या दृष्टीने उचललेलं हे सर्वांत मोठं पाऊल म्हणता येईल.

शतकपूतीर्पासून केवळ सहा वर्षं दूर असलेली ' आयबीएम ' ही कोका-कोला आणि मायक्रोसॉफ्ट या कंपन्यांच्या खालोखाल जगातली तिसरी सर्वांत मोठी कंपनी आहे. चीनमधल्या दर चारपैकी एक कम्प्युटर ' लिनोवो ' चा असतो. ' पीसी ' च्या व्यवसायातून ' आयबीएम ' ला फारसं काही हाती लागत नव्हतं. त्यामुळे हा व्यवसाय विकण्याचा निर्णय झाला. ही सुवर्णसंधी सोडेल तो चिनी कसला ? ' लिनोवो ' ने ताबडतोब खरेदी करत ' आयबीएम ' च्या ' थिंक पॅड ' आणि ' थिंक सेंटर ' वर आपलं नाव कोरलंय. अगदी शब्दश: नसली, तरी या उत्पादनांवर आता ' लिनोवो ' ची मालकी असेल. या उत्पादनांवर आता ' आयबीएम मेड बाय चायना ' अशी अक्षरं पाहायला मिळतील.

या व्यवहारानंतर ' लिनोवो ' चीनचा पहिला ' ग्लोबल ब्रॅण्ड ' ठरलाय. इतर चिनी कंपन्यांसाठी तो पथदर्शक ठरणार आहे. ' लिनोवो-आयबीएम ' करार हा आगामी काळाची चुणूक आहे.

परदेशी कंपन्यांचे ब्रॅण्ड, वितरणाचं जाळं, तंत्रज्ञान आणि व्यवस्थापन कौशल्याचं चिनी उत्पादकांना नेहमीच आकर्षण वाटत आलंय. त्यामुळे परदेशी ब्रॅण्ड बरोबर ' टाय अप ' करण्यासाठी चिनी आसुसलेले आहेत. अर्थात त्यामागची कारणंही बरीच आहेत.

अमेरिकेतली ' ब्युरो ऑफ स्टॅटिस्टिक्स ' ही संस्था जगभरातल्या कामगारांना मिळणाऱ्या मजुरीची नोंद ठेवते. तिचा अहवाल कामगारांसंबंधीची धोरणं ठरवण्यासाठी आधारभूत मानला जातो. या संस्थेनं एक उत्कंठावर्धक निरीक्षण नोंदवलंय. गेल्या काही वर्षांपासून चीनमधल्या आकडेवारीला जगभरातून मागणी वाढत असल्याचं त्यांच्या लक्षात आलंय. चीनमधल्या किमान वेतनाची आकडेवारी ही संस्था लवकरच प्रसिद्ध करणार आहे. ती धक्कादायक ठरण्याची शक्यता आहे.

2003 या वर्षात अमेरिकेतल्या कामगाराला दर तासाला सरासरी 21.97 डॉलर वेतन मिळत होतं. हेच प्रमाण ब्रिटनमध्ये 20.37 होतं. मेक्सिकोतल्या कामगारांना सर्वांत कमी म्हणजे 2.48 डॉलर मिळत होते. धक्कादायक बाब म्हणजे साम्यवादी चीनमधल्या कामगाराचं त्यावेळचं वेतन होतं अवघे 64 सेंट!

चीनमध्ये ग्राहकांची संख्या एक अब्ज 30 कोटी इतकी प्रचंड आहे. त्यामुळे अर्थातच स्पर्धाही जीवघेणी आहे. केवळ चिनी बाजारपेठच डोळ्यापुढे ठेवून व्यवसाय करणं पायावर कुऱ्हाड मारून घेण्यासारखं आहे, याची चिनी उद्योगांना खात्री पटू लागलीय.

कमी दरातल्या रोजगारीमुळं चिनी उद्योगांवर अवकळा आली होती. स्वस्तात उत्पादन होतं म्हणून अनेक कंपन्यांनी चीनकडे मोर्चा वळवला. हे चित्र बदलून टाकण्यासाठी चीनने कंबर कसलीय. त्यांना आता ' बिझनेस लीडर ' व्हायचंय. चिनी वस्तू स्वस्त पण निकृष्ट असतात, अशी जगभरात प्रतिमा आहे. ती बदलून ' चिनी वस्तू स्वस्त आणि उत्कृष्टही असतात ', अशी करण्यावर आता भर असेल. चीन ही परदेशी उत्पादनांचे सुटे भाग जुळवणारी कार्यशाळा नाही, तर चिनी अर्थव्यवस्थेवर चिनी उद्योगांचंच नियंत्रण असेल, असे स्पष्ट संकेत ' डेल ' ने दिले आहेत.

परदेशी कंपन्यांबरोबर ' टाय अप ' केल्याने बाजारातला हिस्सा झटकन वाढतो. आपले कम्प्युटर उच्च दर्जाचे आहेत, हे जागतिक बाजारपेठेत बिंबवण्यास ' लिनोवो ' ला कित्येक वर्षं लागली असती. पण ' आयबीएम ' बरोबर नाव जोडलं गेल्याने त्यांचा उद्देश अल्पावधीतच साध्य झाला. इतका की डेल आणि हेवलेट पॅकर्डच्या खालोखाल त्यांचंच नाव घेण्यात येऊ लागलं.

परदेशी कंपन्यांशी हातमिळवणी करण्याचं प्रमाणही वाढू लागलंय. गेल्या वषीर् थॉमसन या फ्रान्समधल्या आघाडीच्या इलेक्ट्रिक आणि इलेक्ट्रॉनिक्स कंपनीने आपला कारखाना टीसीएल या चिनी कंपनीला विकला. अमेरिकी दूरसंचार क्षेत्रात दबदबा निर्माण केलेल्या ' ग्लोबल क्रॉसिंग ' च्या वितरणाचे हक्क ' चायना नेटकॉम ' ने एक अब्ज डॉलर देऊन विकत घेतले. ' शांघाय ऑटोमोटिव्ह ' ने दक्षिण कोरियाच्या ' सांगयाँग ' या मोटार कंपनीचा अर्धा हिस्सा पाचशे कोटी डॉलरना विकत घेतला.

या विलीनीकरणांमुळे चिनी उद्योगांच्या वागणुकीतही आमूलाग्र बदल झाले आहेत. बौद्धिक संपदा हक्क मिळवण्याइतपत जागृतीही त्यांच्यात झालीय. ' जागतिक अर्थव्यवस्थेतल्या आपल्या स्थानाची चीनला जाणीव झाली आहे आणि आता तो ते मिळवण्यासाठी झटत आहे ', वॉशिंग्टनमधले अर्थतज्ज्ञ डॅनियल ग्रिसवोल्ड यांची ही टिप्पणी बरंच काही सांगून जाते.
