महाराष्ट्र
वाघणच्या 'गावपळणी'ची कहाणी
[ Thursday, March 03, 2005  07:29:16 pm]


देवीचा मानकरी कोण, या मुद्यावरून वाघण गावात दोन तट पडले आहेत. त्यापैकी सहा कुटुंबांच्या एका गटाला गावातील उर्वरित लोकांनी नऊ वषेर् बहिष्कृत केले आहे. असे असूनही ही सहा कुटुंबे मागील सर्व ' गावपळण ' मध्ये इतरांसोबत सहभागी झाली होती. यंदा मात्र या सहा कुटुंबांनी व उर्वरित गावकऱ्यांनी दोन स्वतंत्र ' गावपळण ' साजऱ्या केल्या आणि या रुढी-परंपरेतील फोलपणा स्वत:च सिद्ध केला.

आंगणेवाडीची श्रीदेवी भराडी ही आता लाखो लोकांच्या श्रद्धेचे स्थान बनली आहे. ती नवसाला पावते अशा समजुतीने तिच्या जत्रोत्सवाला भक्तांचा महापूर लोटतो. हा जत्रोत्सव गेल्या आठवड्यात झाला. योगायोगाने त्याच्या दुसऱ्याच दिवशी रत्नागिरी जिल्ह्याच्या लांजा तालुक्यातील वाघण येथील ' गावपळण ' ची समाप्ती झाली. या दोन्ही घटनांपैकी पहिली महाश्रद्धेची, तर दुसरी टोकाच्या अंधश्रद्धेची! तशा दोन्हीही बाबी वैज्ञानिक दृष्टिकोनाला तडा देणाऱ्या असल्या, तरी वाघण गावातील ' गावपळण ' ही त्यातील अधिक चिंतेची बाब म्हणावी लागेल. कोकणी माणूस जेवढा म्हणून श्रद्धाळू, तेवढाच तो देव- देवस्की, भुते-खेते, पिशाच्च यामध्येही गुरफटल्याचे अनेकदा दिसून येते. ' गावपळण ' ही प्रथासुद्धा त्याच प्रकारात मोडणारी आहे.

' गावपळण ' म्हणजे ग्रामदेवतेच्या आदेशाचे पालन म्हणून गावातील एकूण एक व्यक्तीने, अगदी नुकत्याच जन्मलेल्या बालकापासून ते म्हाताऱ्या-कोताऱ्यापर्यंत सर्वांनी आपला सारा संसार, गुरे-ढोरे, कुत्री-मांजरी, कोंबड्या यासह काही दिवस गावाच्या सीमेबाहेर जाऊन राहणे. तसे केले नाही तर देवीचा कोप होऊन काहीतरी अघटित घडते, असा समज आहे. कोकणातील अनेक गावांतून अशा ' गावपळण ' होत असतात. त्यात सिंधुदुर्गातील आचरे, चिंदर, वायंगणी गावांचा समावेश आहे. कर्नाटकातसुद्धा काही गावांतून या प्रथेचे पालन केले जाते. वाघण गावात एक सतीदेवीचे मंदिर आहे. त्या देवीचा प्रकोप टाळण्यासाठी तेथील गावकरी दर तीन वर्षांनी ' गावपळण ' साजरी करतात. गावाबाहेर पडण्यापूवीर् गावातील मानकऱ्याकडून वाघजाई देवीचा कौल घेतला जातो. वाघण गावची ' गावपळण ' पूवीर् तीन महिने चाले. कालांतराने तिचा कालावधी कमी कमी होत आता 17 दिवसांवर येऊन ठेपला आहे. या ' गावपळण ' च्या संदर्भात अनेक आख्यायिका ऐकायला मिळतात. मात्र गेल्या नऊ वर्षांपासून ही ' गावपळण ' वादग्रस्त बनली आहे.

देवीचा मानकरी कोण, या मुद्यावरून वाघण गावात दोन तट पडले आहेत. त्यापैकी सहा कुटुंबांच्या एका गटाला गावातील उर्वरित लोकांनी नऊ वषेर् बहिष्कृत केले आहे. असे असूनही ही सहा कुटुंबे मागील सर्व ' गावपळण ' मध्ये इतरांसोबत सहभागी झाली होती. यंदा मात्र या सहा कुटुंबांनी व उर्वरित गावकऱ्यांनी दोन स्वतंत्र ' गावपळण ' साजऱ्या केल्या आणि या रुढी-परंपरेतील फोलपणा स्वत:च सिद्ध केला.

सुरुवातीला वाघण गावाची ' गावपळण ' गेल्या 24 जानेवारीला सुरू होणार होती. नंतर ती 7 फेब्रुवारीपर्यंत पुढे ढकलली गेली. हा निर्णय बहिष्कृत केलेल्या सहा कुटुंबांनी मानला नाही. त्यांनी ठरल्याप्रमाणे 24 जानेवारीलाच गावाबाहेर पडून शेजारच्या पडवण गावी मुक्काम ठोकला. इतरांनी आपली ' गावपळण ' नव्या सुधारित वेळापत्रकानुसार 7 फेब्रुवारीला सुरू केली आणि 10 तारखेला बहिष्कृत सहा कुटुंबे आपली ' गावपळण ' संपवून गावात माघारी फिरली.

या दरम्यान परतलेल्या जवरे कुटुंबातील महेश यशवंत जवरे या शाळकरी मुलाचा अकाली मृत्यू झाला. वास्तविक या प्रसंगाबद्दल संपूर्ण गावाने हळहळ व्यक्त करावयास हवी होती. मात्र त्या काळात ' गावपळण ' पाळणाऱ्या मंडळींनी हा ' देवीचा कोप ' आहे, असा दावा करण्यात धन्यता मानली. विशेष म्हणजे त्यांचा हा दावा जवरे कुटुंबियांनीच फेटाळून लावला. यंदाच्या ' गावपळण ' च्या दरम्यान घडलेली आणखी एक विशेष घटना म्हणजे गावातील राजाराम पांचाळ आणि त्याच्या कुटुंबियांनी कोणत्याच ' गावपळण ' मध्ये सहभागी न होता त्या प्रथेविरुद्ध एकाकी लढा सुरू केला आहे. मागील खेपेच्या ' गावपळण ' च्या काळातही स्वत: पांचाळ यांनी अंधश्रद्धा निर्मूलन समितीच्या कार्यर्कत्यांसमवेत गावातच मुक्काम केला होता. यंदा गावकऱ्यांनी ' अंनिस ' च्या कार्यर्कत्यांना गावात येण्यास मज्जाव केला. या पार्श्वभूमीवर पांचाळ यांनी दाखविलेले धैर्य आणि चालविलेला लढा वाखाणण्याजोगा म्हटला पाहिजे. ' गावपळण ' साठी सांगितली जाणारी ' देवीचा कोप ' अशासारखी कारणे ही सारी थोतांड असल्याने त्याविरुद्ध आपण प्रखर झुंज देऊ, असा पांचाळ यांचा निर्धार आहे. कोकणातल्या ' गावपळण ' पाळणाऱ्या अन्य गावांतूनही पांचाळ यांच्यासारखे लढवय्ये तयार होणे गरजेचे झाले आहे. गावोगावच्या ' गावपळण ' प्रथेमुळे तेथील बायका, मुले, वयोवृद्ध, आजारी माणसे यांचे आतोनात हाल होतात. रानात वास्तव्य करणाऱ्यांना सर्पदंश-विंचूदंश अशा संकटांना सामोरे जावे लागते. या काळात शाळा बंद ठेवली जात असल्याने विद्यार्थ्यांच्या अभ्यासाचे वाटोळे होते. या सर्वांचाही आता गांभीर्याने विचार करावा लागेल.

नवी दिल्ली येथे , 27 फेब्रुवारी ते 2 मार्च या दरम्यान, केंदीय विज्ञान आणि तंत्रज्ञान मंत्रालयातफेर् ' विज्ञान जागरुकता वर्षा ' निमित्त भरविण्यात आलेल्या प्रदर्शनाचे उद्घाटन करताना केंदीय मंत्री कपिल सिब्बल यांनी ' महाराष्ट्रातील विज्ञान जागृतीचे कार्य आणि पद्धती संपूर्ण देशात उल्लेखनीय आहे ,' अशा शब्दात ज्या महाराष्ट्राचा गौरव केला, त्याच महाराष्ट्रात ' गावपळण ' सारख्या अनेक अंधश्रद्धांना लोकमानसात अजूनही थारा मिळतो ; यावरून आपल्याला विज्ञान जागृतीच्या कार्यात अजूनही किती मजल गाठायची आहे, याचा अंदाज येऊ शकतो. नव्या पिढीला हे आव्हान स्वीकारावे लागेल.
