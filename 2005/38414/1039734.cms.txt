सदर
चवीनं खाणार त्याला...
[ Thursday, March 03, 2005  05:34:59 pm]


तसं ते उपनगरच... पण तेव्हाच काळ वेगळाच होता. नीरव शांततेसाठी वेगळ्या ठिकाणी जाण्याची गरज नसायची. निसर्गाचा वरदहस्त मन तृप्त करत असे. धावपळ, दगदग, यापासून कोसभर दूर असणारा तो भाग म्हणजे विलेपालेर्. आतापर्यंत इथल्या भागात सुंदर वाड्या नजरेत पडायच्या, काळाच्या ओघात तिथेही उत्तुंग बिल्डिंग उभ्या राहू लागल्यात. सांस्कृतिक केंद म्हणून प्रसिद्ध असलेलं विलेपालेर् अगदी काही वर्षांपर्यंत धकाधकीपासून मुक्त होतं. नंतर मुंबईच्या बदलत्या रुपाप्रमाणे इथेही बदल घडत गेले. मग गदीर् हा इथला अविभाज्य घटक बनू लागला. तरीही विलेपालेर्चं स्थानमहात्म्य मोठं आहे.

देशभर प्रसिद्ध असलेलं जुहू बीच, दीनानाथ नाट्यगृह, पालेर्श्वर मंदिर, भाईदास हॉल, एअरपोर्ट, शिक्षणसंस्था अशा अनेक वैशिष्ट्यपूर्ण गोष्टी पालेर्करांसाठी भूषण ठरतात. इतक्या गोष्टी इथे असताना खवय्ये कसे काय मागे राहतील नाही का ? उत्तमोत्तम रेस्तराँची संख्या विलेपालेर्त कमी नाही, एवढं तरी नक्की. पूवेर्कडे बऱ्याच वर्षांपासून लोकांच्या पसंतीला उतरलेलं एक रेस्तराँ म्हणजे ' समर हावेर्स्ट '. स्टेशन आणि हायवेपासूनजवळ असणाऱ्या या रेस्तराँत उत्तम पदार्थांचा आस्वाद घेता येईल. इंडियन फूड प्रामुख्याने इथलं वैशिष्ट्य. आता तिथे चायनीज, कोस्टल फूडसाठीही बरेचसे खवय्ये येत असतात. 1989 साली सुरू झालेल्या या रेस्तराँचे मालक मोहन भोगले असून रेस्तराँच्या मॅनेजमेण्टची जबाबदारी दोन वर्षांपासून राहुल लिमये यांच्याकडे खांद्यावर आहे.

' सूप ' ने खाद्ययात्रा सुरू करणाऱ्यांना ' पाया सूप ' चा पर्याय एकदम योग्य मानता येईल. तंदूरमध्ये फिशमधील ' कालीमिरी टिक्का ' पसंतीस येईल. मिरीच्या पेस्टचं कोटिंग केल्यानंतर मॅरिनेट केली जाणाऱ्या डिशचा वेगळा प्रकार म्हणून उल्लेख करावा लागेल. पण समर हावेर्स्टचं नाव जोडलं गेलंय ते ' कोलंबी फ्राय ' शी. स्टार्टर्समध्ये कोलंबी फ्राय सुरूवात उत्तम करणारी असते. हळद, तिखट, लिंबू आणि थोडासा रव्यात परतवून घेतलेली कोलंबी डीप फ्राय केली जाते. चायनीजचा मेन्यू चांगल्याप्रकारे डेव्हलप केला असल्याची माहिती मॅनेजर आकाश शिंदे आवर्जून देतात.

लेग पीसमधील ' चिकन हरी मिर्च ' आवडू शकतं ते त्याच्या चवीने. यात मिरच्यांचा असल्या तरी बिया काढलेल्या असल्याने ही डिश तिखट नसते. ' चिकन चेट्टीनाड ' मूळ पद्धतीपेक्षा थोडी वेगळी बनवली जाते. खूप खोबरं न घालता आणि मुख्य म्हणजे ही डिश फार तिखट बनवली जात नाही. ' स्टफ्ड् पॉम्फ्रेट मसाला ' चांगलीच खुमारी आणल्यावाचून राहत नाही. कोस्टलमध्ये ' सुकं मटण ' समोर आल्यावर ताव मारण्याच्या मोहापासून चुकलात तर बेहत्तर!

फक्त नॉन-व्हेज डिशचं वैशिष्ट्य सांगताना व्हेजला कसं काय मागे सारता येईल ? व्हेजवाल्यांनाही तेवढेच चांगले पर्याय इथे आहेत. व्हाइटिश ब्राऊन गेव्हीतील ' पनीर मुस्तफा ' राजम्यामुळे चांगला वाटतो. ' मिर्च भरवाँ डिश ' ची विशिष्ट चव नक्कीच पसंतीस येईल. ' कढई मशरुम ', ' केसरी पनीर कुर्मा ' सारख्या अनेक डिशेस चांगल्या आहेत.

जवळपासच्या लोकांसाठी ' फॅमिली टिफिन फॉर फोर ' ही पार्सलची पद्धत अतिशय चांगली सुविधा उपलब्ध आहे. व्हेज, नॉन-व्हेज दोन्हीसाठी यांचा वापर केला जातो. व्हेजमध्ये उसळ, भाजी, सोलकढी, कांदाभजी/कोथिंबीर वडी, डाळ, भात, चपात्या, खरवस/गुलाबजाम मिळतात. नॉन-व्हेजमध्ये फिश फ्राय ( सुरमय/बोंबिल), चिकन/मटण सागुती, आमटी, चपात्या, भात, खरवस/गुलाबजाम पदार्थ असतात. चौघांचं पोट चांगल्यापैकी भरू शकतं.

सध्यातर बिर्यानीप्रेमींसाठी मोठी मेजवानी इथे उपलब्ध आहे. ' बिर्यानी हाऊस ' मध्ये ' चिक्कन मख्खनी चॉप बिर्यानी ', ' चिकन शाहजानी पुलाव ', ' कोल्हापुरी गोली पुलाव ', ' दम आलू मटर पुलाव ', ' दम काबुली बिर्यानी ', ' कोल्हापुरी मसाला चावल ' असे एकाहून एक सरस बिर्यानी तुमच्या प्रतिक्षेत आहेत!

हेमंत साटम
