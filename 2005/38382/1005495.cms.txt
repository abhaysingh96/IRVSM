महाराष्ट्र
दशावतार अन् रिमिक्स!
[ Sunday, January 30, 2005  09:47:36 pm]


अमित भंडारी

पु. भा. भावे नाट्यनगरी (डोंबिवली) : मऱ्हाटमोळ्या मातीचा ठसका असलेल्या लावणीपासून रिमिक्सच्या धडाक्यापर्यंत आणि नाट्यसृष्टीतील ज्येष्ठांपासून श्रेष्ठांपर्यंतच्या टिंगलटवाळीच्या मालवणी दशावतारापासून युवा कलाकारांच्या ' जोशफुल ' परफॉर्मन्सनी नाट्यसंमेलनाच्या उद्घाटनाच्या दिवशीच्या उत्तररात्रीत चांगलाच रंग भरला. आपलं नाणं चोखपणे वाजवून दाखवणाऱ्या कलाकारांना रसिकांनी उत्स्फूर्त दाद दिली.

कार्यक्रमपत्रिकेतील यादीत असलेला प्रत्येक कलावंत संमेलनास उपस्थित होता. त्यामुळे त्याचा ' ग्लॅमरस ' टच रात्रीही कायम राहिला. तालमी करून रंगमंचावर उभं राहण्याचे कष्ट चकाचक परफॉर्मन्स करणाऱ्या कलावंतांनी घेतले होते.

' मालवणी दशावतरा ' त नाट्य निर्मात्यांपासून ' स्ट्रगलर ' पर्यंत साऱ्यांना टपली मारली असल्याने रसिकांची हसून हसून पुरेवाट लागली. मोहन वाघ, मोहन जोशीपासून नाट्य परिषदेच्या कार्यर्कत्यांपर्यंतचे टीकास्त्र प्रेक्षकांनी डोक्यावर घेतले. मच्छिंद कांबळींना कोपरखळी बसली नसती तरच नवल! संतोष मयेकर, अतुल तोडणकर व किशोर रावराणे यांच्या परफॉर्मन्सचा ' मस्का-चस्का ' चटकदार होता.

' रिमिक्स ' चा ठेका दुमदुमल्यावर प्रेक्षकांच्या भुवया उंचावल्या खऱ्या ; पण आशिष पवार, श्वेता भोसले, सिद्धार्थ जाधव, शेखर फडके यांनी ' प्रेम-स्पर्श ' संकल्पनेवर ' अश्विनी ये ना ' पासून ' मेरी मधुबाला ' पर्यंतच्या गाण्यांवर केलेल्या परफॉर्मन्सने दाद मिळवली. ' लटका- झटका ' तर प्रेक्षकांच्या लक्षात राहिला, पण त्यापेक्षा ' शॅडो डान्स ' च्या सिक्वेन्सने पसंतीची पावती मिळवली. ' भोले सुरत दिल के खोटे ' वर जुन्या- नव्या नाट्यकलावंतांनी एकत्र परफॉर्म केलं. मधु कांबीकर यांच्या ठसकेबाज लावणीने उत्तररात्रीला अनोखा साज चढविला.

त्या अगोदर ढोलकीवर पडलेल्या कडकडीत थापेला रसिकांनी शिट्ट्यांच्या गजरात दाद दिली. अवधूत गुप्तेच्या गायनाने ' युवा क्राऊड ' ला आकषिर्त केले. त्या सोबत उपस्थित कलाकारांची फौज रंगमंचावर अवतरली आणि साऱ्यांनी अवधूतच्या ठेक्यावर पाय थिरकवण्यास सुरुवात केली. चिमुरड्या विपुल म्हात्रेने डोक्याने कॅसिओ वादन करत ' गर्जा महाराष्ट्र माझा ' वाजवून वन्समोअर मिळवला. लाडक्या ' लक्ष्या ' ला आदरांजली वाहिली ती विजय कदम आणि मनोहर गायकवाडने! अभिनेता सचिन पिळगावकरच्या ' लागा चुनरी में दाग ' ने या कार्यक्रमाची सांगता झाली.

मनोरंजनाच्या कार्यक्रमातील चकचकीतपणा प्रेक्षकांच्या नजरेला भावला. ' हम में है दम ' म्हणत रंगमंचावर अवतरलेल्या कलाकारांच्या पदरी पसंतीचे दान पडले खरे ; पण ' शूटिंग ' साठी तारखा देऊ नका, असे फोन यापूवीर् खणखणले होते. मात्र या ' परफॉर्मन्स ' चे मानधन म्हणून नाट्यपरिषदेतफेर् ' किट बॅग ' देण्यात आली. या कार्यक्रमांचे आयोजन करणाऱ्या आणि त्यांचे समन्वयक म्हणून काम पाहणाऱ्यांनी हात मारल्याची कुजबूज होती.
