लेख: जाता जाता
संक्रांतीचा मेसेज!
[ Thursday, January 13, 2005  07:56:46 pm]


चहासाठी सगळे जमले. पण चकोरी चहा करण्याची काही लक्षणं दिसत नव्हती.

बघता बघता गरमागरम गुळपोळ्या प्लेटमध्ये पडू लागल्या. वर भरपूर तूप. चवीला लोणचं. ज्यु. कंपनी तर उडालीच.

ज्यु. चकोरने तर उठून पहिला तुकडा आईलाच भरवला. चकोरने मुलांकडे बघत ' आता मी काय करू! ' अशा अर्थाची खूण केली.

यावर मुलं हसली.

' आज काय आहे काय! ' चकोरने विचारलं.

' कमाल आहे तुमची! ' असं म्हणत चकोरीने कॅलेंडरमधील तारीख दाखवली.

' अरेच्चा विसरलोच की! ' चकोरने कान पकडले.

' म्हणून तर आई इतकी छान साडी नेसली आहे. ' मुलीने माहिती दिली.

' आधी लक्षात आलं असतं तर.... ' असं म्हणून चकोर थांबला.

' आईसाठी नवीन साडी आणली असती. ' मुलीने वाक्य पूर्ण केलं.

' एखादा दागिना आणला असता. ' मुलाने भर घातली.

' पण आज तिळगूळ, तिळाचे लाडू वगैरे... ' चकोरने विषय बदलला.

' बाबा, इतक्या सही गुळपोळ्या समोर असताना तुम्हाला बाकी सगळं कसं सुचतं हो! ' मुलीने विचारलं.

' अगं, त्या त्या दिवसाचं महत्त्व असतं पदार्थाला. ' चकोरने युक्तिवाद केला.

' पदार्थापेक्षा भावना महत्त्वाची. ' चकोरीने चर्चा गंभीर वळणावर नेली.

' कसली भावना ?' ज्यु. चकोरने तोंडात घास कोंबत विचारणा केली.

' आजच्या दिवसाची! '

' तुला नाही कळणार. आज गोड बोलायचं असतं. ' बहिणीने भावाला चिडवलं.

' मी तुझ्याशी नेहमीच गोड बोलतो. डायबेटिस होईल इतका. ' त्याचं प्रत्युत्तर.

यावर चकोर जोरात हसला.

' बघ, बाबांचा पण तुझ्या बोलण्यावर विश्वास नाही. ' मुलीने साक्ष काढली.

' अरे पण गोड बोलण्याच्या दिवशीच गोड बोलतो की नाही यावर भांडता कशाला! ' चकोरीने विचारलं.

' पण या एक दिवशीच गोड का बोलायचं ?' मुलाने शंका काढली.

' कायमच गोड बोलावं. ' चकोरीने उपदेश केला.

' आपण शहरी, ब्लॉकवाले तर एकमेकाशी बोलायचं टाळतो. तर गोड काय बोलणार! ' चकोरने मुद्दा मांडला.

' निदान या निमित्ताने तरी हातावर तिळगूळ ठेवत गोड बोला असं आपण सांगतो हेच नशीब. ' मुलीने मत मांडलं.

' वाद झाला तरी गोड बोलण्याचं व्रत सोडायचं नाही हे आपण व्यापाऱ्यांकडून शिकलो तरी आपली भरभराट होईल हा आजच्या दिवसाचा मेसेज आहे. ' चकोरने दिनमहात्म्य सांगितलं.

चकोर
