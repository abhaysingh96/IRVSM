सदर
गली परांठेवाली!
[ Thursday, January 13, 2005  07:24:59 pm]


दिल्लीच्या लाल किल्ल्याजवळील गली पराठेंवालीचा स्वाद काही वेगळाच असतो. या गल्लीतल्या दुकानांमध्ये अनेक प्रकारचे पराठे शुध देसी घीमध्ये बनत असतात आणि ते पाहून आपल्या तोंडाला पाणी सुटल्यावाचून राहत नाही. सध्याच्या थंडीच्या दिवसांत ठिकठिकाणहून लोक तिथले पराठे खायला जात असतात. पण ठाणेकरांना इतकं लांब जाण्याची गरज नाही. त्यांच्यासाठी ठाण्यातच गली पराठेवाली अवतरली आहे... जस्ट पराठाजच्या नावानं.

तसं म्हटलं तर ती गल्ली नव्हे, रेस्तराँच आहे. पण तिथंही असंख्य प्रकारचे उत्तर भारतीय आणि पंजाबी ष्टाइलचे पराठे आणि तोंडाला पाणी आणणारे चाट यांचा खजिना तुम्हाला सापडेल.

गंमत म्हणजे चक्क एका मराठी माणसानं पराठे आणि चाट यांची टोकरी ठाणेकरांसाठी आणली आहे. थेट दिल्लीतले गोलगप्पे, तिथली चव, चटणी हा सर्व जामानिमा ' जस्ट पराठाज ' मध्ये आहे. घंटाळी मंदिरासमोरच्या ' जस्ट पराठाज ' मध्ये सर्व खासियत दिल्लीची वा पंजाबी ढंगाची आहे.

पराठा या प्रकारात किती वैविध्य असू शकतं याचा अनुभव इथे येतो. नेहमीच्या ' आलू पराठ्या ' पासून ' मशरूम पराठा ' पर्यंतचा लांबचा लांब मेन्यू इथे आहे. पराठे म्हणजे पोट भरणं हा सरळ हिशेब. पण तो फस्त करणं सर्वांना शक्य नसतं. म्हणूनच ' जस्ट पराठाज ' चे व्ही.एल. जोशी यांनी पराठा सिंगल व डबल लेयरमध्ये मिळण्याची व्यवस्था केली आहे. तरीही सिंगल लेअरमधला पराठा फार लहान असतो असं नव्हे. छोले, राजमा, अख्खे मसूर, रायता, लोणचं यांच्या सोबतीने पराठ्यावर ताव मारणं कोणालाही आवडेल.

' आलू विथ बेबी कॉर्न पराठा ', ' आलू-ग्रीन पीज पराठा ', ' आलू विथ-पराठा ', ' टोमेटो-पनीर पराठा ', ' गोबी-पुदिना पराठा ', ' गोबी- स्प्रिंग ओनिअन पराठा ', ' मेथी-चीज पराठा ', ' ग्रीन पीस-स्प्रिंग ओनिअन पराठा ' असे पराठ्यांचे कित्येक प्रकार आहेत. ' पापडी-ओनिअन-टोमेटो पराठा ', ' चीज-गालिर्क पराठा ' असे वेगळे पराठेही आहेत. ' दिल्ली दा जान ' मधली ' आलू टिक्की ' ( पॅटिस) निश्चितच आवडेल. त्यासह ' पापडी पुरी ' ची चव घ्यायलाही विसरू नका. शेवपुरीच्या पुरीवरची ही करामत मजा आणते. अस्सल भटुरे काय असतात, हे पाहण्यासाठी ' जस्ट पराठाज ' मध्येच जावं लागेल. पण पदार्थांची ही यादी इथेच संपत नाही. बटाट्याच्या (सळ्यांच्या) बास्केटमध्ये गोड चटणी, आंबट चटणी, दही, साखर, चणाडाळ, कांदा आदींचं मिश्ाण काय चव आणतं, त्याचा अनुभव केवळ ' जस्ट पराठाज ' मध्येच घ्यावा!
