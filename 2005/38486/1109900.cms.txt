महामुंबई
...आता 'दिनूकाका'
[ Saturday, May 14, 2005  01:08:34 am]


कल्पना राणे

मुंबई : मे महिन्याच्या ' हॉट ' उन्हाळ्यात शीतपेयांचे ' कोल्ड ' वॉर आता भलतेच रंगात आले आहे. शाहरूख खान आणि आमीर खान या खानद्वयीने दोन ' दादा ' शीतपेयांना ' एन्डॉर्स ' करण्याचा उद्योग सुरू केल्यापासून जाहिरात विश्वामध्ये ' मन्नो भाभी ' आणि ' बबली ' चीच चर्चा आहे. ' बबली ' च्या भडीमारापुढे ' मन्नोभाभी ' स्क्रिनवर कमी वेळा दिसूनही तिचा इम्पॅक्ट मात्र बबलीला ट्रबलमध्ये टाकणाराच ठरतो आहे.

मन्नोभाभी अन् एनआरआय अशा दोन रुपात येऊन आमीर खानने अशी धम्माल उडवून दिलीय की, स्वयंपाकघरात शीतपेय असणे ' मस्ट ' असल्याचे गृहिणींना वाटू लागले आहे. या जाहिरातीच्या प्रसारणानंतर गेल्या काही महिन्यात या शीतपेयाचे माकेर्ट एकदम वधारल्याची माहिती कोला जायंट्च्या माकेर्टिंग विभागाकडून मिळाली आहे. मन्नोभाभीच्या यशस्वी आगमनानंतर आता तिच्याकडे अनेक वषेर् घरकाम करणारा ' दिनूकाका ' आपल्या भेटीला येतोय... आणि हा दिनूकाका आहे अर्थातच आमीर खान! घरातल्या सगळ्या व्यक्तिरेखांमध्ये खास जादू असते, याचं महत्त्व जाहिरातीने वेळोवेळी पटवून दिले आहेच. आईबाबा, भाऊबहिणी या नात्यांपेक्षा थोड्या वेगळ्या वाटेने जाण्याची जाहिरातवाल्यांची जुनी खोड. आता वहिनीचे किंवा खूप वर्षांपासून घरात काम करणारा नोकराचे नाते एक्सप्लोअर करायचे जाहिरातवाल्यांनी मनावर घेतले आहे.

वर्षानुवर्षे घरकाम करणारा नोकर घरातला सदस्यच बनतो आणि त्याच्या शब्दाला तेवढाच मान असतो. अशा मजीर्तल्या ' दिनूकाका ' ला आणून मेकॅन एरिक्सनच्या प्रसून जोशीने पुन्हा बाजी मारली आहे. शाहरूखला ' लेडी ' लक फेवरेबल असतेच ; पण त्याच्या तुलनेत ' बुद्धीमान ' अभिनेता म्हणून नेहमीच बाजी मारणाऱ्या आमीरने इथेही शाहरूखपेक्षा काकणभर सरसच कामगिरी करून दाखवली आहे. आजच संध्याकाळपासून आमीरचा ' दिनूकाका ' छोट्या पडद्यामार्फत तुमच्या घरात प्रवेश करायला सज्ज झाला आहे!
