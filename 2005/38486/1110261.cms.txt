संवाद
पुणे तिथे रस्ते उणे
[ Saturday, May 14, 2005  06:14:10 pm]


.. कोणाचा कोणाला मेळ नसला की, फज्जा कसा उडतो, याचं तंतोतंत उदाहरण पुण्याच्या वाहतूक समस्येतून मिळतं. सध्याची परिस्थिती बघता आगामी पाच वर्षांतल्या वाहुतकीचं चित्र डोळ्यासमोर उभं करायचं म्हणलं, तर नको वाटतं. म्हणूनच, समस्येचं उत्तर शोधून त्याच्या निराकरणासाठी वेळीच प्रयत्न केले नाहीत, तर ' इंच इंच लढवू ' असं वाहनचालकांना म्हणावं लागेल..!

सांस्कृतिक राजधानी एवढ्या विशेषणानंच गेल्या कित्येक वर्षांपासून ओळखल्या जाणाऱ्या पुण्यात झपाट्याने बदल होत आहेत. ' एक्स्प्रेस वे ' मुळे अवघ्या अडीच तासांमध्ये राज्याच्या राजधानीत पोचता येणं शक्य झाल्यानं पुण्यातला काहीशा संथ जीवनालाही गती आली आहे. विशेषत: ' आयटी ' क्षेत्रातल्या मोठमोठ्या कंपन्या पुण्याच्या परिसरात एकटवल्यापासून तासांचा हिशेब मांडणाऱ्या पुणेकरांना ' मिनिट ' काट्याचंही महत्त्व समजू लागलंय. ' एक्स्प्रेस वे ' नं पुण्यात लवकर पोचूनही औंध ते हडपसर हे परस्परविरोधी दिशांना असणाऱ्या उपनगरातलं अंतर पार करायचं म्हणलं तर ' मुंबई-पुणे ' प्रवासाच्या निम्मा वेळ खर्च होतो. म्हणूनच, पुण्यातल्या वाहतूक व्यवस्थेत सुधारणा घडवून आणण्यासाठी आमूलाग्र बदलांची आवश्यकता आहे. हे बदल कागदावर अनेकदा मांडून झालेत, त्यावर चर्चाही झडल्या ; पण त्याची अंमलबजावणी अजून ' कोसो ' दूर आहे. पुण्याचा सध्या होणारा विस्तार पाहता आगामी पाच वर्षांचा विचार करून कोट्यवधी रुपयांच्या योजनांना वेळीच चालना देण्याची गरज आहे, अन्यथा रस्त्यांबाबत नियोजनाचं ' भिजत पडलेलं घोंगडं ' असंच त्याचं वर्णन करावं लागेल.

काही मोजक्या पेठा आणि पूवेर्ला कँप, दक्षिणेला पर्वती, पश्चिमेला चतु:श्ाृंगी आणि उत्तरेला भांबुर्डा अर्थात शिवाजीनगर अशा चार दिशांच्या खुणा सांगणारी वेस एवढ्यापुरतंच पुणं मर्यादित होतं. मात्र, कालचक्राच्या ओघात पानशेतच्या पुराचं निमित्त घडलं अन् शहर फोफावलं. त्यामुळे वाहतुकीचा चेहरामोहरा बदलायला सुरुवात झाली. मात्र, त्यानुसार आवश्यक असणाऱ्या सोयी- सुविधांमध्ये फार मोठा बदल झाल्याचं चित्र दिसत नाही. गेल्या 10 वर्षांमध्ये ' एक्स्प्रेस वे ' चा अपवाद वगळता दखल घेण्याजोगं काम शहराच्या क्षितीजावर झालेलं नाही.

सध्या पुण्यातली वाहतूक सुधारण्यासाठी काही ठिकाणी कामं सुरू असल्याचं निदर्शनास येतं. मात्र, त्यातील कित्येक कामं काही ना काही कारणानं रखडलीचं आहेत. पुणे-सोलापूर रस्त्यावर सासवड फाट्यानजीक होणारी वाहतूक कोंडी सोडवण्याकरता सुरू असलेलं उड्डाणपुलाचं काम अद्याप पूर्ण झालेलं नाही. (गेल्या सरकारमधल्या सार्वजनिक बांधकाममंत्र्यांनी त्यासाठी अपार कष्ट घेतले. मात्र, नव्या राज्य सरकारला वर्ष पूर्ण होतं आलं, तरी उड्डाणपुलाची परिस्थिती ' जैसे थे ' चं आहे.) त्याच्या कामासाठी रस्त्यावर उभारण्यात आलेल्या बॅरिकेड्समुळे वाहतुकीच्या खोळंब्यात भरच पडत चालली आहे.

हडपसरप्रमाणे विद्यापीठ रस्त्यावरील वाहतुकीचा ताण कमी व्हावा, यासाठीदेखील उड्डाणपुलाच्या कामाचा मुहूर्ताचा नारळ फोडला गेला ; पण त्यानंतर कामात कोणतीच प्रगती नाही. उलट, रोजच्या रोज या रस्त्याने प्रवास करणाऱ्यांच्या समस्येत वाढ झाली आहे. राज्य सरकार, महापालिका आणि महाराष्ट्र राज्य रस्ते विकास महामंडळ (एमएसआरडीसी) यांच्या एकत्रित प्रयत्नांमधून मुंबईतल्या ' फ्लायओव्हर्स ' ची संख्या चढत्या क्रमाने वाढत असताना मुंबईचं जुळं शहर म्हणून ओळखल्या जाणाऱ्या पुण्यात अजून पाच फ्लायओव्हरही नसावेत, ही खेदाची गोष्ट आहे. माजी पंतप्रधान अटलबिहारी वाजपेयी यांच्या संकल्पनेतून सुरू झालेल्या ' सुवर्ण चतुष्कोन ' योजनेचा काही अंशी फायदा होऊन पुणे-सातारा रस्त्याचं रुपडं पालटलंय. हडपसरच्या पुढे लोणीपासून ' पुणे-सोलापूर ' हमरस्ताच्या काही भागही चौपदरीकरणानं नटलायं. मात्र, त्याच धतीर्वर पुण्याशी जोडल्या गेलेल्या इतर महामार्गांचं काम हाती घेण्याचं दाखवलेलं आमिष ' दिवास्वप्न ' ठरण्याचीच भीती आहे.

या सगळ्या ' निगेटिव्ह ' गोष्टींमध्ये काही चांगल्या गोष्टी मात्र नक्कीच आगामी काळात अस्तित्त्वात येऊ शकतील. त्यामध्ये प्रामुख्याने उल्लेख करायला हवा तो, या वर्षअखेरीपर्यंत सुरू होणाऱ्या ' कात्रज घाटाला ' पर्याय ठरू शकणाऱ्या रस्त्याचा. मुंबईहून बंगलोरला जाऊ इच्छिणाऱ्या सर्वांसाठी वडगावपासून थेट कात्रज घाट संपल्यानंतर लागणाऱ्या शिंदेवाडीपर्यंत जाण्याची सोय उपलब्ध होणार आहे. त्यामुळे प्रवासाच्या वेळेत तर बचत होणार आहेच ; त्याशिवाय कात्रजच्या परिसरात सध्या होणाऱ्या वाहतूक कोंडीतूनही वाहनचालकांची सुटका होऊ शकेल. दीड किलोमीटर लांबीचे दोन मोठे बोगदे आणि 35 मीटर उंचीचा पूल ही या रस्त्यावरील प्रवासाची वैशिष्ट्यं ठरतील. खंबाटकी घाटात काही वर्षांपूवीर् सुरू झालेल्या बोगद्यामुढे जशी वाहतुकीला गती प्राप्त झाली, त्याचप्रमाणे या ' कात्रज बायपास ' चा मोठा फायदा होऊ शकेल.

पुण्याच्या वाहुतकीत सुधारणेसाठी पालिका दरवषीर्च्या अंदाजपत्रकांमध्ये कोट्यवधींची तरतूद करते ; पण त्या खर्चाला फुटणारे ' फाटे ' च जास्त असतात. (एकात्मिक रस्ते विकास योजनेसाठी पालिकेच्या अंदाजपत्रकात 350 कोटींची तरतूद आहे. मात्र, त्याचा विनियोग कसा होणार, याचा पत्ता नगरसेवकांनाही नसावा.) त्यामुळे पाहिजे तिथे पैसा खर्च केला जात नाहीच. मुंबईत ' पायलीला 50' फ्लायओव्हर असताना, शहरात फ्लायओव्हर बांधायला जागा कमी आहे, अशी ओरड किती दिवस केली जाणार आहे ? एका बाजूला ' एमएसआरडीसी ' तफेर् शहरात सुरू असणारी काम अगदी नगण्य असताना, शहरात येणाऱ्या सर्व राज्य महामार्गांवर ' टोल ' आकारण्याचा त्यांचा प्रस्ताव म्हणजे ' रोगापेक्षा इलाज भयंकर ' असंच म्हणायला हवं!

कोणाचा कोणाला मेळ नसला की, फज्जा कसा उडतो, याचं तंतोतंत उदाहरण पुण्याच्या वाहतूक समस्येतून मिळतं. सध्याची परिस्थिती बघता आगामी पाच वर्षांतल्या वाहुतकीचं चित्र डोळ्यासमोर उभं करायचं म्हणलं, तर नको वाटतं. म्हणूनच, समस्येचं उत्तर शोधून त्याच्या निराकरणासाठी वेळीच प्रयत्न केले नाहीत, तर ' इंच इंच लढवू ' असं वाहनचालकांना म्हणावं लागेल!
