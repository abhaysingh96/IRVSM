सामना समकालीन वास्तवाचा
[ Saturday, January 15, 2005  05:09:47 pm]


' हॅम्बर्ग सेल ', ' अमू ', ' ब्लॅक फ्रायडे ', ' यस्टडेर् ', ' बियाँड ऑनर ', ' हॅनेले ', ' द हॉर्समन ', ' बो बरॅक्स फॉर एव्हर ', ' सेंट्रल स्टेशन ',' प्रायव्हेट ', ' मानसरोवर ', ' मॉनिर्ंग रागा '... नुकत्याच संपलेल्या ' मामि ' चित्रपट महोत्सवातील चांगल्या चित्रपटांची नावंच द्यायची झाली तर किमान तीस-पस्तीस चित्रपटांचा उल्लेख करावा लागेल. कुठे समकालीन वास्तवाला भिडण्याची धिटाई,

तर कुठे वेदनेचा आत्यंतिक संयत आविष्कार. चित्रपटनिमिर्तीच्या अगदी परमोच्च कसोट्यांवर हे सगळेच्या सगळे चित्रपट पूर्णत: कदाचित उतरणार नाहीत. पण तरीही ' चांगलं ' म्हणावं असं खूप काही त्यात सापडतं. महोत्सव संपला तरी हे चित्रपट मनात रुंजी घालत राहतात, भावसंपन्न करून जातात...

अमेरिकेत आईसोबत राहणारी तरुण अमू सुटीत भारतभेटीवर येते आणि अनपेक्षितपणे तिचे जन्मरहस्य उकलण्याची ओढ तिला लागते. कबीर या नव्या मित्राच्या साह्याने त्या जन्मरहस्याचे धागेदोरे ती उलगडू लागते, तर ते धागेदोरे चक्क 1984 च्या, इंदिरा गांधींच्या हत्येनंतर दिल्लीत झालेल्या शिखांच्या शिरकाणापर्यंत पोचतात! अमू खरी कोण असते ?

दि. 12 मार्च 1993 रोजी मुंबईत बारा ठिकाणी बॉम्बस्फोट झाले आणि हजारो निरपराध जिवांचे बळी गेले. त्यामागचा कट कसा, कुठे शिजला, याचा वेध एस. हुसैन झैदी यांना कादंबरीतून घ्यावासा वाटला तर ' हत्या ', ' शूल ' लिहिणारा तरुण पटकथालेखक अनुराग कश्यप याला त्यावर ' ब्लॅक फ्रायडे ' चित्रपट लिहून, दिग्दशिर्तही करावासा वाटला...

दि. 11 सप्टेंबर 2001 रोजी अमेरिकेतील ट्विन टॉवर्सवर हवाई हल्ला झाला, पुन्हा हजारो निरपराध्यांचे बळी गेले. इंग्लंडमधील तरुण दिग्दशिर्का अँटोनिआ बर्ड हिला त्या हत्याकांडात गुंतलेल्या दहशतवाद्यांच्या मानसिकतेचा अभ्यासपूर्ण वेध तटस्थपणे घ्यावासा वाटला आणि ' हॅम्बर्ग सेल ' हा अत्यंत संयत चित्रपट निर्माण झाला...

दि. 7 ते 13 जानेवारीपर्यंत मुंबईत झालेल्या ' मामि ' ( मुंबई अॅकॅडेमी ऑफ मूव्हिंग इमेजेस) महोत्सवातील चित्रपट पाहताना, ऐतिहासिक मूल्य असलेल्या समकालीन वास्तवाला धिटाईने भिडण्याची तरुण दिग्दर्शकांची धडपड जाणवत राहिली. त्यापैकी अमेरिकास्थित शोनाली बोस दिग्दशिर्त ' अमू ' या चित्रपटाला तर फीप्रेस्की (आंतरराष्ट्रीय चित्रपट समीक्षक मंडळ) पुरस्कार मिळाला. ' हॅम्बर्ग सेल ' मधील कलात्मक तटस्थता न जमलेले ' अमू ' व ' ब्लॅक फ्रायडे ' हे चित्रपट समीक्षकांच्या दृष्टीने वादग्रस्त ठरले. 1984 च्या शीख हत्याकांडाशी दूरान्वयानेही संबंध नसलेल्या आणि ज्या संबंधातील न्यायालयीन चौकशी त्वरेने सुरू आहे, त्या गोध्रा हत्याकांडाशी त्याचा संबंध ओढूनताणून जोडून ' अमू ' ने बहुसंख्याक हिंदंूना नाहक दोषी ठरवले आहे. ' ब्लॅक फ्रायडे ' मध्ये मृतदेहांचा बाजार मांडण्यात आला आहे, असा आक्षेप घेण्यात आला आणि ' हॅम्बर्ग सेल ' च्या संयत हाताळणीच्या तुलनेने तो कोणालाही पटावा! पण तरी ताज्या विषयाला हात घालण्याची या तरुण दिग्दर्शकांची धडपड दाद देण्याजोगी निश्चितच वाटली...

पण भडक हाताळणी हे आपल्या बॉलिवुडच्या चित्रपटांचे वैशिष्ट्यच आहे! एड्स ही आजच्या काळातील प्रमुख समस्या. स्पर्धात्मक भारतीय चित्रपटांपैकी रेवती दिग्दशिर्त, शिल्पा शेट्टी, सलमान खान, अभिषेक बच्चन यांच्या प्रमुख भूमिका असलेला ' फिर मिलेंगे ' हा एड्सग्रस्तांना मिळणाऱ्या वागणुकीविषयीचा ' ग्लॉसी ' हिंदी चित्रपट बघताना , ' यस्टडेर् ' हा दक्षिण आफ्रिकेचा, डरेल जेम्स रूड दिग्दशिर्त याच विषयावरील चित्रपट तुलनेने आठवत राहिला. कुठलंही ग्लॅमर न वापरता, शहरात खाणकामगार असलेल्या नवऱ्याकडून मिळालेला एड्सचा शाप त्याची नायिका यस्टडेर् आपल्या मुलीसाठी कशी धीरोदात्तपणे साजरा करते ; ती ज्या रुग्णालयात उपचारांसाठी दोन/ तीन तास रस्ता तुडवून जाते, तो लांबलचक रस्ता तिच्या कंटाळवाण्या, निर्मम आयुष्याचे प्रतीक कसा बनतो, हे त्या चित्रपटात कुठलाही आकांत न करता, वास्तवाला जवळ जाणारी रंगभूषा एड्सग्रस्तांना करून इतक्या संयतपणे सांगितले होते! ' यस्टडेर् ' सारखे चित्रपट आमच्या दिग्दर्शकांनी आवर्जून बघितले पाहिजेत, असे ' फिर मिलेंगे ' बघताना सारखे वाटत राहिले...

एकविसाव्या शतकात प्रवेश केला तरी, जगभरच्या पुरुषप्रधान समाजाची मानसिकता अजिबात बदललेली नाही. स्त्रीला स्वत:च्या आयुष्याबाबतचा कोणताही निर्णय घेण्याचे वैचारिक स्वातंत्र्य ही मानसिकता नाकारते आणि ' स्त्रीजन्मा, ही तुझी कहाणी ' म्हणून रडगाणे गाण्याची पाळी आपल्यावर येत राहते. मुलगी बहकली आणि वाममार्गाला लागली, म्हणून तिला वाऱ्यावर सोडून तिच्या नावे घटश्ााद्ध घालण्याचा प्रकार काय केवळ आपल्या दक्षिण भारतातच घडत नाही ; झेकोस्लोव्हाकियाच्या ज्यूंमध्ये आणि क्रोशियाच्या अरबांतही बहकलेल्या मुलींना मृत्युदंड देण्याची प्रथा होतीच, हे करेल कचन्याचा ' हॅनेले ' व ब्रॅन्को इव्हान्डाचा ' द हॉर्समन ' बघताना मनावर ठसलेच! पण तो अमानुषपणा आजच्या अमेरिकेत डॉक्टरी शिकणाऱ्या, रासवट इजिप्शियन बापाच्या पोटी जन्माला येण्याचे दुदैर्व वाट्याला आलेल्या नायिकेच्या नशिबीही कसा येतोे, याची सुन्न करणारी कहाणी ' बियाँड ऑनर ' या चित्रपटात वरुण खन्ना या मुंबईतील झेवियर्स कॅालेजच्या माजी विद्यार्थ्याने सांगितली. आपली मुलगी तिच्या अमेरिकन सहकारी डॉक्टरासोबत शरीरसोबत करून आपला धर्म बुडवते आहे, अशी धारणा झालेला तो नालायक बाप त्यांच्या धर्मगुरूंना बोलावून स्वत:च्या निदिस्त मुलीवर, तिची कामवासना मरावी, म्हणून तिचे गुह्यांगच कापून टाकण्याची अघोरी शस्त्रक्रिया ' क्रूडली ' करवतो! आता अमेरिकेत स्थायिक झालेल्या वरुणने केलेला या चित्रपटाच्या नायिकेचा दु:खान्त दाखवणारा फिल्मी शेवट प्रेक्षकांना अजिबात पटला नाही. पण त्या नायिकेचे आणि रोज रात्री नवऱ्याकडून अमानुष अनैसगिर्क संभोगाच्या बलात्काराला सामोरे जावे लागणाऱ्या तिच्या भेदरट/गोऱ्या आईचे दु:ख प्रेक्षकांचे काळीज पिळवटून गेले...

विदेशी चित्रपट महोत्सवांतील ' अन्सेन्सॉर्ड ' चित्रपटांतील सेक्स यात नावीन्य काहीच नाही. पण यावेळच्या चित्रपटांत हे सेक्सचे चित्रण कित्येकदा अगदी अनावश्यक व किळसवाणे होते. ' डॉटर्स ऑफ द विंड ' या दोन बहिणींमधील मानसिक तणावाचे छान चित्रण असलेल्या ब्राझिलियन चित्रपटात तर संभोगदृश्यांची एक मोठी मालिका का दाखवली गेली, हा प्रश्ान्च प्रेक्षकांना पडला. अगदी मुखमैथुन / गुदमैथुनादि अनैसगिर्क संभोगापासून कंडोम कसा वापरायचा, हे नवख्या मित्राला प्रात्यक्षिकाने शिकवणाऱ्या ' बोअर्ड इन ब्रुनो ' या चित्रपटापर्यंत सेक्स बटबटीतपणे विदेशी चित्रपटांतून तर डोळ्यांवर आदळतच राहिला. पण कोलकात्याच्या अंजन दत्तने लिलियट दुबे, व्हिक्टर बॅनजीर्, गायिका उषा उथप आदींना घेऊन चाळकऱ्यांच्या जीवनावर बनवलेल्या ' बो बरॅक्स फॉर एव्हर ' या सुरेख इंग्रजी चित्रपटातील नेहा दुबेवर चित्रित प्रदीर्घ शृंगारदृश्य खटकलंच. अपूच्या मधुचंदाच्या रात्रीनंतर सकाळी त्याच्या नवपरिणित सलज्ज पत्नीच्या कपाळावरील फरफटलेल्या कंुकवातून त्यांच्या श्ाृंगाराचे संयत सूचन करणाऱ्या सत्यजित रायच्या कोलकात्यातील तरुण दिग्दर्शकाला देखील नग्नतेचा मोह टळू नये ना ? पण महोत्सवाला प्रचंड गदीर्ने येणारे आंबटशौकीन प्रेक्षक ' त्या ' दृश्यांसाठीच येतात, या सर्रास होणाऱ्या आरोपात अगदीच तथ्य नाही, असेही म्हणता येणार नाही, कारण अॅमेझॉनच्या नयनरम्य जंगलाच्या पार्श्वभूमीवर पशुपक्ष्यांची शिकार करणाऱ्यांपासून त्यांना वाचवण्यासाठी धडपडणाऱ्या छोट्या ' तायना ' या रेड इंडियन मुलीचा सहजसुंदर अभिनय असलेल्या, पर्यावरण प्रेमाचा संस्कार करणाऱ्या

ब्राझिलियन बालचित्रपटाचा निर्मळ बाज लक्षात येताच अनेक प्रेक्षकांनी प्रेक्षागारत्याग केलाच! असो!

जीवनसातत्याचे भारतीय तत्त्वज्ञान सांगणारे ' काऊ ' व ' जम्पिंग ओव्हर द पडल्स ', आजोबा/ नातीमधले उलगडत गेलेले ऋजू भावबंध चितारणारा ' बटरफ्लाय ', खाणकामगारांचे पहिले आंदोलन टिपणारा ' सबटेरा ', निवृत्तीनंतर रेल्वेस्थानकावर पत्रलेखनाचे काम करणाऱ्या एका वृद्धेचे एका अनाथ मुलाशी जुळत गेलेले भावबंध दाखवणारा ' सेंट्रल स्टेशन ',' प्रायव्हेट ', ' मानसरोवर ', ' मॉनिर्ंग रागा ' असे अनेक चांगले चित्रपट या सातव्या ' मामि ' महोत्सवात बघायला मिळाले ; मनात रुंजी घालत राहून भावसंपन्न करून गेले. आपला अतुल कुलकणीर् सवोर्त्कृष्ट अभिनेता म्हणून परीक्षकांचे खास पारितोषिक ' देवराई ', ' मानसरोवर ' मधील भूमिकांबद्दल पटकावून गेला, ही या महोत्सवाची उपलब्धी म्हणावी लागेल...
