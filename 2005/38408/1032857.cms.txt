अर्थव्यवहार
अनाहूत 'टेलिमाकेर्टिंग कॉल्स'वर 'हच'चा तोडगा
[ Friday, February 25, 2005  09:24:51 pm]


वृत्तसंस्था

नवी दिल्ली : मोबाइल फोनधारकांना होत असलेला अनाहूत ' टेलिमाकेर्टिंग ' वाल्यांचा उपदव थांबविण्यासाठी सवोर्च्च न्यायालयाने दिलेल्या आदेशापाठोपाठ ' हच ' या मोबाइल फोनसेवा पुरविणाऱ्या कंपनीने आपल्या ग्राहकांना एक पर्याय शुक्रवारी उपलब्ध करून दिला. आता ' हच ' च्या ग्राहकांनी त्यांना जे ' टेलिमाकेर्टिंग कॉल्स ' नको आहेत, त्यांची यादी ' एसएमएस ' द्वारे कंपनीला कळवावयाची आहे. संदेशात ह्यह्लश्ाश्च हा शब्द व त्यानंतर अनाहूत ' कॉलर ' चा क्रमांक टाइप करून तो संदेश 123 क्रमांकावर पाठवावयाचा आहे. असा ' एसएमएस ' संदेश ' हच ' ला मिळाल्यानंतर कंपनी त्या ग्राहकाला नको असलेल्या कॉल्सचे (' अनसॉलिसिटेड कॉल्स ') प्रकरण ज्या टेलिमाकेर्टिंग कंपनीकडून ते पाठविले जातात तिच्याकडे नेईल. त्यानंतर ग्राहकाला अशा अनाहूत कॉल्स घ्यावे लगणार नाहीत, असे कंपनीने प्रसृत केलेल्या पत्रकात म्हटले आहे.
