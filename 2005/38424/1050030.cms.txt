महामुंबई
वाचकांनी दिलेला सन्मान धन्य करणारा!
[ Sunday, March 13, 2005  02:03:28 am]


म. टा. प्रतिनिधी

मुंबई : ' आतापर्यंत सरकारकडून तसेच विविध संस्थांकडून अनेक पुरस्कार मिळाले, अनेकांनी गौरव केला. पण या पुरस्काराचं मोल फार मोठं आहे. ' महाराष्ट्र टाइम्स ' हा महाराष्ट्राचा मुकुटमणी आहे. अशा वृत्तपत्राच्या वाचकांनी दिलेला हा सन्मान स्वीकारण्यात धन्यता वाटते, असे हृद्य उद्गार ' महाराष्ट्रभूषण ' शाहीर साबळे यांनी ' राम बंधु म. टा. सन्मान ' सोहळ्यात शनिवारी काढले.

' रवींद ' मध्ये ' राम बंधु म. टा. सन्मान ' चा दिमाखदार सोहळा सुरू झाला तेव्हा लखलखती सन्मानचिन्हं कोणाच्या हाती विसावणार, याची उत्कंठा उपस्थितांना होतीच, त्यातही सर्वाधिक आकर्षण होतं ते ' महाराष्ट्रभूषण ' आणि ' महाराष्ट्राची शान ' कोण होणार याचं. हे अत्यंत प्रतिष्ठेचे पुरस्कार अनुक्रमे शाहीर साबळे आणि आशुतोष गोवारीकर यांना जाहीर झाले आणि अवघे प्रेक्षागृह टाळ्यांच्या कडकडाटानं दुमदुमून गेलं. आशुतोष गोवारीकर यांना ' महाराष्ट्राची शान ' देऊन गौरवलं ते ज्येष्ठ दिग्दर्शक जब्बार पटेल यांनी, तर मुख्यमंत्री विलासराव देशमुख यांनी शाहीर साबळे यांचा ' महाराष्ट्र भूषण ' नं गौरव केला.

मी सातत्याने या सोहळ्यासाठी येतो आहे. यंदा या निमित्तानं शाहीर साबळेंसारख्या वयाच्या 82 व्या वषीर्ही तजेलदार असलेल्या व्यक्तित्वाचा गौरव करण्याची संधी मिळाली, अशा शब्दांत मुख्यमंत्र्यांनी शाहीरांच्या कलेला दाद दिली. पुरस्कार स्वीकारल्यानंतर शाहीरांनी ' महाराष्ट्र गीत ' नेहमीच्या जोशात सादर केलं.

हा पुरस्कार म्हणजे देश-विदेशातील मराठी माणसाने दिलेली शाबासकी आहे, असे आशुतोष गोवारीकर यांनी ' महाराष्ट्राची शान ' स्वीकारताना नमूद केलं. हा पुरस्कार कधी तरी मिळेल, असा मला विश्वास होता. मात्र तो एवढ्या लवकर मिळेल, असं वाटलं नव्हतं, अशीही पुस्ती त्यांनी जोडली.

' राम बंधु म. टा. सन्मान 2005' च्या नाटक विभागात लोकशाहीर विठ्ठल उमप थिएटरच्या ' जांभुळ आख्यान ' नं सवोर्त्कृष्ट नाटक, सवोर्त्कृष्ट दिग्दर्शक ( अजित भगत), सवोर्त्कृष्ट अभिनेता ( विठ्ठल उमप) असे तीन विभागात पुरस्कार मिळाले. तर चित्रपट विभागात ' सा ' व ' का '. स. वाणी ट्रस्ट ' च्या ' देवराई ' नं सवोर्त्कृष्ट चित्रपट, सवोर्त्कृष्ट दिग्दर्शक (सुमित्रा भावे व सुनील सुकथनकर), सवोर्त्कृष्ट अभिनेता (अतुल कुलकणीर्), सवोर्त्कृष्ट संवाद (सुमित्रा भावे), सवोर्त्कृष्ट छायाचित्रण (देबू देवधर) असे पाच पुरस्कार पटकावून स्पधेर्त आघाडी घेतली. अत्यंत चुरशीच्या लढतीच्या टीव्ही मालिका विभागात मात्र निविर्वाद यश कोणत्याही एका मालिकेला मिळालं नाही. सवोर्त्कृष्ट मालिका ठरलेल्या ' भाग्यविधाता ' ला सवोर्त्कृष्ट सहाय्यक अभिनेत्री (सुचित्रा बांदेकर) आणि सवोर्त्कृष्ट शीर्षकगीत असे एकूण तीन पुरस्कार मिळाले. तर ' सोनपावलं ' ला दोन स्वतंत्र कथानकांसाठी सवोर्त्कृष्ट दिग्दर्शक (माधवी कुलकणीर्), सवोर्त्कृष्ट पटकथा (अरुणा जोगळेकर), सवोर्त्कृष्ट संकलन (जमील शेख) आणि सवोर्त्कृष्ट छायाचित्रण (अनिल रेवणकर) असे चार पुरस्कार मिळाले. ' महाराष्ट्र कल्चरल सेंटर ' चे ' छोट्याशा सुटीत ' हे सवोर्त्कृष्ट प्रायोगिक नाटक ठरले.

अभिनेत्यांच्या स्पधेर्त अतुल कुलकणीर् (देवराई), विठ्ठल उमप ( जांभुळ आख्यान) आणि भरत जाधव ( साहेब, बिबी आणि मी) हे विजेते ठरले. तर अभिनेत्रींमध्ये गिरीजा ओक ( मानिनी), स्वाती चिटणीस (रिमोट कंट्रोल) आणि शिल्पा तुळसकर (मेघ दाटले) यांनी यावषीर्च्या ' राम बंधू म. टा. सन्मान 2005' वर आपले नाव कोरले. सवोर्त्कृष्ट बालकलाकाराचा पुरस्कार ' सलाम द सॅल्यूट ' मधील भूमिकेसाठी मेघन जाधव याला मिळाला. मेघनचा चुणचुणीतपणा यावेळी सगळ्यांचच दाद घेऊन गेला.

टीव्ही मालिकांमधील कथाबाह्य कार्यक्रमांमध्ये अल्फा टीव्हीचा ' होम मिनिस्टर ' हा कार्यक्रम सर्वाेत्कृष्ट ठरला. त्याच कार्यक्रमासाठी आदेश बांदेकरला सवोर्त्कृष्ट सूत्रधाराचा पुरस्कार मिळाला. तर शर्वरी पाटणकर हिला ' हाऊसफुल्ल ' साठी सवोर्त्कृष्ट स्त्री सूत्रधाराचा पुरस्कार मिळाला. ' नक्षत्रांचे देणे ' ही सवोर्त्कृष्ट संकल्पना ठरली. ' वृत्तविषयक कार्यक्रमां ' मध्ये दूरदर्शन केंदाची ' महाचर्चा ' सवोर्त्कृष्ट ठरली. तर त्याच कार्यक्रमासाठी उज्वला बवेर् यांना सवोर्त्कृष्ट स्त्री सूत्रधाराच्या सन्मानानं गौरवण्यात आलं. पुरुष सूत्रधाराचा पुरस्कार ' धोबीपछाड ' साठी राहुल सोलापूरकर यांना मिळाला.

' मुंबई टाइम्स ' सवोर्त्कृष्ट युवा कार्यक्रम म्हणून ' कॅम्पस अ फेअर वॉर ' ला गौरवण्यात आलं. तर ' मुंबई टाइम्स दमदार पदार्पणा ' साठी दिला जाणारा विशेष सन्मान ' स्वदेस ' मय गायत्री जोशीला आणि ' पेज थ्री ' मधल्या उपेंद लिमये ' ला प्रदान करण्यात आला.

या स्पधेर्तील तांत्रिक पुरस्कार मुकेश शर्मा, सोनाली खरे, उज्ज्वल निरगुडकर यांच्या हस्ते देण्यात आले. प्रतिमा कुलकणीर् आणि सुरेश वाडकर यांनी लेखनासाठीचे पुरस्कार मिळालेल्यांचा गौरव केला. गीत-संगीताचे पुरस्कार सुदेश भोसले आणि शिल्पा कदम यांनी प्रदान केले तर ' मुंबई टाइम्स विशेष पुरस्कार ' राज ठाकरे, किशोरी शहाणे, मनोज बिडवई आणि आरजू गोवित्रीकर यांच्या हस्ते विजेत्यांना देण्यात आले.

' राम बंधु म. टा. सन्मान ' ची संध्या रंगली ती विविध कलाकारांच्या अविष्कारानं. कविता कोळी आणि त्यांच्या सहकाऱ्यांनी साकारलेल्या ' शिवराज्याभिषेक ' मुळे उपस्थितांनी महाराष्ट्र गौरव ठरलेल्या ' स्वराज्या ' च्या सुवर्णकाळाची सैर घडवली. तर गेली तीन दशके ' हास्यदुनिये ' चा सम्राट ठरलेल्या ' लक्ष्या ' ला ' पॅडी ' पंढरीनाथ कांबळे, अदिती भागवत आणि नीलम शिकेर् यांनी अनोखी श्रद्धांजली वाहिली तेव्हा सहकलाकारांबरोबरच रसिकांच्या डोळ्यांच्या कडाही ओलावल्या. राजू परुळेकर आणि श्वेता भोसले यांच्या सूत्रसंचलनामुळे रंगलेल्या या कार्यक्रमावर आदिती गोवित्रीकर यांच्या लावणीनृत्यानं कळस चढवला. त्याआधी झाला तो फॅशन डिझायनर विक्रम फडणीस यांच्या ' नववधू ' कलेक्शनचा फॅशन शो. सत्यजित तळवळकर, विजय चव्हाण आणि श्रीधर पार्थसारथी यांंची जुगलबंदी संस्मरणीय ठरली. जुन्या मराठी गीतांची अविट गोडी वाढवणारे अमृता नातू यांचे रिमिक्स गायन आणि मराठी गाण्यांवर लिडिया शॉनब्रॉड या जर्मन युवतीनंं साकारलेल्या लाजवाब अदाकारीनं प्रेक्षकांची वाहवा मिळवली.

' म. टा. सन्मान ' साठी उभारण्यात आलेल्या रंगमंचाचे कला दिग्दर्शन अश्विन पाटील यांनी केले होते. आदेश बांदेकर व त्यांचे सहकारी, नॅशनल स्कूल ऑफ इव्हेंट आणि 360 डिग्रीज यांच्या सहकार्याने हा सोहळा उभा राहिला.
