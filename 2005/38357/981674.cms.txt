बहुतांची अंतरे
'सूनामी' नव्हे, त्सूनामी
[ Wednesday, January 05, 2005  08:56:10 pm]


बऱ्याच मराठी वृत्तपत्रांनी ' सूनामी ' म्हटले आहे. ते सुनामी नसून ' त्सूनामी ' ( त्सू दीर्घ) असे आहे.

यातील ' टी ' चा उच्चार हवा, तो सायलेण्ट नाही! हा जपानी जोडशब्द आहे. ' त्सू ' म्हणजे बंदर व ' नामी ' म्हणजे ' लाट '. तेव्हा त्सूनामी ' म्हणजे किनाऱ्यावर अचानक आदळणारी भयंकर विध्वंसक लाट.

जपानी भाषेत ' त्सू ' प्रत्यय असणारे अनेक शब्द व नामे आहेत. उदा: ' कोमात्सू ', ' दैहात्सू ', ' मित्सूबिशी ', '' मात्सूशिटा ', ' जूजूत्सू ' वगैरच्या त्सूनामीस प्राचीन भारतीयांनी ' प्रलय ' हे नाव दिले आहे. इंडोनेशिया, मलेशिया, थायलंड, लंका या देशांत भारतीय प्रभावामुळे यास ' प्रलय ' असेच म्हणतात.

मंगेश ना. सरदेशपांडे, बेळगाव.
