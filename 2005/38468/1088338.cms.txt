महामुंबई
'मम आत्मा गमला'ची रंगीन शाम...
[ Tuesday, April 26, 2005  12:35:11 am]


म. टा. विशेष प्रतिनिधी

मुंबई : मराठी संगीत रंगभूमीचा पाया घालणाऱ्या अण्णासाहेब किलोर्स्करांपासून, तो थेट बोलपटांच्या आगमनामुळे 1935 /1943 दरम्यान मरणकळा लागलेल्या संगीत रंगभूमीला नवचैतन्य देणाऱ्या विद्याधर गोखल्यापर्यंत व ताज्या संवेदनशीलतेने ' मत्स्यगंधा ' पासून ' लेकुरे उदंड जाहली ' पर्यंत नाट्यसंगीतात नवनवे प्रयोग करणाऱ्या पं. जितेंद अभिषेकीपर्यंत साऱ्यांचे पुण्यस्मरण ओघानेच होत होते आणि नीलाक्षी पेंढारकर, सुरेश बापट, महेश कुलकणीर् आदींच्या सुरेल नाट्यसंगीतात रंगायला होत होते...

हा योग शनिवारी जुळवून आणला होता यशवंतराव चव्हाण केंदाच्या रंगस्वर शाखेसाठी शरद पवारकन्या सुप्रिया सुळे यांनी! गोखलेकन्या शुभदा दादरकर, समीक्षक विजय तापस यांच्या मार्गदर्शनाखाली त्यांनी चव्हाण केंदाच्या प्रेक्षागारात आयोजिलेल्या ' मम आत्मा गमला ' या बालगंधवांंनी गायलेल्या नाट्यगीताच्या शीर्षकाच्याच कार्यक्रमाला रसिकांनी अलोट गदीर् केली होती. पडदा उठताच रंगमंचाच्या पार्श्वभूमीच्या पडद्यावर चितारलेली नाट्याचार्य कृ. प्र. खाडिलकर, अण्णासाहेब किलोर्स्कर, देवलमास्तर, राम गणेश गडकरी आदि नाटककारांची आणि संगीत रंगभूमीचे कोहिनूरच ठरलेले नारायण श्ाीपाद राजहंस तथा बालगंधर्व व मास्टर दीनानाथ यांची भव्य तैलचित्रे एका बाजूला तर दुसरीकडे ' खडा मारायचा झालाच तर- ' या सुप्रसिद्ध आविर्भावातील नऊवारी साडी नेसलेल्या बालगंधर्वांचे पूर्ण चित्र-जाणत्या रसिकांकडून टाळ्यांच्या कडकडाटात दाद घेऊन गेली...

' पंचतंुडनररुंडमाळधर पार्वतीश आधी नमितो ' या किलोर्स्कररचित मराठी रंगभूमीवरील आद्य नांदीने या दोन तास रंगलेल्या नाट्यसंगीत मैफिलीचा शुभारंभ झाला. सुभगसुंदर दिसणाऱ्या आणि तितकेच सुंदर आत्मविश्वासाने गाणाऱ्या नीलाक्षी पेंढारकरने लगेच सुरेख ऑर्गन वाजवणाऱ्या मक रंद कंुडलेशी जुगलबंदी करीत ' सौभद ' मधील ' वद जाऊ कुणाला शरण गं ' आणि सुरेश बापट यांनी ' बहुत दिन नच भेटलो सुंदरीला ' दमदारपणे सादर केले. छोट्या संचिता गगेर्नेहीे ' शारदा ' तील ' म्हातारा इतुका न अवघे पाऊणशे वयमान ' व ' कठीण कठीण किती पुरुषहृदय बाई ' गीते छान ठसक्यात गाऊन साऱ्यांची दाद घेतली...

1930 च्या सुमारास स्वातंत्र्य चळवळीचे जनमानस प्रकट करणारे, मास्टर दीनानाथांच्या ऊर्जस्वल गायकीने अजरामर झालेले ' परवशता पाश दैवें ज्यांच्या गळीं लागला ' सीमा दामलेने तर ' मानापमाना ' तील ' शूरां मी वंदिले ' महेश कुलकणीर्ंनी चांगली गायली. धनंजय पुराणिकची तबल्यावरील व राजेंद भावे यांची व्हायोलिनवरील साथही दाद घेणारी होती. अजित भुरे यांनी सूत्रसंचालन केलेल्या या कार्यक्रमात ज्येष्ठ रंगकमीर् दामू केंकरे यांना चव्हाण केंदाचे विश्वस्त डॉ. रवि बापट यांच्या हस्ते सन्मानित करण्यात आले. ' कट्यार काळजात घुसली ' मधील रागमालिकेने संपलेल्या, तीसएक पदांच्या या मैफलीत नीलाक्षीने सादर केलेले गोखल्यांच्या ' मदनाची मंजिरी ' मधील ' ये मौसम है रंगीन, ये रंगीन शाम ' हा शेर तर शनिवारची संध्याकाळ रंगीनच करून गेला...
