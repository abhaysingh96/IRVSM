संवाद
'वेडे' करून सोडावे...
[ Saturday, January 08, 2005  07:34:26 pm]


' तहलका की तमाशा ?' हा भारतकुमार राऊत यांचा दृष्टिकोन ( संवाद, रविवार , 26 डिसेंबर) वाचला. समर्थ आज असते तर या बेछूट आणि बेताल उधळलेल्या माथेफिरू माध्यमांना नक्कीच म्हणाले असते , ' जे जे आपणांसि ठावे, त्यात पदरचे घालावे , ' वेडे ' करून सोडावे, सकळजन! ' नाहीतरी साँस-बहूच्या झगड्यांनी हैराण झालेल्या तमाम भारतीय प्रेक्षकांना ' तहलका ' काय किंवा ' मिड डे ' काय, यांनी केवढी बरं बहुमोल माहिती पुरवली!

म्हणे थॉमस जेफरसन म्हणाला होता की , ' गव्हन्मेर्न्ट विदाउट प्रेस आणि प्रेस विदाउट गव्हन्मेर्न्ट यापैकी मी दुसरा पर्याय निवडेन '! भोळा जेफरसन... बाबारे, प्रेस विदाउट गव्हन्मेर्न्ट हे ठीक आहे ; पण प्रेस विदाउट न्यायव्यवस्थेची भीती ? हा कुठला न्याय ?

धन्य धन्य ती जाहिरा. धन्य तो तहलका.

या साऱ्या प्रकरणावर एक फर्मास बालनाट्य लिहिलं जाऊ शकेल... पाचवीतला बंड्या हळूच स्वयंपाकघरातला लाडू चोरतो आणि त्याची सातवीतली ताई चोरीछुपे त्याचे व्हीडिओ शूटिंग करते! लग्गेच ' आज तक ' वर , ' स्वैपाकघरमे सनसनी '! व्वा रे संवादक्रांती... व्वा रे प्रसामाध्यमं!

अॅड. चैतन्य वि. धारुरकर, औरंगाबाद

फुसके धमाके

' तहलका की तमाशा ' या लेखाबद्दल आपले मनापासून आभार! वेगवान पळणारा घोडा अधिकच उत्तेजित होऊन हवेत उडण्याचा आभास निर्माण करू लागला, तर त्याला लगाम घालण्याचे काम कुणीतरी करायलाच हवे ; जे आपण या लेखाद्वारे समर्थपणे केले आहे.

' नूज चॅनल ' ची शक्ती आणि पोहोच लक्षात घेऊन ' तहलका ' सारखे ' धमाकाप्रेमी ' जव्हा भंडाफोड अथवा गौप्यस्फोट करण्याचा मक्ताच घेतल्यासारखे वागतात, तेव्हा निश्चितच मनात चीड निर्माण होते. वास्तविक या घटनेवर ' प्रकाश ' टाकल्याचा दावा करणारी काही क्षणांची ती अंधकारमय चित्रफीत प्रेक्षकांना फार काही सांगते, असं नाही. त्यातच, सार्वजनिक आयुष्य उपभोगणाऱ्या मंडळींपर्यंत पोहोचणं कठीण असतं, असं नाही ; आणि पत्रकारांना तर मुळीच नाही! परंतु या सुलभतेचा गैरफायदा घेऊन कॅमेऱ्याच्या साह्याने असले फुसके धमाके करणाऱ्यांचा पर्दाफाश करण्याची, त्यांचा ' करविता धनी ' शोधण्याची वेळ आता आली आहे, असं वाटतं.

अर्थात, यानंतरची संपूर्ण जबाबदारी आहे मुदित माध्यमांवर ; कारण ' फास्ट फूड ' तत्त्वानुसार न्यूज चॅनलमार्फत बातम्या ' र्सव्ह ' करण्याच्या या युगात मुदित माध्यमांनाच सर्वार्थाने परिपूर्ण असे वृत्तांकन, अवलोकन आणि दिशादर्शन करणे निकडीचे झाले आहे.

राजेश अग्निहोत्री, नाशिक.

' स्वदेस ' ला दाद!

' स्वदेस ' चं स्वागत ' हम लोग ' कसे करणार, याबद्दल 26 डिसेंबरच्या ' संवाद ' मध्ये मुकेश माचकर यांनी साशंकता व्यक्त केली आहे. यासंदर्भात मला नवीन पिढीची प्रातिनिधिक प्रतिक्रिया नोंदवावीशी वाटते ; ती माझ्या मुलाची!

बहुतेक सगळेच ' पॉप्युलर ' हिंदी सिनेमे बघणाऱ्या त्याच्या गँगबरोबर तो ' स्वदेस ' पाहायला गेला, तेव्हा आल्यावर त्याची प्रतिक्रिया विचारण्याचं धाडसच मी केलं नव्हतं... कारण सुपरफास्ट गेम खेळणाऱ्या, आणि फॅण्टसीप्रधान, अतिवेगवान परदेशी चित्रपट एकचित्ताने पाहणाऱ्या या कम्प्युटरपिढीच्या शिलेदाराला हा संथ, गंभीर, वास्तवाच्या जवळ जाणारा ' स्वदेस ' कितपत भावतोय, याची मला धाकधुकच होती. पण त्यानंच मला गंभीरपणे सांगितलं , ' स्वदेस ' चांगला आहे ; पण फार संथ आहे... ' वर म्हणाला , '... पण संकलकानं कात्री चालवायची ठरवली, तरी कुठं चालवणार ?' त्याला म्हटलं, अरे याला ' लगानसारखा रिपीट ऑडियन्स नाही, असा नॉनग्लॅमरस चित्रपट पुन्हा पाहावासा कुणाला वाटणार ? तर तो म्हणाला , ' पण मला वाटतोय ना! '

मला वाटतं ही प्रतिक्रिया पुरेशी बोलकी आहे. पण ' स्वदेस ' चा ' संदेस ' पिटातल्या प्रेक्षकांपर्यंत पोहोचण्यासाठी, आधी हा प्रेक्षकवर्ग ' स्वदेस ' पाहायला येणार का, हा प्रश्ान् आहेच. नॉनग्लॅमरस विषय, कुठलीही सवंग तडजोड न करता सादर करणाऱ्या आशुतोष गोवारीकरच्या धाडसाला दाद द्यायला हवी! मात्र, यातली समस्या प्रभावीपणे व्यक्त होत नाही, हेही सखेद नमूद करायला हवं.

प्रतिभा खरे-कुलकणीर्, मालेगाव कॅम्प.

तरुणांना ' एकांत ' मिळायलाच हवा!

' जब प्यार किया तो... ' हा अमिता दरेकर यांचा (तरुण-तरुणींच्या वास्तववादी प्रतिक्रियांसह) चालू घडामोडींवर प्रकाशझोत टाकणारा लेख वाचला. (संवाद , 28 नोव्हें.) तरुणांना एकांत मिळणे, ही तशी एक नैसगिक बाब आहे. म्हणून अधिकारपदावरील व्यक्तींनी, विशेषत: पोलिसांनी तसेच ज्येष्ठ नागरिकांनी तरुण पिढीला एकांत कसा मिळू शकेल, याबाबत मार्गदर्शन करायला हवे. तरुणांचं चुकतंय, तरुणांत स्वैराचार बोकाळलाय, असं म्हणून त्यांना ' अक्कल ' शिकवण्यापेक्षा त्यांनी नेमकं काय करावं, कुठे रंगात यावं, येऊ नये, याबाबत ' अनुभवा ' च्या गोष्टी सांगाव्यात.

तरुणांनी प्रेम करू नये, एकांतात करायच्या गोष्टी गदीर्तही करू नयेत, असे म्हणणे त्यांच्यावर अन्यायकारक ठरणार नाही का, याचाही विचार त्या सर्व मंडळींनी करणे गरजेचे आहे. किंबहुना आपणही कधी तरुण होतो, याचाही हलकेच विचार त्यांनी करायला हवा. ' प्रेम ' काही आज अचानक जन्माला आलेले नाही. दुनियेच्या उत्पत्तीपासून व मानवाच्या निमिर्तीपासूनच ते निरंतर चालत आले आहे ; आणि म्हणूनच ते करू नका, असे म्हणण्याचा कुणालाही अधिकार नाही. एकांत मिळणे, हा प्रेमी युगुलांचा मुलभूत अधिकार, हक्क त्यांना मिळवून देणे, हे ज्येेष्ठ नागरिकांनी आपले कर्तव्य समजायला हवे. ज्या दिवशी सगळीकडे अशी एकांताची व्यवस्था तरुणांसाठी केली जाईल, त्या दिवशी भररस्त्यातली थेरं आपोआप थांबतील.

विद्याधर बबन पोखरकर, माजिवडा-ठाणे.
